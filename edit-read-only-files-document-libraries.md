# <a name="edit-a-read-only-file"></a>Редагувати файл лише для читання

<p style="mso-margin-top-alt: auto; mso-margin-bottom-alt: auto; line-height: normal;"><span style="mso-fareast-font-family: 'Times New Roman'; mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Ви можете виявити, що, коли ви відкриваєте файли, вони відкрито лише для читання. У деяких випадках це для додаткової безпеки, наприклад, коли ви відкриваєте файли з Інтернету а в інших випадках, це може бути пов'язано параметр, який можна змінити. У статті &ldquo; <a href="https://support.office.com/en-us/article/Why-did-my-file-open-read-only-3ab4b792-da50-4b38-8628-14c64e1f1d15"><span style="color: blue;">чому моя файл відкрито в читали лише</span></a> &rdquo; ми покажемо деякі сценарії, де відкривається файл лише для читання і деякі кроки можна почати, щоб змінити цю ситуацію.</span></p> <p style="mso-margin-top-alt: auto; mso-margin-bottom-alt: auto; line-height: normal;"><strong><span style="mso-fareast-font-family: 'Times New Roman'; mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Тут&rsquo;s як відкриті відділення файлів для редагування.</span></strong></p> <ul> <li style="mso-margin-top-alt: auto; mso-margin-bottom-alt: auto; line-height: normal;"><span style="mso-fareast-font-family: 'Times New Roman'; mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Замість натискання заголовок документа, натисніть кнопку <strong>Відкрити меню</strong> (три крапки) і натисніть кнопку <strong>редагувати</strong>.</span></li> </ul> <p style="mso-margin-top-alt: auto; mso-margin-bottom-alt: auto; line-height: normal;"><strong><span style="mso-fareast-font-family: 'Times New Roman'; mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Примітки:</span></strong></p> <ul> <li style="mso-margin-top-alt: auto; mso-margin-bottom-alt: auto; line-height: normal;"><span style="mso-fareast-font-family: 'Times New Roman'; mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Якщо ви&rsquo;d достатньо мати документів, відкрито для редагування у Office Online, при натисканні на заголовок, бібліотеку можна настроїти для цього (оскільки у вас є дозвіл змінювати параметри бібліотеки):</span></li> </ul> <ul> <li style="mso-margin-top-alt: auto; mso-margin-bottom-alt: auto; line-height: normal;"><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">За наявності бібліотек з <strong style="box-sizing: border-box;">Checkout</strong>, <strong style="box-sizing: border-box;">необхідних </strong>стовпці/метаданих,&nbsp;або коли <strong style="box-sizing: border-box;">Параметри безпеки чорнового елемента</strong> встановлений або&nbsp;<strong style="box-sizing: border-box;">лише користувачі, які можна редагувати</strong> або&nbsp;<strong style="box-sizing: border-box;">лише користувачі, які можуть затверджувати елементи</strong> у версії Параметри бібліотеки документів SharePoint, ці елементи буде синхронізовано як <strong style="mso-bidi-font-weight: normal;">читання.</strong></span></li> </ul> <ul> <li style="mso-margin-top-alt: auto; mso-margin-bottom-alt: auto; line-height: normal;"><span style="mso-fareast-font-family: 'Times New Roman'; mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Документа Office відкриває в безпечному поданні, незважаючи на те, що ви увімкнути параметр політики "Open Office документи як читання/записування у режимі перегляду". Для отримання додаткової інформації, будь ласка, зверніться до цієї <a href="https://support.microsoft.com/en-us/help/983047/an-office-document-opens-in-protected-view-even-though-you-enable-the"> <span style="color: blue;">статті.</span></a></span></li> </ul>
