# <a name="credential-messages-in-sharepoint-or-onedrive"></a>Повідомлення облікових даних у SharePoint або OneDrive

<p>Користувачі можуть періодично будучи запитано додатками Office облікові дані на сайті SharePoint або OneDrive під час спроби відкрити документи.</p> <p>Нижче наведено кілька прикладів облікових даних повідомлень.</p> <p>&bull;<span style="mso-tab-count: 1;"> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; </span>Увійдіть організаційний обліковий запис</p> <p>&bull;<span style="mso-tab-count: 1;"> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; </span>Облікові дані обов'язкові для заповнення</p> <p>&bull;<span style="mso-tab-count: 1;"> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; </span>Ми не в змозі підключитися прямо зараз, Перевірте підключення до мережі.</p> <p>Якщо ви відчуваєте ці симптоми, будь ласка, спробуйте сходами, розташованими в <a href="https://support.microsoft.com/en-us/help/2913639/office-applications-periodically-prompt-for-credentials-to-sharepoint">цій статті</a>.</p> <p>Крім того, якщо використовується єдиного входу (SSO) для автентифікації, будь ласка, зверніться до <a href="https://support.microsoft.com/en-us/help/4025962/cant-sign-in-after-update-to-office-2016-build-16-0-7967-on-windows-10">цієї статті</a> замість цього.</p> <p>&nbsp;</p>
