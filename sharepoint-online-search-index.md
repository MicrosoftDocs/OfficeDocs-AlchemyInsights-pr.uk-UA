# <a name="search-in-sharepoint-online"></a>Пошук у SharePoint онлайн

<p><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin; color: black; background: white;">Зміст повинен бути поповз і додано до індексу пошуку для користувачів, щоб знайти те, що вони шукають у SharePoint Online. Для отримання додаткових відомостей див. <a href="https://docs.microsoft.com/en-us/sharepoint/make-site-content-searchable">Увімкнути вміст на сайті, щоб бути придатним для пошуку</a>. <br /><br /></span></li> <li style="color: black; Це питання повинен реєструватися як інциденту до служби підтримки для подальшого розслідування. </span></li> </ol>
