# <a name="microsoft-teams-will-stop-working-on-internet-explorer-11-from-nov-30th"></a>Команди Microsoft перестануть працювати в браузері Internet Explorer 11 з 30 листопада

Як повідомляється в MC225570, починаючи з 30 листопада 2020, веб-програма Microsoft Web App більше не підтримуватиме Internet Explorer (IE) 11. 

Радимо оновити нову [програму Microsoft EDGE](https://www.microsoft.com/edge) для більш безпечного та інноваційного онлайнового досвіду. 

Щоб отримати докладніші відомості, ознайомтеся [з веб-програмами microsoft 365, що відповідальні за користування браузером Internet Explorer 11 і Windows 10 (Microsoft EDGE Legacy) – Microsoft Tech спільноти](https://techcommunity.microsoft.com/t5/microsoft-365-blog/microsoft-365-apps-say-farewell-to-internet-explorer-11-and/ba-p/1591666)
