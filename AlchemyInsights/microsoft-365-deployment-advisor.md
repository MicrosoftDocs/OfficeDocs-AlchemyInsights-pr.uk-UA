# <a name="deploy-microsoft-365"></a>Розгортання Microsoft 365

[Порадник з розгортання Microsoft 365](https://go.microsoft.com/fwlink/?linkid=2072646) надає комерційним клієнтам рекомендації, коли ви настроюєте засоби підвищення продуктивності, політики безпеки та можливості керування пристроями. За допомогою цього консультанта можна настроїти та настроїти пристрої вашої організації, використовуючи програму Microsoft 365 Business Premium або Microsoft 365 для підприємств.

Ви отримаєте рекомендації та доступ до ресурсів, щоб активувати хмарні служби, оновити пристрої до найновішої версії Windows 10, а також приєднатися до пристроїв у службі Azure Active Directory (Azure AD), усі в одному централізованому розташуванні.
