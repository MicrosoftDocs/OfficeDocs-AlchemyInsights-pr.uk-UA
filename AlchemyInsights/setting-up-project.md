# <a name="setting-up-project"></a>Настроювання проекту

 Щоб настроїти програму Microsoft Project, виконайте кроки, описані в [цій статті](https://support.office.com/article/7059249b-d9fe-4d61-ab96-5c5bf435f281.aspx).

**Повідомлення для користувачів Mac:** Продукт Project недоступний для MacOS в цей час. 
  
Якщо під час настроювання виникають помилки під час інсталяції, [інсталюйте програму Project за допомогою помічника з підтримки й відновлення](https://aka.ms/SaRA-ProjectSetupScenario).
