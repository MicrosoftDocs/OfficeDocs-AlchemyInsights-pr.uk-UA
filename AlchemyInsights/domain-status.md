# <a name="domain-status---no-services-selected"></a>Стан домену – не вибрано жодних служб

Стан справності домену доступний, якщо підключитися до домену безпосередньо в Microsoft 365 таких служб, як Exchange. Для цього перейдіть до Центру адміністрування Microsoft 365 виберіть пункт Записи **DNS керування**  >  **службою DNS.**

Якщо не вдалося підключити домен безпосередньо до Microsoft 365 служб, стан справності домену недоступний. Наприклад, якщо використовується настроювана маршрутизація DNS для електронної пошти, зовнішніх служб фільтрування спаму або використовується локальна та Exchange Online служби Exchange (також відома як гібридне розгортання).
