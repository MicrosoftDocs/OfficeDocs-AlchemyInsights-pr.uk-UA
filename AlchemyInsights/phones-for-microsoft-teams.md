# <a name="are-you-looking-for-phones-that-are-compatible-with-teams"></a>Ви шукаєте телефони, сумісні з командами?

Як адміністратор ви [керуєте всіма пристроями, використовними](https://docs.microsoft.com/microsoftteams/device-management) для робочих груп у вашій організації з центру адміністрування Microsoft 365. Для отримання додаткової інформації: 

- Перегляньте список [телефонів для Microsoft teams](https://docs.microsoft.com/microsoftteams/phones-for-teams). 
- Перегляньте список [сертифікованих IP-телефонів](https://docs.microsoft.com/microsoftteams/teams-ip-phones#currently-certified-ip-phones). 
- Перегляньте список [пристроїв із відомими обмеженнями та проблемами](https://support.office.com/article/control-calls-using-a-headset-in-teams-65d6e104-444d-4013-b8c2-f11317dd69a8).
