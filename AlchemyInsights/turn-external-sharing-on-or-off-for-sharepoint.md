# <a name="turn-external-sharing-on-or-off-for-sharepoint"></a>Увімкнення зовнішніх обміну або вимкнути для SharePoint

1. У Microsoft Outlook 365 Центр адміністрування, перейдіть на [послуги &amp; надбудови сторінці](https://portal.office.com/adminportal/home#/Settings/ServicesAndAddIns)і натисніть кнопку **сайти**.
    
2. Вмикається параметр, або виберіть варіант обміну, якщо його ввімкнено.
    
> [!NOTE]
> Вимкнення спільного доступу для SharePoint вимикається його OneDrive. Щоб дозволити зовнішня обмін на будь-який сайт, ваш всієї організації має бути на. > дозволити зовнішнім обміну для OneDrive або будь-які окремі сайти SharePoint, слід увімкнути її всієї організації для SharePoint. Ви можете вимкнути її для певного сайту колекцій. Керувати зовнішніми параметри спільного доступу для зв'язку сайтів та сайтів, що належать до Office 365 груп, ви повинні використовувати нові центру адміністрування SharePoint або PowerShell. Щоб отримати додаткові відомості про керування зовнішніми параметри спільного доступу побачити [Увімкнення зовнішніх обміну або вимкнути](https://go.microsoft.com/fwlink/?linkid=866426). 
  
[Дізнайтеся більше про зовнішні обміну в SharePoint](https://go.microsoft.com/fwlink/?linkid=734908)
