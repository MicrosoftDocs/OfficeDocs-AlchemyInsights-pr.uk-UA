# <a name="how-to-customize-sharepoint-site-permissions-via-sharepoint-groups"></a>Настроювання дозволів сайту SharePoint за допомогою груп SharePoint 

Група SharePoint – це набір користувачів, які мають однаковий набір дозволів для сайтів і вмісту. Замість того, щоб призначати дозволи одній особі за раз, можна скористатися групами, щоб зручно призначати однаковий рівень дозволів багатьом користувачам одночасно.

[Настроювання дозволів на сайт SharePoint](https://docs.microsoft.com/sharepoint/customize-sharepoint-site-permissions)
