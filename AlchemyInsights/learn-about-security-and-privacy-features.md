# <a name="learn-about-the-security-and-privacy-features-of-microsoft-search-in-bing"></a>Відомості про функції безпеки та конфіденційності служби Microsoft Search у службі Bing

Функції безпеки та конфіденційності служби Microsoft Search у службі Bing допомагають організації захищати свої дані користувача та дані про робоче місце на робочому місці. Наприклад, запити здійснюються за протоколом HTTPS, а з'єднання буде зашифровано від кінцевого до кінцевого.

Докладні відомості наведено в статті [Безпека та конфіденційність для служби Microsoft Search у службі Bing](https://docs.microsoft.com/microsoftsearch/security-for-search).
