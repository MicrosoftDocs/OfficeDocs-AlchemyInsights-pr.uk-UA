# <a name="prevent-files-from-being-downloaded-from-sharepoint-or-onedrive"></a>Заборона завантаження файлів із SharePoint або OneDrive

Існують різні методи, які можна використовувати для запобігання виникненню або блокування завантаження. Перейдіть на посилання нижче відповідних способів:

- [Блокування завантаження файлів за допомогою умовного доступу Azure AD](https://docs.microsoft.com/cloud-app-security/use-case-proxy-block-session-aad#create-a-block-download-policy-for-unmanaged-devices)

- [Керування доступом із некерованих пристроїв](https://docs.microsoft.com/sharepoint/control-access-from-unmanaged-devices)

- [Настроювання політик IRM у бібліотеках документів SharePoint для обмеження завантаження вмісту](https://docs.microsoft.com/microsoft-365/compliance/set-up-irm-in-sp-admin-center)
