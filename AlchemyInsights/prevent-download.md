# <a name="prevent-files-from-being-downloaded-from-sharepoint-or-onedrive"></a>Запобігти автоматичному завантаженню від SharePoint або OneDrive файлів

- [Завантаження файлу блок з блакитні оголошення умовного доступу](https://docs.microsoft.com/cloud-app-security/use-case-proxy-block-session-aad#create-a-block-download-policy-for-unmanaged-devices)

- [Керування доступом від некерованих пристроїв](https://docs.microsoft.com/sharepoint/control-access-from-unmanaged-devices)

- [Настроювання політики КПДД бібліотек документів SharePoint, щоб обмежити завантаження вмісту](https://docs.microsoft.com/office365/securitycompliance/set-up-irm-in-sp-admin-center)
