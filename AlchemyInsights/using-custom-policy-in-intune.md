# <a name="using-custom-policy-in-intune"></a>Використання настроюваної політики в програмі Inune

Inune має багато параметрів, вбудованих для android, iOS, macOS і пристроїв Windows. Ви також можете створити настроюваний профіль, а потім розгорнути ці користувацькі параметри на пристроях за допомогою функції Inune. Користувацькі профілі призначено для додавання параметрів пристрою та функцій, які не вбудовано в програму Inune.

[Додаткові параметри для пристроїв Android у програмі Inune](https://docs.microsoft.com/intune/custom-settings-android)

[Додаткові параметри для пристроїв із корпоративними пристроями Android у програмі Inune](https://docs.microsoft.com/intune/custom-settings-android-for-work)

[Додаткові параметри для пристроїв iOS у програмі Inune](https://docs.microsoft.com/intune/custom-settings-ios)

[Додаткові параметри для macOS пристроїв у програмі Inune](https://docs.microsoft.com/intune/custom-settings-macos)

[Додаткові параметри для пристроїв із Windows 10 у програмі Inune](https://docs.microsoft.com/intune/custom-settings-windows-10)

[Додаткові параметри для голографічних пристроїв у Windows для бізнесу в програмі Inune](https://docs.microsoft.com/intune/custom-settings-windows-holographic)
