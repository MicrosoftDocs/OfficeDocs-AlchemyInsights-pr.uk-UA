# <a name="use-microsoft-edge-extensions"></a>Використання Microsoft EDGE розширення

[Розширення](https://go.microsoft.com/fwlink/?linkid=2135619) – це додаткові компоненти, які можна інсталювати в Microsoft EDGE, щоб блокувати рекламні оголошення, керувати паролями, перекладати текст, а також настроювати різні можливості перегляду веб-сторінок.
