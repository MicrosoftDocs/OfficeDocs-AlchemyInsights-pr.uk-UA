# <a name="create-a-site"></a>Створити сайт

- Перейдіть на [сторінку "сайти" центру адміністрування Microsoft 365](https://portal.office.com/adminportal/home#/SitesList)і виберіть **Додати сайт**. 
    
- У новому центрі адміністрування SharePoint на сторінці "сайти" натисніть кнопку " **створити**". 
    
**Примітка:** Не можна створити новий сайт із такою ж URL-адресою з наявним сайтом. 
  
Для отримання додаткових відомостей [про створення сайту див.](https://go.microsoft.com/fwlink/?linkid=866295)
  
Для отримання відомостей про дозвіл користувачам створювати власні [сайти див.](https://go.microsoft.com/fwlink/?linkid=866296)
