# <a name="why-software-updates-are-not-being-deployed"></a>Причини, через які оновлення програмного забезпечення не розгорнуто?

Перегляньте наведені нижче дії, якщо настроєно оновлення програмного забезпечення за допомогою кільця оновлення Windows 10, але оновлення не розгорнуто.  

- Спробуйте змінити обслуговування Windows із  **Напівщорічного**  типу випуску каналів на більш суворий і частий тип випуску.  
- Установіть прапорець деферальний період для  **оновлення якості**  та  **оновлення функцій**. Період дефермального може призвести до затримки оновлень протягом 180 днів.
