# <a name="seeing-a-false-alert"></a>Бачачи помилкове оповіщення

- Надіслати файл до корпорації Майкрософт для аналізу. Щоб отримати додаткові відомості, Дізнайтеся, [що робити з помилковими спрацьовувань/негативи в Антивірусі Microsoft Defender](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-antivirus/antivirus-false-positives-negatives#submit-a-file-to-microsoft-for-analysis).
- Створити індикатор "Дозволити", щоб запобігти помилковий позитивний оповіщення. Для отримання додаткових відомостей див. [створення індикаторів для файлів](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/indicator-file).  
- Налаштуйте виключення на пристрої, щоб запобігти скануванню елемента. Для отримання додаткових відомостей див. [визначення виключення на окремому пристрої Windows, щоб запобігти скануванню елемента](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-antivirus/antivirus-false-positives-negatives#define-an-exclusion-on-an-individual-windows-device-to-prevent-an-item-from-being-scanned).
