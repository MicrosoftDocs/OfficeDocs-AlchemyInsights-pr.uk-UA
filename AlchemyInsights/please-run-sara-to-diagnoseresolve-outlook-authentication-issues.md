# <a name="use-sara-to-diagnose-and-resolve-outlook-authentication-issues"></a>Використання Сара для діагностики та вирішення проблем із автентифікацією Outlook

**Зверніть увагу**: перевірте, чи ввімкнуто за [промовчанням безпеки](http://aka.ms/securitydefaults) для вашої організації. Якщо ваш клієнт був створений після 21 жовтня, 2019 і ваш Outlook неодноразово запитують вас про пароль, можливо, у вашому клієнті увімкнено **стандартні настройки безпеки** .

Ми настійно рекомендуємо вам використовувати [Outlook продовжує просити для мого пароля](https://aka.ms/SaRA-OutlookPwdPrompt-Alchemy) діагностики на проблемному комп'ютері для усунення проблем, коли Outlook постійно запитує пароль. Ця функція [Сара](https://diagnostics.office.com/#/) діагностики автоматично перевіряє і повертає можливі рішення для використання для вирішення будь-які виявлені проблеми.
