# <a name="how-to-select-all-in-outlookcom"></a>Як "Виділити все" в Outlook.com

Можна вибрати всі якщо відображаються всі повідомлення. Це для запобігання випадкового дії повідомлень, ви не можете бачити.

**Клацніть, утримуючи натиснутою клавішу CTRL** вибирає кілька окремих повідомлень.

**SHIFT + клацання** вибираються всі повідомлення між першого і останнього повідомлення ви виберете.
