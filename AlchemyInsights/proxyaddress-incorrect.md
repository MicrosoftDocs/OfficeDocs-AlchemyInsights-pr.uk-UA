# <a name="proxyaddress-incorrect"></a>ProxyAddress неправильні

Після того, як об'єкт синхронізується Azure AD, значення, указані в proxyAddresses атрибут Active Directory в порівнянні з Azure AD правила, і потім proxyAddresses атрибут, що вказано в Azure AD. Таким чином, значення атрибута proxyAddresses для об'єкта в Active Directory може бути не так само, як значення атрибута proxyAddresses Azure AD.
  
Щоб дізнатися більше про те, як вказано proxyAddress, дивіться, [як атрибут proxyAddress, що вказано в AZURE AD](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
