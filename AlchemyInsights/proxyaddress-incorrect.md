# <a name="proxyaddress-incorrect"></a>Неправильна адреса Pro"Address"

Коли об'єкт синхронізується з Azure AD, значення, указані в атрибуті Proxyадресами в Active Directory, порівнюють з правилами Azure AD, а потім атрибут proxyAddresses заповнюється в Лазурне AD. Тому значення атрибута proxyAddresses для об'єкта в Active Directory може бути не таким самим, як і значення атрибута proxyAddresses в Azure AD.
  
Щоб дізнатися більше про те, як буде населений адресу, Дізнайтеся, [як атрибут Pro"Address" заповнюється в Azure AD](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
