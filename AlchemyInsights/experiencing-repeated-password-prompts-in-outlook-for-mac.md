# <a name="experiencing-repeated-password-prompts-in-outlook-for-mac"></a>Відчувають повторила підказок пароль в Outlook для Mac?

Ми можемо допомогти, надавати більш спеціалізовані Outlook для Macs Довідка з програми.
  
Будь ласка, перейдіть до постраждалих користувачів Mac, відкрийте застосунок Outlook і клацніть на **допомога** \> **Звернутися до служби підтримки**.
  
Щоб дізнатися більше про використання Outlook для Mac у застосунку служби підтримки, перегляньте цю статтю: [новий контакт підтримує функцію в 2016 Outlook для Mac](https://answers.microsoft.com/msoffice/forum/msoffice_outlook-mso_mac-mso_mac2016/new-contact-support-feature-in-outlook-2016-for/d4fc21c4-25e2-4e10-b943-1fba6542b517.aspx)
