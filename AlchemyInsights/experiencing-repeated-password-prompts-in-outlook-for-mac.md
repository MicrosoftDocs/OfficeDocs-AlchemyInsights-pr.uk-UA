# <a name="experiencing-repeated-password-prompts-in-outlook-for-mac"></a>Виникають запити на повторний пароль у програмі Outlook для Mac?

Ми можемо допомогти забезпечити більш спеціалізований Outlook для Mac допомогу безпосередньо з програми.
  
Будь ласка, перейдіть на проблемному Mac користувача, відкрийте Outlook і натисніть кнопку **Довідка** \> **зверніться до служби підтримки**.
  
Щоб дізнатися більше про використання Outlook для Mac-застосунок підтримки клієнтів, див у цій статті: [новий контакт підтримки функція в Outlook 2016 для Mac](https://answers.microsoft.com/msoffice/forum/msoffice_outlook-mso_mac-mso_mac2016/new-contact-support-feature-in-outlook-2016-for/d4fc21c4-25e2-4e10-b943-1fba6542b517.aspx)
