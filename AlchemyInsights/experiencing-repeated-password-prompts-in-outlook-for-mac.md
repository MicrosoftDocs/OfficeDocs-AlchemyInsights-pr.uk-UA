# <a name="experiencing-repeated-password-prompts-in-outlook-for-mac"></a>Чи виникають повторювані запити пароля в програмі Outlook для Mac?

Ми допоможемо вам надати більш спеціалізовану програму Outlook для Mac безпосередньо в програмі.
  
Перейдіть на сторінку Mac, що торкнулася користувача, відкрийте програму Outlook і виберіть пункт **Довідка** \> **зі служби підтримки**.
  
Щоб дізнатися більше про використання програми Outlook для Mac у програмі підтримки клієнтів, ознайомтеся з цією статтею: [нова функція підтримки контакту в програмі Outlook 2016 для Mac](https://answers.microsoft.com/msoffice/forum/msoffice_outlook-mso_mac-mso_mac2016/new-contact-support-feature-in-outlook-2016-for/d4fc21c4-25e2-4e10-b943-1fba6542b517)
