# <a name="error-onedrive-cannot-connect-to-windows"></a>Помилка: OneDrive не вдалося підключитися до Windows

Щоб усунути помилку, шукати cldflt.sys файл, щоб побачити, якщо він працює і якщо ні, ввімкніть медіацентр. 
  
За додатковою інформацією щодо ввімкнення файлів cldflt.sys побачити [OneDrive не вдалося підключитися до Windows "помилка під час доступу до файлів в OneDrive.](https://go.microsoft.com/fwlink/?Linkid=2031032)
