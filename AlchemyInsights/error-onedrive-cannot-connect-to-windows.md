# <a name="error-onedrive-cannot-connect-to-windows"></a>Помилка: OneDrive, не вдається підключитися до Windows

Щоб виправити повідомлення про помилку, знайдіть файл cldflt. sys, щоб побачити, якщо він працює, і якщо це не так, увімкніть його. 
  
Щоб отримати додаткові відомості про ввімкнення файлу cldflt. sys, див. [OneDrive не вдається підключитися до Windows "помилка під час доступу до файлів у OneDrive.](https://go.microsoft.com/fwlink/?Linkid=2031032)
