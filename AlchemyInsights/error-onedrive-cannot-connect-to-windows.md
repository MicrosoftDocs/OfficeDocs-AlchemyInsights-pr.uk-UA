# <a name="error-onedrive-cannot-connect-to-windows"></a>Помилка: OneDrive не вдалося підключитися до Windows

Щоб вирішити повідомлення про помилку, знайдіть файл cldflt.sys, щоб дізнатися, чи запущено її, і якщо це не так, увімкніть її. 
  
Щоб отримати додаткові відомості про те, як увімкнути cldflt.sys файл, у розділі [onedrive не вдалося підключитися до Windows "помилка під час доступу до файлів у службі onedrive.](https://go.microsoft.com/fwlink/?Linkid=2031032)
