# <a name="download-and-share-updates"></a>Завантаження та обмін оновленнями

Замість завантаження оновлень індивідуально, режим завантаження оптимізації доставки дозволяє обмінюватися між кількома комп'ютерами. [Щоб отримати додаткові відомості див.](https://docs.microsoft.com/windows/deployment/update/waas-delivery-optimization)  

Доступні режими завантаження включають:  
- Лише HTTP  
- Lan  
- Група  
- Інтернет  
- Простий  
- Обійти
