# <a name="restrict-default-video-upload-permissions"></a>Обмеження дозволів на завантаження відео за промовчанням

Коли Користувачі завантажують відео до Microsoft Stream через портал потоку за межами контексту групи або групи каналів, прапорець "дозволити всім у вашій компанії, щоб переглянути це відео" перевіряється за промовчанням. Це робить відео доступним для перегляду всіма у вашій компанії.

- [Установлення дозволів на завантаження відео за промовчанням](https://docs.microsoft.com/stream/default-video-permissions)
