# <a name="troubleshoot-error-code-aadsts50011"></a>Виправлення неполадок із кодом помилки AADSTS50011

Щоб вирішити помилку AADSTS50011, виконайте Рекомендовані дії, описані нижче.

**AADSTS50011**: InvalidReplyTo – адреса відповіді відсутня, неправильно налаштована або не відповідає адресами відповідей, настроєних для програми.

Щоб додати цю відсутчу відповідь до веб-програми "Блакитний Active Directory (AD) або отримати дозвіл на керування програмами в ОГОЛОШЕННІ, виконайте цю дію. Щоб отримати докладніші відомості, перегляньте статтю виправлення неполадок у статті [AADSTS50011](https://docs.microsoft.com/troubleshoot/azure/active-directory/error-code-aadsts50011-reply-url-mismatch)(помилка).
