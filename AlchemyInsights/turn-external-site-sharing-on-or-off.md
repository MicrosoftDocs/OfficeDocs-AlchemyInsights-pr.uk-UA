# <a name="turn-external-sharing-on-or-off-for-a-site"></a>Увімкнення або вимкнення зовнішнього спільного доступу для сайту

Щоб змінити значення зовнішнього спільного доступу для сайту ("колекція сайтів"), потрібно бути глобальним або адміністратором SharePoint. Власникам сайтів заборонено змінювати цей параметр. 

Докладні відомості наведено в статті [Увімкнення або вимкнення зовнішнього спільного доступу для сайту](https://docs.microsoft.com/sharepoint/change-external-sharing-site).
