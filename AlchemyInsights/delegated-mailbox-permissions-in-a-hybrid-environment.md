# <a name="delegated-mailbox-permissions-in-a-hybrid-environment"></a>Делеговані поштової скриньки дозволи в середовищі гібридні

Залежно від версії інсталяції в локальній організації Exchange може знадобитися зробити деякі додаткові кроки, щоб настроїти дозволи делеговані поштових скриньках між локальною організацією та Exchange Online, у службі Office 365. Щоб отримати додаткові відомості про те, як встановити делеговані дозволи Погляньте на [Налаштувати Exchange для підтримки поштової скриньки делеговані дозволи співіснування](https://technet.microsoft.com/en-us/library/mt784505%28v=exchg.150%29.aspx).
