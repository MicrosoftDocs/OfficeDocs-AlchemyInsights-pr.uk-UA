# <a name="turn-off-focused-inbox-in-email-clients"></a>Вимкнути зосереджено вхідних повідомлень, поштових клієнтів

Щоб вимкнути зосереджено папки «Вхідні» в 2016 Outlook, Outlook в Інтернеті, Outlook.com/Hotmail.com, 2016 Outlook для Mac, пошта для Windows або 10, або переглянути [Вимкнути спрямовані "Вхідні"](https://support.office.com/article/f714d94d-9e63-4217-9ccb-6cb2986aa1b2.aspx).
  
Щоб вимкнути спрямовані "Вхідні" для всіх, в організації, побачити [Налаштувати спрямовані "Вхідні" для всіх, у вашій організації](https://support.office.com/article/613a845c-4b71-41de-b331-acdcf5b6625d.aspx).
