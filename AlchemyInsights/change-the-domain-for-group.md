# <a name="change-the-domain-for-microsoft-365-group"></a>Змінення домену для Microsoft 365 групи

Якщо потрібно змінити домен для групи Microsoft 365 під час створення, виконайте дії, описані в [області виберіть домен для створення Microsoft 365 групи](https://docs.microsoft.com/microsoft-365/admin/create-groups/choose-domain-to-create-groups).

Після того, як Microsoft 365 групи вже створено, наразі немає способу змінити домен. Єдиний спосіб полягає у створенні нової групи Microsoft 365 і видалити стару, якщо необхідно (додаткові адреси електронної пошти можуть бути додані через [Центр адміністрування Exchange](https://outlook.office365.com/ecp).
