# <a name="add-an-extension-to-microsoft-edge-from-the-chrome-web-store"></a>Додавання розширення до Microsoft EDGE із веб-магазину Chrome

Щоб додати розширення до краю Microsoft у веб-магазині Chrome, виконайте наведені нижче дії.

1. Перейдіть до веб-магазину Chrome.
2. У банері у верхній частині сторінки виберіть **дозволити розширення з інших сховищ**.
3. Виберіть розширення, яке потрібно додати, а потім натисніть кнопку **Додати до Chrome**.
4. У діалоговому вікні підтвердження уважно ознайомтеся з змінами, які можуть призвести до розширення, а потім натисніть кнопку **Додати розширення**.
Остаточне діалогове вікно підтвердить, що розширення додано.
