# <a name="get-started-with-microsoft-edge--bing"></a>Початок роботи з Microsoft EDGE + Bing

Для початку роботи з Microsoft EDGE + Bing не потрібно настроювати додаткове настроювання. Щоб отримати відомості про роботу, просто вкажіть свій запит у рядку адреси Microsoft EDGE.

Щоб отримати докладні відомості, ознайомтеся [з додатковими можливостями Microsoft EDGE + Bing](https://go.microsoft.com/fwlink/?linkid=2152963).
