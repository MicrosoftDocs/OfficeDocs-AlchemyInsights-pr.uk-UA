# <a name="using-planner-with-microsoft-teams"></a>Використання планувальника з Microsoft teams

Упорядкуйте свої завдання, додавши один або кілька вкладок планувальника до каналу teams: 

- [Додайте вкладку планувальника до каналу групи](https://support.office.com/article/62798a9f-e8f7-4722-a700-27dd28a06ee0#bkmk_addaplannertabtoateamchannel). Користувач, який додає вкладку, має бути членом команди, а не лише власником. Ви можете додати користувача як учасника за допомогою інтерфейсу Microsoft teams.
- [Додайте наявний план до Microsoft teams](https://techcommunity.microsoft.com/t5/Planner-Blog/Bringing-a-Plan-into-Microsoft-Teams/ba-p/57463).
- [Видалення вкладки планувальника або видалення плану](https://support.office.com/article/62798a9f-e8f7-4722-a700-27dd28a06ee0#bkmk_removeaplannertabordeleteaplan). Видалені плани не можна відновити групами. Якщо групу вже видалено, можна [відновити видалені плани та групи](https://blogs.msdn.microsoft.com/brismith/2017/03/29/microsoft-planner-now-you-can-recover-deleted-plans-and-groups).
- [Увімкнення або вимкнення сповіщень про призначення планувальника в teams](https://support.office.com/article/62798a9f-e8f7-4722-a700-27dd28a06ee0#bkmk_getplannerassignmentnotificationsinteams).

**Примітка:** Для загального огляду інтеграції планувальника з Microsoft teams дивіться [використання планувальника в Microsoft teams](https://support.office.com/article/62798a9f-e8f7-4722-a700-27dd28a06ee0).
