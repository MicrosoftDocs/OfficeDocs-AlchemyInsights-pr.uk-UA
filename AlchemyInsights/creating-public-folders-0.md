# <a name="creating-public-folders"></a>Створення спільних папок

Перш ніж користувачі зможуть використовувати спільні папки, потрібно створити принаймні одну поштову скриньку загальнодоступної папки. Щоб створити, виконайте наведені нижче дії.
  
1. У центрі адміністрування Microsoft 365 перейдіть до Exchange **центри адміністрування** \> **Exchange**.

2. Перейдіть до **Public folders** папки \> " **поштові скриньки**спільних папок", а потім натисніть кнопку Add ( **Додати**).

3. У **поштовій скриньці загальнодоступної папки**введіть ім'я поштової скриньки загальнодоступної папки.

4. Натисніть кнопку **Зберегти**.

Після створення поштової скриньки можна почати створювати окремі папки або повідомляти користувачів про [початок створення та використання спільних папок у програмі Outlook](https://support.office.com/article/Create-and-share-a-public-folder-in-Outlook-a2835011-d524-4a5c-a207-05c159bb2a97).
