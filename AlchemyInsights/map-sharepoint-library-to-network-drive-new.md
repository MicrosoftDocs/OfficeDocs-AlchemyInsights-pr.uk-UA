# <a name="map-a-sharepoint-library-to-a-network-drive"></a>Карта бібліотеці SharePoint на мережному диску

Зіставлення бібліотеку як мережевий диск є тимчасовим і підтримуваних тільки за допомогою браузера Internet Explorer. Час від часу, слід відкрити сайт SharePoint у браузері Internet Explorer та виберіть "Залишатися в системі", щоб запобігти сесія закінчується. Замість цього, [Синхронізація файлів SharePoint з новим клієнтом синхронізації OneDrive](https://support.office.com/article/sync-sharepoint-files-with-the-new-onedrive-sync-client-6de9ede8-5b6e-4503-80b2-6190f3354a88) </a> яка надає [Файли на вимогу](https://support.office.com/article/learn-about-onedrive-files-on-demand-0e6860d3-d9f3-4971-b321-7092438fb38e). Отримувати доступ до ваших файлів у OneDrive за допомогою місцевих дискового простору.

Якщо ви вирішили зіставлення диска замість того щоб[використовувати новий клієнт OneDrive синхронізації](https://support.office.com/article/sync-sharepoint-files-with-the-new-onedrive-sync-client-6de9ede8-5b6e-4503-80b2-6190f3354a88), переконайтеся, що ви прямуєте дії, описані в статті нижче. 


**Як налагодити та виправляти неполадки зіставлено мережеві диски**


- [Настроювання та виправлення неполадок зіставлено мережеві диски](https://support.office.com/article/troubleshoot-mapped-network-drives-that-connect-to-sharepoint-online-ef399c67-4578-4c3a-adbe-0b489084eabe?ui=en-US&amp;rs=en-US&amp;ad=US).

Примітка: Для Internet Explorer 10 з Windows 8 або Windows 7, які отримують "Немає доступу" або "контур не доступний" під час проекції ілюстрації на диску, інсталюйте це виправлення для усунення цієї проблеми. Перейти до [помилки, коли ви відкриваєте бібліотеку документів SharePoint, у провіднику Windows або підключити мережний диск до медіатеки після інсталяції Internet Explorer 10](https://support.microsoft.com/help/2846960).
