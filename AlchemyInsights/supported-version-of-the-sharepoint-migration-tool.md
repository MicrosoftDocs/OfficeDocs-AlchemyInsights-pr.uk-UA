# <a name="supported-version-of-the-sharepoint-migration-tool"></a>Підтримувана версія засобу перенесення SharePoint



Зараз засіб перенесення SharePoint підтримує лише міграції з SharePoint 2013. Версії 2010 і 2016 не підтримуються в цей час.
  
Щоб отримати докладніші відомості, ознайомтеся з [введенням засобу перенесення SharePoint](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).
