# <a name="supported-version-of-the-sharepoint-migration-tool"></a>Підтримувані версії засобу міграції SharePoint



Наразі засіб міграції SharePoint підтримує лише міграції з SharePoint 2013. Версії 2010 і 2016 не підтримуються на даний момент.
  
Докладніше перегляньте [представляючи інструмент міграції SharePoint](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).
