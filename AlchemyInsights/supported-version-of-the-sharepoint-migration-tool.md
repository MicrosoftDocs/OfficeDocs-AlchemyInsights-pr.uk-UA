# <a name="supported-version-of-the-sharepoint-migration-tool"></a>Підтримувана версія засобу міграції SharePoint



Наразі засіб міграції SharePoint підтримує лише міграції з SharePoint 2013. Версії 2010 і 2016 не підтримуються в цей час.
