# <a name="content-type-publishing"></a>Публікування типів вмісту

Якщо потрібно, щоб типи вмісту були доступні в організації для всіх користувачів, то це – добрий кандидат, який потрібно опублікувати в усіх колекціях сайтів. Щоб опублікувати тип вмісту, перегляньте статтю [публікація типу вмісту з концентратора публікування вмісту](https://support.office.com/article/publish-a-content-type-from-a-content-publishing-hub-58081155-118d-4e7a-9cc5-d43b5dbb7d02).
