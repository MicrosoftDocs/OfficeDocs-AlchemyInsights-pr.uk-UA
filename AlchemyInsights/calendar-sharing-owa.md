# <a name="troubleshooting-issues-with-calendar-sharing"></a>Усунення проблем із спільним календарем

- Щоб діагностувати проблеми з дозволами, використання [Exchange віддаленого підключення аналізатор вільний зайнятий](https://testconnectivity.microsoft.com/Default.aspx?testId=freeBusy), що дає дуже конкретні повідомлення про помилку.
- Ви також можете скористатися [помічником з підтримки та відновлення Microsoft для microsoft 365](https://diagnostics.office.com/) , щоб діагностувати різні проблеми з календарем. 
- Якщо користувач має Desktop Outlook, [засіб перевірки календаря для Outlook](https://www.microsoft.com/download/details.aspx?id=28786) може вирішити потенційні проблеми.

Додаткові ресурси:

- [Надання спільного доступу до календаря Outlook іншим користувачам](https://support.office.com/article/353ed2c1-3ec5-449d-8c73-6931a0adab88)
- [Спільний доступ до календаря в Outlook, в Інтернеті для бізнесу](https://support.office.com/article/7ecef8ae-139c-40d9-bae2-a23977ee58d5)
