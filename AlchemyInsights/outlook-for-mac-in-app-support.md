# <a name="in-app-support-in-outlook-for-mac"></a>Підтримка в програмі Outlook для Mac

Щоб отримати підтримку в програмі Outlook для Mac, відкрийте програму Outlook і виберіть пункт **Довідка** \> **зі служби підтримки**. Введіть своє запитання, а також учасник групи підтримки в програмі Outlook для Mac. 

Докладні відомості: [зверніться до служби підтримки в Outlook для Mac](https://support.office.com//article/d0410177-8e65-4487-93f7-206a3a3d71a8)
