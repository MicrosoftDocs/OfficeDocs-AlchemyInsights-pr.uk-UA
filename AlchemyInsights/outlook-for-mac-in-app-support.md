# <a name="in-app-support-in-outlook-for-mac"></a>Підтримка в додатку в Outlook для Mac

Щоб отримати підтримку в застосунку Outlook для Mac проблеми, відкрийте Outlook і натисніть кнопку, **допоможіть** \> **звернутися до служби підтримки**. Введіть своє запитання, а також член команди підтримки Outlook для Mac, буде зв'язатися з вами. 

Дізнатися більше: [зверніться до служби підтримки в Outlook для Mac](https://support.office.com//article/d0410177-8e65-4487-93f7-206a3a3d71a8)
