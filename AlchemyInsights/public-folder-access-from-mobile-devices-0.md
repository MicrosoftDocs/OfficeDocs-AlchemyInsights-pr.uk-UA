# <a name="public-folder-access-from-mobile-devices"></a>Доступ до спільних папок із мобільних пристроїв

Загальнодоступні папки недоступні для мобільних пристроїв.
  
Якщо потрібно, щоб функціональні можливості, подібні до спільних папок і доступ до нього з мобільних пристроїв, перегляньте [відомості про Microsoft 365 групи](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) для кращої альтернативою.
