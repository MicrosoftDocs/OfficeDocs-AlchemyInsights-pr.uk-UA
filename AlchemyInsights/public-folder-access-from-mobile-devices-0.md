# <a name="public-folder-access-from-mobile-devices"></a>Доступ до спільної папки з мобільних пристроїв

Спільні папки недоступні на мобільних пристроях.
  
Якщо потрібно мати функціональність, подібну до спільних папок, і отримати доступ до неї з мобільних пристроїв, ознайомтеся з відомостями [про групи Microsoft 365](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) для кращої альтернативи.
