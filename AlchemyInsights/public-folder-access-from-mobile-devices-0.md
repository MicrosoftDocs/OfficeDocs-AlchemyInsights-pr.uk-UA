# <a name="public-folder-access-from-mobile-devices"></a>Доступ до спільної папки з мобільних пристроїв

Спільні папки недоступні з мобільних пристроїв.
  
Якщо вам потрібні функції, схожі на спільні папки та доступ до них із мобільних пристроїв, див. статті Додаткові відомості про Групи [Microsoft 365,](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) щоб отримати кращі альтернативні можливості.
