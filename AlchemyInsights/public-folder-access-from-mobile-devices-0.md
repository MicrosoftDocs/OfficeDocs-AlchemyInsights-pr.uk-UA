# <a name="public-folder-access-from-mobile-devices"></a>Спільна папка доступу з мобільних пристроїв

Спільні папки не будуть доступні з мобільних пристроїв.
  
Якщо ви хочете мати функції, аналогічні спільні папки і отримати до нього доступ з мобільних пристроїв, див. [Докладніше про Office 365 групи](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) за кращу альтернативу.
