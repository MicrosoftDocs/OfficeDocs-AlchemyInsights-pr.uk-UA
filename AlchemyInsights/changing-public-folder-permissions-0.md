# <a name="changing-public-folder-permissions"></a>Зміна дозволів для спільної папки

Користувачами та адміністраторами системи в Outlook можна змінити дозволи спільної папки. Можна також контролюють адміністратори дозволи від обміну Admin центр (EAC), виконавши такі дії:
  
1. O365 на порталі за допомогою лівому навігаційному меню, перейдіть до **Адміністратора центрів** \> **обміну**.
    
2. Перейти до **спільних папок**.
    
3. Звідти ви можете змінити дозволи для окремих спільних папок, призначення групи безпеки дозволи. Для кінцевого користувача змінити дозволи спільної папки що користувачеві необхідно мати права власника папки.
