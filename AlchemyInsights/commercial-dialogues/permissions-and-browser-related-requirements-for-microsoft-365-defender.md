# <a name="permissions-and-browser-related-requirements-for-microsoft-365-defender"></a>Дозволи та відповідні вимоги до браузера для захисника Microsoft 365

- Дозволи: щоб активувати захисника Microsoft 365, ви маєте бути глобальним адміністратором або адміністратором безпеки в Лазурові Active Directory. Щоб переглянути список ролей, які можуть використовувати захисник Microsoft 365, а також дізнатися, як регулюється доступ до даних, зверніться до статті [керування доступом до захисника Microsoft 365](https://go.microsoft.com/fwlink/?linkid=2143626).

- Вимоги до браузера: щоб отримати доступ до захисника Microsoft 365 у центрі безпеки Microsoft 365, використовуйте Microsoft EDGE, Internet Explorer 11 або будь-який HTML 5 – сумісний веб-браузер.
