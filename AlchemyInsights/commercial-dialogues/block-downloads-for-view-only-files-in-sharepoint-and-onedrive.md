# <a name="block-downloads-for-view-only-files-in-sharepoint-and-onedrive"></a>Блокування завантажень для файлів лише для перегляду в SharePoint і OneDrive

Коли ви надаєте спільний доступ до об'єктів іншим користувачам у OneDrive або SharePoint, ви можете надати їм доступ до перегляду, але не завантажувати. Докладні відомості наведено в статті [блокування завантажень для файлів, які доступні лише для перегляду, у службі SharePoint і в службі "OneDrive](https://support.microsoft.com/office/block-downloads-for-view-only-files-in-sharepoint-and-onedrive-6051184b-62ac-4149-b874-13dcd40ef91e)".
