# <a name="report-suspicious-attachments"></a>Повідомляти про підозрілі вкладення

1. У меню " **нове подання** " виберіть елемент **вкладення**, а потім завантажте файл, який потрібно відсканувати.
    > [!NOTE]
    > Цей час має обмеження на 3 Мб. Якщо файл перевищує 3 Мб, спробуйте надіслати його за допомогою ІДЕНТИФІКАТОРА електронної пошти замість цього.
2. Якщо вибрати параметр **має бути заблоковано**, його буде автоматично позначено як **зловмисне програмне забезпечення**.
3. Натисніть кнопку **Надіслати**.
