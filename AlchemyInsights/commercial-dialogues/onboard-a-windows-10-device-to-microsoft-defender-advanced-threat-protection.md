# <a name="onboard-a-windows-10-device-to-microsoft-defender-advanced-threat-protection-mdatp"></a>На борту пристрою з Windows 10 на пристрій захисту від загроз Microsoft Defender розширений захист (MDATP)

MDATP підтримує такі засоби розгортання та методи для Windows 10:

- Групова політика
- Microsoft Endpoint Configuration Manager
- Керування мобільними пристроями, зокрема Microsoft InTune
- Локальні сценарії

Щоб дізнатися більше, перегляньте елементи [Onboarding Tools і методи для пристроїв із Windows 10](https://go.microsoft.com/fwlink/?linkid=2143460).
