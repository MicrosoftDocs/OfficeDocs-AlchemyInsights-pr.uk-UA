# <a name="permissions-in-the-sharepoint-modern-experience"></a>Дозволи в сучасному досвіді SharePoint

За традицією, дозволи SharePoint вдалось керувати за допомогою набору груп дозволів на сайті (власники, учасники, відвідувачі тощо). У службі SharePoint у Microsoft 365 ця дія залишається істинною для певних типів сайтів, але доступні додаткові можливості.  

Щоб отримати докладніші відомості, перегляньте, надайте [спільний доступ і дозволи в сучасному досвіді SharePoint](https://docs.microsoft.com/sharepoint/modern-experience-sharing-permissions).
