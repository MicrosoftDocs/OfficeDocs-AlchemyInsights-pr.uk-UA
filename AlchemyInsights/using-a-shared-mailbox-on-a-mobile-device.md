# <a name="using-a-shared-mailbox-on-a-mobile-device"></a>Використання спільної поштової скриньки на мобільному пристрої

Для того, щоб використовувати спільну поштову скриньку на мобільному пристрої, слід [завантажити Outlook Mobile](https://products.office.com/outlook-mobile-for-android-and-ios) , а потім [Додати спільну поштову скриньку до Outlook Mobile](https://support.office.com/article/Add-a-shared-mailbox-to-Outlook-mobile-f866242c-81b2-472e-8776-6c49c5473c9f).
  
Щоб отримати допомогу: [отримати довідку в додатку для Outlook для IOS та Android](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
