# <a name="give-users-access-to-sharepoint-site"></a>Надання користувачам доступу до сайту SharePoint

Функція "запит на доступ" дає змогу користувачам запитувати доступ до вмісту, у якому вони зараз не мають дозволу на перегляд. Як власник сайту можна настроїти функцію, щоб надсилати повідомлення електронної пошти, коли хтось запитує доступ до сайту. Щоб отримати докладніші відомості, перейдіть на веб [-сайті настроювання та керування запитами на доступ](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3).

Нижче наведено кілька поширених посилань на запити на доступ.

- [Відмовлено в доступі "для отримання доступу до списку запитів або" затвердження запиту не виконано "під час обробки запиту на відкладене в службі SharePoint Online](https://docs.microsoft.com/sharepoint/support/sharing-and-permissions/request-approval-failed)

- [Змінення настройок запитів на доступ](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#bk_enableallow)

- [Затвердження або відхилення запитів на доступ для сайту](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc374462558)

- [Керування запрошеннями для зовнішніх користувачів (лише для служби SharePoint Online)](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc334189260)

- [Повторне надсилання запрошення](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc374462560)
