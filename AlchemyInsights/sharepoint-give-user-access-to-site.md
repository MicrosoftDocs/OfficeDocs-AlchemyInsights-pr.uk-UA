# <a name="give-users-access-to-sharepoint-site"></a>Надати користувачам доступ до сайту SharePoint

Функцію запиту доступу дозволяє людям запитують доступ до вмісту, що вони в даний час не мають дозволу переглядати. Власник сайту можна настроїти функцію для надсилання пошти ви, коли хтось запитує доступ до сайту. Для отримання додаткової інформації, будь ласка, відвідайте [встановити вгору і керувати доступом прохання](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3).

Ось деякі загальні посилання стосовно доступу запити:

- [Доступ до відмовлено в доступі"доступ запити список чи"Запитати затвердження помилка"Коли процес очікує запит у SharePoint Online](https://support.office.com/article/-Access-Denied-to-Access-Requests-list-or-Request-approval-failed-when-you-process-a-pending-request-in-SharePoint-Online-3ec71fe1-9111-4a4c-84a5-bdfe05c2e558)

- [Зміна доступу просить настройки](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#bk_enableallow)

- [Затвердити або відхилити доступ запити на сайт](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc374462558)

- [Керувати запрошення на зовнішні користувачі (SharePoint Online)](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc334189260)

- [Повторно надіслати запрошення](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc374462560)
