# <a name="give-users-access-to-sharepoint-site"></a>Надання користувачам доступу до сайту SharePoint

Функція запиту доступу дає змогу користувачам запитувати доступ до вмісту, який наразі не має дозволу на перегляд. Як власник сайту, ви можете налаштувати функцію надсилання пошти, коли хтось запитує доступ до сайту. Для отримання додаткової інформації, будь ласка, відвідайте [налаштування та керування запитами на доступ](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3).

Нижче наведено кілька поширених посилань щодо запитів на доступ.

- [Відмовлено в доступі "для доступу до списку" запити "або" запит на схвалення не вдалося "під час обробки запит на очікування в SharePoint Online](https://docs.microsoft.com/sharepoint/support/sharing-and-permissions/request-approval-failed)

- [Змінення параметрів запитів доступу](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#bk_enableallow)

- [Схвалення або відхилення запитів доступу для сайту](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc374462558)

- [Керування запрошеннями зовнішніх користувачів (лише для SharePoint Online)](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc334189260)

- [Повторне надсилання запрошення](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3#__toc374462560)
