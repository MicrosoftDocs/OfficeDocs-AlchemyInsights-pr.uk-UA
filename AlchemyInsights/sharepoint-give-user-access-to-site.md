# <a name="give-users-access-to-sharepoint-site"></a>Надати користувачам доступ до сайту SharePoint

Функцію запиту доступу дозволяє людям запитують доступ до вмісту, що вони в даний час не мають дозволу переглядати. Власник сайту можна настроїти функцію для надсилання пошти ви, коли хтось запитує доступ до сайту. Для отримання додаткової інформації, будь ласка, відвідайте [встановити вгору і керувати доступом прохання](https://support.office.com/en-us/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3).
