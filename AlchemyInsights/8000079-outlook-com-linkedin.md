# <a name="issues-with-junk-email-claiming-to-be-from-linkedin"></a>Проблеми з небажаної електронної пошти, стверджуючи, бути з LinkedIn

Якщо ви отримуєте небажані листи, які нібито від LinkedIn, можна створити правило для фільтрації ці листи.
Переглянути [Вхідні правила використання в Outlook.com](https://aka.ms/OutlookComInboxRules) для отримання додаткової інформації.
