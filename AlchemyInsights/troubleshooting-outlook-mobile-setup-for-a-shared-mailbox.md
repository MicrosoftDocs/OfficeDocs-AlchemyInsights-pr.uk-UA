# <a name="troubleshooting-outlook-mobile-setup-for-a-shared-mailbox"></a>Виправлення неполадок настроювання для мобільних пристроїв Outlook для спільної поштової скриньки

Не вдається отримати доступ до спільної поштової скриньки з програми Outlook, інстальованих на мобільному пристрої. Ви хочете цю функцію? Нехай команда Outlook інженерних знати. [Голосувати тут](https://go.microsoft.com/fwlink/?linked=862116)!
  
Ви можете використовувати спільну поштову скриньку на мобільному пристрої, відкрийте браузер, увійшовши в Office 365, а потім перейдіть до Outlook в Інтернеті. Вказівки наведено [в полі Додавання спільної поштової скриньки в Outlook в Інтернеті](https://support.office.com/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207). З Outlook, в Інтернеті, ви зможете отримати доступ до спільної поштової скриньки.
  
Докладніше: [отримати довідку в додатку для Outlook для IOS та Android](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
