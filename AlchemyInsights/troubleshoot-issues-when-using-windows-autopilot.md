# <a name="troubleshoot-issues-when-using-windows-autopilot"></a>Усунення проблем під час використання Windows автопілот

Якщо у вас виникли проблеми з Автопілот, зверніться до [автопілот виправлення неполадок](https://docs.microsoft.com/windows/deployment/windows-autopilot/troubleshooting) і [відомі проблеми](https://docs.microsoft.com/windows/deployment/windows-autopilot/known-issues) сторінки.

Щоб отримати додаткові відомості про настроювання автопілота Windows, перегляньте розділ [Реєстрація пристроїв Windows у InTune за допомогою автопілота Windows](https://docs.microsoft.com/intune/enrollment/enrollment-autopilot).
