# <a name="add-a-content-type-to-a-list-or-library"></a>Додавання типу вмісту до списку або бібліотеки

1. Перейдіть до [налаштувань списку або бібліотеки](https://support.microsoft.com/en-us/office/edit-list-settings-in-sharepoint-online-4d35793b-246e-42a3-990c-563a83795b7f).
2. У розділі **типи вмісту**виберіть пункт **Додати з наявних типів вмісту сайту**. Якщо **типи вмісту** не відображаються, виберіть **додаткові настройки**.
3. Виберіть зі списку **вибрати типи вмісту сайту** .
4. У списку доступні типи вмісту сайту виберіть ім'я потрібного типу вмісту та перемістіть вибраний тип вмісту до типів вмісту, щоб додати список, натисніть кнопку Додати.
5. Повторіть кроки для додаткових типів вмісту.

Для отримання додаткових відомостей див. [Додавання типу вмісту до списку або бібліотеки](https://support.microsoft.com/en-us/office/add-a-content-type-to-a-list-or-library-917366ae-f7a2-47ad-87a5-9689a1884e60).
