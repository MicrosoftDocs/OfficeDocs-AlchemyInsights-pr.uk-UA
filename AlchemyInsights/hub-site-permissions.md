# <a name="hub-site-permissions"></a>Дозволи на сайт-концентратор

Керування дозволами для сайту концентратора залежить від базового типу сайту. Якщо сайт – сайт групи, до якої підключено групу, ви маєте керувати дозволами в групі Microsoft 365. Якщо це сайт для спілкування, ви маєте керувати дозволами в групах SharePoint.

- [Дозволи на сайт-концентратор](https://docs.microsoft.com/sharepoint/modern-experience-sharing-permissions#hub-site-permissions)  
- [Веб-вузли центру планування](https://docs.microsoft.com/sharepoint/planning-hub-sites)
