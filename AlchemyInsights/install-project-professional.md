# <a name="install-project-professional"></a>Інсталяція програми Project Professional

Перегляньте статтю [Інсталяція програми Project](https://support.office.com/article/install-project-7059249b-d9fe-4d61-ab96-5c5bf435f281) , щоб дізнатися, як інсталювати абонентську версію Project Online, а також версію Project 2019, 2016 або 2013. 

У програмі Project 2010 або 2007 виконайте кроки, описані в [office 2010](https://support.office.com/article/install-office-2010-1b8f3c9b-bdd2-4a4f-8c88-aa756546529d) або [Office 2007](https://support.office.com/article/install-office-2007-88a8e329-3335-4f82-abb2-ecea3e319657). 

Проект продається як окрема програма, тому якщо у вас ще немає, [придбайте або випробуйте проект](https://www.microsoft.com/evalcenter/evaluate-project).
