# <a name="troubleshoot-issues-with-microsoft-intune"></a>Усунення несправностей із Microsoft Intune

Огляд ресурсів, перераховані нижче, щоб вирішити проблему, ваш тепер.
  
1. Використовувати [Intune виправлення неполадок портал](https://devicemanagement.microsoft.com/#blade/Microsoft_Intune_DeviceSettings/TroubleshootBlade) діагностувати і вирішити деякі поширені помилки. Огляд [цей](https://docs.microsoft.com/en-us/intune/help-desk-operators) документ для більш докладної інформації.  
2. Огляд [Intune TechNet ](https://social.technet.microsoft.com/forums/en-us/home?forum=microsoftintuneprod), щоб знайти рішення для загальних питань і відповідей.  
3. Від'їзд [Служби охорони здоров'я](https://portal.office.com/AdminPortal/Home#/servicehealth) , щоб побачити стан служби.   
4. Огляд [Intune документації](https://docs.microsoft.com/en-us/intune/) для планування, як-к, усунення несправностей статті та багато іншого.
