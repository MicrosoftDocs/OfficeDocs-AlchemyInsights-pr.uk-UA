# <a name="screen-sharing-with-skype-consumer-users"></a>Спільний доступ до екрана з користувачами Skype споживач

Загальні інструкції з надання спільного доступу до програми, щоб дізнатися про обмеження, які мають спільний доступ, а для ресурсів для виправлення неполадок, перегляньте [спільний доступ до екрана у Skype для бізнесу](https://support.office.com/article/Share-your-screen-in-Skype-for-Business-2d436dc9-d092-4ef1-83f1-dd9f7a7cd3fc).  

Наразі Skype для бізнес-користувачів не може надати спільний доступ до екрана Skype споживачів користувачів. Будь ласка, [натисніть тут](https://www.skypefeedback.com/forums/299913-generally-available/suggestions/12335259-enable-screen-sharing-to-consumer-skype-users) , щоб забезпечити зворотній зв'язок щодо ввімкнення спільного доступу до екрана з користувачами Skype.
