# <a name="screen-sharing-with-skype-consumer-users"></a>Надання спільного доступу до екрана зі споживачем користувачів Skype

Щоб отримати докладні вказівки про те, як надати спільний доступ до програми, щоб дізнатися про обмеження надання спільного доступу, а також для виправлення неполадок, перегляньте [спільний доступ до екрана в програмі "Skype для бізнесу](https://support.microsoft.com/office/share-and-present-content-from-skype-meetings-app-skype-for-business-web-app-234b0c06-a88d-4707-904c-4fd6c571fc01)".  

Зараз користувачі Skype для бізнесу не можуть надавати спільний доступ до екрана користувачам Skype. [Натисніть тут](https://www.skypefeedback.com/forums/299913-generally-available/suggestions/12335259-enable-screen-sharing-to-consumer-skype-users) , щоб надати відгук про ввімкнення спільного доступу до екрана з користувачами користувачів Skype.
