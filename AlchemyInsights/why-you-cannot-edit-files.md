# <a name="why-cant-i-edit-files"></a>Чому не можна редагувати файли?

Якщо документа, аркуша або презентації заблоковано для редагування і не може бути відкритим у програмами Windows desktop, цілком можливо, що файл вже був відредагований кимось іншим, або вони мають файл узято на редагування. Давайте коротко розглянемо деякі з причин, [чому ви не зможете редагувати файл](https://support.office.com/article/why-can-t-i-edit-this-file-97315f48-aa5e-49d3-a4ae-a14b73daf87b).

За наявності бібліотек у **касі, необхідні** або **перевірка** стовпців або метаданих або **Параметри безпеки чорнового елемента** установити **лише користувачі, які можна редагувати** або **лише користувачі, які можуть затверджувати елементи** у налаштуваннях версія на бібліотеки:

- Для бібліотек документів SharePoint ці елементи буде синхронізовано лише для читання.

- Для OneDrive бібліотек, OneDrive буде показувати значок замка (OneDrive файл заблоковано значок) поруч із sync статус і користувач не зможуть синхронізувати бібліотеку поки не вказані параметри були видалені. 

За додатковою інформацією щодо закриті файли для Word або Excel будь ласка посилання статті нижче.

- [Слово файл заблоковано для редагування іншим користувачем](https://support.microsoft.com/help/313472/the-document-is-locked-for-editing-by-another-user-error-message-when)
