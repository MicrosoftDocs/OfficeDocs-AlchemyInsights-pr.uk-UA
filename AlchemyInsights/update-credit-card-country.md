# <a name="update-credit-card-country-information"></a>Оновлення відомостей про країну або регіон кредитної картки

Оновити відомості про країну для свого методу оплати не можна. Країна, призначена для оплати, – це країна, вибрана, коли організацію настроєно на Microsoft 365.
