# <a name="welcome-to-the-microsoft-365-lighthouse-private-preview"></a>Ласкаво просимо до приватного зразка Microsoft 365 маяка

Ви берете участь в особистому попередньому перегляді для Microsoft 365 маяка. Якщо ви бажаєте надати відгук про Microsoft 365 Lighthouse, надішліть докладні відомості про наш [голосовий сайт користувача](https://aka.ms/M365Lighthouseuservoice). Якщо у вас є запитання або проблема під час використання функції Microsoft 365 маяка, створіть запит на підтримку, щоб отримати додаткову допомогу.
