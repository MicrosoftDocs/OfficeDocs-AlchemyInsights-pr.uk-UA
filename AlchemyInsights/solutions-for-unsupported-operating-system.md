# <a name="solutions-for-unsupported-operating-system"></a>Рішення для "Непідтримувана операційна система"

Під час спроби інсталювати Office 365, може відобразитися повідомлення про помилку **не підтримується операційною системою** якщо ваш пристрій не відповідає [системним вимогам](https://products.office.com/office-system-requirements).
  
Office 365 вимагає:
  
- ПК: Windows 10, Windows 8.1, ОС Windows 8, Windows 7 пакета оновлень 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 або Windows Server 2008 R2

    OR

- Mac OS X 10.10

Якщо під час спроби інсталяції Office на мобільному пристрої або Chromebook отримали це повідомлення, переглянути [як встановити та запустити Microsoft Office на Chromebook](https://support.office.com/article/32f14a23-2c1a-4579-b973-d4b1d78561ad?wt.mc_id=Alchemy_ClientDIA).
