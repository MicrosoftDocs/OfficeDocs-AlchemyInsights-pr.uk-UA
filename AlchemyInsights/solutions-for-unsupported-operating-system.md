# <a name="solutions-for-unsupported-operating-system"></a>Рішення для "Непідтримувана операційна система"

Під час спроби інсталювати Microsoft 365 може відображатися повідомлення про помилку **непідтримуваної операційної системи** , якщо пристрій не відповідає [системним вимогам](https://products.office.com/office-system-requirements).
  
Microsoft 365 вимагає:
  
- ПК: Windows 10, Windows 8,1, Windows 8, пакет оновлень 1 для Windows 7, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 або Windows Server 2008 R2

    Або

- Mac OS X 10,10

Якщо ви отримали це повідомлення під час спроби інсталювати Office на мобільному пристрої або Chromebook, Дізнайтеся, [як інсталювати та запустити Microsoft Office на Chromebook](https://support.office.com/article/32f14a23-2c1a-4579-b973-d4b1d78561ad?wt.mc_id=Alchemy_ClientDIA).
