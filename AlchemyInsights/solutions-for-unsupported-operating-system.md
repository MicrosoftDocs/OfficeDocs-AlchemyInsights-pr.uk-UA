# <a name="solutions-for-unsupported-operating-system"></a>Рішення для "непідтримуваної операційної системи"

Під час спроби інсталювати Microsoft 365 може з'явитися повідомлення про помилку **операційної системи** , якщо ваш пристрій не відповідає [системним вимогам](https://products.office.com/office-system-requirements).
  
Корпорація Майкрософт 365 потребує:
  
- ПК: Windows 10, Windows 8,1, Windows 8, Windows 7 із пакетом оновлень 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 або Windows Server 2008 R2

    АБО

- Mac OS X 10,10

Якщо під час спроби інсталювати пакет Office на мобільному пристрої або у Chromebook з'являється таке повідомлення, Дізнайтеся, [як інсталювати та запустити Microsoft Office у Chromebook](https://support.office.com/article/32f14a23-2c1a-4579-b973-d4b1d78561ad?wt.mc_id=Alchemy_ClientDIA).
