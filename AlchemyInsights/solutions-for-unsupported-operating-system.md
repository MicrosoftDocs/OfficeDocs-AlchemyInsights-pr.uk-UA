"Непідтримувана операційна система"

# <a name="solutions-for-unsupported-operating-system"></a>Рішення для "Непідтримувана операційна система"

Під час спроби інсталювати Office 365/2016, може відобразитися таке повідомлення про помилку якщо ваш пристрій не відповідає [системним вимогам](https://products.office.com/office-system-requirements).
  
 **"Непідтримувана операційна система"**
  
Office 365/2016 вимагає:
  
ПК: Windows 10, Windows 8.1, ОС Windows 8, Windows 7 пакета оновлень 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 або Windows Server 2008 R2
  
OR
  
Mac OS X 10.10
  
Якщо отримали це повідомлення при спробі встановити Office на мобільному пристрої або Chromebook, див. [Настроювання Office apps та адресу електронної пошти на мобільному пристрої](https://support.office.com/article/7dabb6cb-0046-40b6-81fe-767e0b1f014f?wt.mc_id=Alchemy_ClientDIA.aspx).
