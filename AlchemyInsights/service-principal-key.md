# <a name="service-principal-key"></a>Головний ключ служби

Щоб отримати вказівки з керування ключовими обліковими даними для основної служби, ознайомтеся з такими статтями:

- [Get-AzureADServicePrincipalKeyCredential](https://docs.microsoft.com/powershell/module/azuread/get-azureadserviceprincipalkeycredential) – щоб отримати ключові облікові дані для основної служби за допомогою PowerShell.

- [New-AzureADServicePrincipalKeyCredential](https://docs.microsoft.com/powershell/module/azuread/new-azureadserviceprincipalkeycredential) – щоб створити новий ключ облікових даних для основної служби за допомогою PowerShell.

- [AzureADServicePrincipalKeyCredential (видалити](https://docs.microsoft.com/powershell/module/azuread/remove-azureadserviceprincipalkeycredential) ) – щоб видалити ключові облікові дані від основної служби за допомогою PowerShell.
