# <a name="manage-live-captions"></a>Керування живими підписами

Для отримання додаткових відомостей про використання функції "пряма назва" див. [використання Live-підписів у зборах teams](https://support.microsoft.com/office/use-live-captions-in-a-teams-meeting-4be2d304-f675-4b57-8347-cbd000a21260) або [використання живих титрів у прямому ефірі](https://support.microsoft.com/office/use-live-captions-in-a-live-event-1d6778d4-6c65-4189-ab13-e2d77beb9e2a).  

За промовчанням Live-підписи не автоматично ввімкнена для користувача під час наради. Користувач бачить параметр **Увімкнути прямі підписи** в меню переповнення **(...)** , щоб увімкнути їх. Щоб керувати або вимикати прямі підписи, перегляньте [функцію увімкнути прямі підписи](https://docs.microsoft.com/microsoftteams/meeting-policies-in-teams#enable-live-captions).

Щоб отримати додаткові відомості про впровадження політики для нарад [див.](https://docs.microsoft.com/microsoftteams/meeting-policies-in-teams)
