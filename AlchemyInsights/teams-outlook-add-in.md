# <a name="teams-outlook-add-in"></a>Команди надбудови Outlook

Якщо користувачі не бачать команди надбудови Outlook після переходу до режиму оновлення, підтримувані команди (групи, SfBWithTeamsCollabAndMeetings або острови), то надбудова може не вдалося завантажити належним чином.  Зверніться до такої статті, щоб вирішити цю проблему: 

- [Чому я не бачу надбудови для наради Microsoft teams для Outlook?](https://techcommunity.microsoft.com/t5/microsoft-teams-blog/why-can-t-i-see-the-microsoft-teams-meeting-add-in-for-outlook/ba-p/174630)
