# <a name="add-or-remove-a-custom-role"></a>Додавання та видалення настроюваної ролі

Щоб керувати ресурсами Azure, ознайомтеся з [надбудовою додавання або вилучення завдань лазуроролей за допомогою порталу "Лазурний"](https://docs.microsoft.com/azure/role-based-access-control/role-assignments-portal).

Щоб керувати настроюваними ролями для служби Azure Active Directory (Azure AD), ознайомтеся [з настроюваними ролями адміністратора в Лазурому Active Directory (ознайомлювальну версію)](https://docs.microsoft.com/azure/active-directory/users-groups-roles/roles-custom-overview).
