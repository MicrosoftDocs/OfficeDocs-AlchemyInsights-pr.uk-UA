# <a name="project-web-app-is-missing-from-the-sharepoint-site-collection"></a>Веб-застосунок Project Web App відсутній у колекції сайтів SharePoint

У колекції сайтів SharePoint веб-застосунку Project Web App бракує таких причин:

1. Організації мають лише ліцензії на веб-сайті Project Online Essentials. Адміністраторам потрібно буде додати ліцензію Project Online Professional і/або Premium. Сайт буде виходити лише для читання на цьому етапі. Докладні відомості наведено в статті [порівняння рішень з керування проектами](https://products.office.com/project/compare-microsoft-project-management-software?tab=1).
2. У клієнті Виявлено проблему підготовки, у якій відкладена підготовка сайту Project Web App. Якщо Минуло більше 24 годин, і ви все ще відчуваєте проблему, увійдіть у службу підтримки. Ми розуміємо, що протягом довгого часу може здатися, що 24 години. У багатьох випадках ми вже працюємо над вирішенням проблеми. Щоб завершити вирішення, надайте нам принаймні 24 години.
