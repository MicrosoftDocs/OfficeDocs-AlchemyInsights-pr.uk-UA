# <a name="use-jamf-to-configure-microsoft-edge-policy-settings-on-macos"></a>Налаштування параметрів політики Microsoft Edge в macOS за допомогою Jamf

Щоб налаштувати Microsoft Edge (стабільний канал 81 або пізнішої версії) в macOS, використовуйте файл маніфесту політики Microsoft Edge у Jamf Pro 10.19.

Докладні відомості див. [в статті Налаштування параметрів політики Microsoft Edge в macOS за допомогою Jamf.](https://go.microsoft.com/fwlink/?linkid=2134761)
