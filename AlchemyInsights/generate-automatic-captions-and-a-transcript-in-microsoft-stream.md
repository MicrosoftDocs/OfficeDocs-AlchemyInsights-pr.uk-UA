# <a name="generate-automatic-captions-and-a-transcript-in-microsoft-stream"></a>Створення автоматичних підписів і Стенограма в Microsoft Stream

Потокове передавання даних може автоматично генерувати підписи за допомогою автоматичних технологій розпізнавання мовлення, а також переглядати вікно "Стенограма прокручування", щоб легко переходити між ними та слідкувати за ними, а також за допомогою усного вмісту відео.

- [Створення автоматичних підписів і Стенограма для відео Microsoft Stream](https://docs.microsoft.com/stream/portal-autogenerate-captions)

- [Відображення вікна «Стенограма прокручування» для відео в Microsoft Stream](https://docs.microsoft.com/stream/portal-configure-transcript-mode)
