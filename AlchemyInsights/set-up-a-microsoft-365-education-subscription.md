Щоб отримати покрокове керівництво, ознайомтеся з [аудіо та відео з описом](https://aka.ms/M365EduSetup) всього процесу налаштування Microsoft 365 Education.

**Додаткові ресурси**

- Перегляньте [, як створити обліковий запис клієнта Office 365](https://docs.microsoft.com/microsoft-365/education/deploy/create-your-office-365-tenant) для огляду.
- Дивіться [початок роботи з Microsoft 365 освіта](https://docs.microsoft.com/education/) для ІТ-адміністраторів, розробників і партнерів.
