# <a name="create-a-shared-mailbox"></a>Створення спільної поштової скриньки

Спільна поштова скринька дозволяє більше однієї людини надсилати та отримувати повідомлення електронної пошти для псевдоніма, наприклад **Sales@contoso.com**, без призначення іншої ліцензії.
  
Щоб дізнатися більше, зверніться до [створення спільної поштової скриньки](https://docs.microsoft.com/office365/admin/email/create-a-shared-mailbox) .
