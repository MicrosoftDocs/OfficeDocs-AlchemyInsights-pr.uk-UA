# <a name="learn-about-external-calendar-sharing-options"></a>Відомості про параметри спільного доступу до календаря

Якщо потрібно запланувати наради з людьми поза межами організації, Microsoft 365 дає змогу робити календарі доступними для "зовнішніх користувачів", які мають бачити вільний/зайнятий час, але не мають облікових записів користувачів для середовища Microsoft 365. Для отримання додаткових відомостей див. [спільний доступ до календарів із зовнішніми користувачами](https://docs.microsoft.com/office365/admin/manage/share-calendars-with-external-users).
