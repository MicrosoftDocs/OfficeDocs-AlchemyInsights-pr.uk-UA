# <a name="desktop-app-for-sharepoint"></a>Застосунок робочого стола для SharePoint

Організації вводиться в приватний перегляд програмою застосунок робочого стола для SharePoint. Microsoft не в даний час підтримує настільний додаток для SharePoint (приватний перегляд). Будь ласка, використовуйте ваш РНБО партнерів Microsoft приватного перегляду отримати допомогу з настільний додаток для SharePoint.
