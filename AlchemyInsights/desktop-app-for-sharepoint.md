# <a name="desktop-app-for-sharepoint"></a>Програма для настільних комп'ютерів у службі SharePoint

Ваша організація вводиться в приватну програму для настільних комп'ютерів для програми SharePoint. Корпорація Майкрософт зараз не підтримує настільну програму для SharePoint (приватний зразок). Щоб отримати допомогу з настільною програмою SharePoint, скористайтеся партнерами для приватних ознайомлювального ознайомлення з Microsoft.
