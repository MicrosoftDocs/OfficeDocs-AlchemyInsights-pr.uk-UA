# <a name="create-an-allow-list-of-endpoints-for-microsoft-edge"></a>Створення списку дозволених кінцевих точок для Microsoft EDGE

Щоб мати повний масив функцій Microsoft EDGE у вашому розпорядженні, необхідно мати підключення до Інтернету. Крім того, щоб браузер міг спілкуватися через брандмауери та інші механізми системи безпеки в Інтернеті, потрібно додати набір URL-адрес доменів до [списку дозволених кінцевих точок](https://go.microsoft.com/fwlink/?linkid=2135054).
