# <a name="control-who-creates-microsoft-365-groups-and-microsoft-teams"></a>Керування, який створює групи Microsoft 365 і Microsoft teams

За промовчанням усі користувачі в організації можуть створювати групи Microsoft 365 і Microsoft teams. Можна змінити значення за промовчанням і призначити право на створення груп Microsoft 365 або команд для підмножини користувачів у вашій організації. Ця зміна також впливає на створення групи з будь-якої служби Microsoft 365, такі як Outlook, команди, планувальник і так далі.

Для отримання додаткових відомостей див. [керування, які можуть створювати групи microsoft 365](https://support.office.com/article/Manage-who-can-create-Office-365-Groups-4c46c8cb-17d0-44b5-9776-005fced8e618) і [Microsoft teams](https://aka.ms/rtsf).
