# <a name="disabling-briefing-email"></a>Вимкнення електронної пошти для брифінгу

Як адміністратор, у вас є гнучкість, щоб вимкнути брифінг для всієї організації або для окремих користувачів. Щоб дізнатися, як керувати цими настройками, перегляньте статтю [Настроювання електронної пошти для брифінгу](https://docs.microsoft.com/briefing/be-admin).
