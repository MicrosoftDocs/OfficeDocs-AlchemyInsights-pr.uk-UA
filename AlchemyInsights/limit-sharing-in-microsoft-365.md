# <a name="limit-sharing-in-microsoft-365"></a>Обмеження спільного доступу в Microsoft 365

Хоча ви не можете повністю вимкнути внутрішній спільний доступ або видалити кнопку "спільний доступ" із сайтів, можна скористатися різноманітними способами обмеження спільного доступу в Microsoft 365 відповідно до потреб вашої організації. Докладні відомості наведено в статті [обмеження спільного доступу в Microsoft 365](https://docs.microsoft.com/Office365/Enterprise/microsoft-365-limit-sharing).
