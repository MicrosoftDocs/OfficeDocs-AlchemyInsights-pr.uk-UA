# <a name="limit-sharing-in-microsoft-365"></a>Обмежити спільний доступ у Microsoft 365

Хоча ви не можете повністю вимкнути внутрішній спільний доступ або видалити кнопку "спільний доступ" із сайтів, існує багато способів обмежити спільний доступ у Microsoft 365 для задоволення потреб організації. Щоб отримати додаткові відомості див. [обмеження спільного доступу в Microsoft 365](https://docs.microsoft.com/Office365/Enterprise/microsoft-365-limit-sharing).
