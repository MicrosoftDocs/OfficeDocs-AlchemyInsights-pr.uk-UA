# <a name="wipe-corporate-data-from-intune-managed-apps"></a>Стирання корпоративних даних із керованих програм

Inune підтримує видалення корпоративних даних із керованих програм політики захисту від програм (програми). 

Щоб дізнатися більше про стирання, видалення або видалення корпоративних даних із цих програм, Дізнайтеся, [як стерти лише корпоративні дані з керованих програм](https://docs.microsoft.com/mem/intune/apps/apps-selective-wipe).
