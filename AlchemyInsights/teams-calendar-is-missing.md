# <a name="teams-calendar-is-missing"></a>Відсутнє календар команд

Можливо, адміністратор групи вимкнув календар, тому спочатку переконайтеся, що він доступний для вас.

Якщо вона доступна, скористайтеся перемикацією своєї версії команд.

- Якщо використовується веб-версія, перейдіть до інстальованої версії.
- Якщо використовується інстальована версія, скористайтеся командами в Інтернеті.
