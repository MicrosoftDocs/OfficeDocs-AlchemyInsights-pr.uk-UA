# <a name="outlook-categories"></a>Outlook категорії

Категорії кольорів дозволяють легко ідентифікувати та групувати пов'язані елементи в Microsoft Outlook. Для отримання додаткової інформації дивіться [створення і призначити категорії кольорів](https://support.office.com/article/A1FDE97E-15E1-4179-A1A0-8A91EF89B8DC).
