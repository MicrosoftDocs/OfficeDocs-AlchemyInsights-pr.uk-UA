# <a name="how-to-install-office-on-an-ipad"></a>Як встановити Office на iPad

- Користувач, [торкніться посилання для завантаження](https://support.office.com/article/9df6d10c-7281-4671-8666-6ca8e339b628?wt.mc_id=Alchemy_ClientDIA) зі свого мобільного пристрою, для Word, Excel, PowerPoint і OneNote.

- Користувач запустити програму, торкніться **вхід** і введіть свої роботи або школа облікові дані.
