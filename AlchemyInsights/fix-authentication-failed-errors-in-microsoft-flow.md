# <a name="fix-authentication-failed-errors-in-microsoft-flow"></a>Виправлення помилок "помилка автентифікації" у Microsoft Flow

Якщо з'являється повідомлення про помилку "помилка автентифікації" під час спроби ввійти до Microsoft Flow, це тому, що служба Flow вимкнуто Azure Active Directory. Докладні відомості та дії, щоб виправити цю помилку, див. [не вдається ввійти до Microsoft Flow, і з'являється повідомлення про помилку "помилка автентифікації"](https://support.microsoft.com/help/4316891).
