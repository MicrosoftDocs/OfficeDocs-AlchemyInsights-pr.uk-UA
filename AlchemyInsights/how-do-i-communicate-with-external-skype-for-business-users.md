# <a name="allow-external-communications-with-skype-for-business"></a>Дозволити зовнішнє спілкування в "Skype для бізнесу" 

Хочете дозволити користувачам Skype для бізнесу зв'язатися з користувачами за межами організації? Докладні відомості див. в статтях Надання користувачам [дозволу зв'язатися з зовнішніми користувачами Skype для бізнесу.](https://docs.microsoft.com/skypeforbusiness/set-up-skype-for-business-online/allow-users-to-contact-external-skype-for-business-users)

Ваші записи DNS необхідно настроїти правильно, щоб зовнішні зв'язки могло працювати належним чином. Перегляньте [тут,](https://docs.microsoft.com/microsoft-365/admin/get-help-with-domains/set-up-your-domain-host-specific-instructions) щоб дізнатися, чи є конкретні інструкції з налаштування записів DNS у свого постачальника послуг. 

Хочете дозволити користувачам Skype лише зв'язатися з користувачами Skype для бізнесу? Див. статтю

- [Дозволити користувачам Skype для бізнесу додавати контакти Skype.](https://docs.microsoft.com/skypeforbusiness/set-up-skype-for-business-online/let-skype-for-business-users-add-skype-contacts) 

- [Додайте контакт у "Skype для бізнесу"](https://support.office.com/article/add-a-contact-in-skype-for-business-89338023-2adf-4f5c-90b6-f8b6f72fadd1).


Якщо потрібно, щоб користувачі Teams могли зв'язатися з користувачами за [](https://docs.microsoft.com/microsoftteams/let-your-teams-users-communicate-with-other-people)межами вашої організації (зокрема, з користувачами "Skype для бізнесу"), ознайомтеся зі цією цією сторолою.
