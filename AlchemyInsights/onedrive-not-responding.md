# <a name="onedrive-not-responding"></a>OneDrive не відповідає

Для новостворених облікові записи користувачі повинні входу до служби Office 365 з власного рахунку та натисніть OneDrive, щоб вручну запустити надання OneDrive сайту.
  
Однак, якщо користувач натиснув вже OneDrive, персональний сайт не підготовлено протягом 24 годин, будь ласка, увійдіть квиток підтримки по допомогу.
