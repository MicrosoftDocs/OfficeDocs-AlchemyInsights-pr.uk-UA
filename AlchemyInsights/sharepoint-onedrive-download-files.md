# <a name="download-or-prevent-download-of-files-from-onedrive-or-sharepoint"></a>Завантаження та заборона завантаження файлів із OneDrive або SharePoint

- [Завантаження файлів і папок із OneDrive або SharePoint](https://support.office.com/article/Download-files-and-folders-from-OneDrive-or-SharePoint-5c7397b7-19c7-4893-84fe-d02e8fa5df05)
- [Блокування завантаження файлів за допомогою умовного доступу Azure AD](https://docs.microsoft.com/cloud-app-security/use-case-proxy-block-session-aad#create-a-block-download-policy-for-unmanaged-devices)
- [Пошук "завантажених файлів" подій у журналах аудиту](https://docs.microsoft.com/microsoft-365/compliance/search-the-audit-log-in-security-and-compliance?view=o365-worldwide#file-and-page-activities) для визначення того, хто завантажив файли
- [Максимальна швидкість передавання даних і завантаження](https://support.office.com/article/Maximize-upload-and-download-speed-8eeadfb8-501f-406d-997b-98ab6ff67f43)
- [Вирішення проблем із синхронізацією OneDrive](https://support.office.com/article/Fix-OneDrive-sync-problems-83ab0d8a-8400-45b0-8dcf-dc8aa8a6bcf8)
