# <a name="control-the-access-from-unmanaged-devices"></a>Керування доступом із некерованих пристроїв

Як SharePoint або Глобальний адміністратор у Microsoft 365 можна заблокувати або обмежити доступ до вмісту SharePoint і OneDrive з некерованих пристроїв. Докладні відомості наведено в статті [керування доступом із некерованих пристроїв](https://docs.microsoft.com/sharepoint/control-access-from-unmanaged-devices).
