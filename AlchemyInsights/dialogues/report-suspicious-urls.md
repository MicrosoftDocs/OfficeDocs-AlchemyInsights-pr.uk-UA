# <a name="report-suspicious-urls"></a>Повідомляти про підозрілі URL-адреси

Щоб надіслати підозрілі URL-адреси, виконайте наведені нижче дії.

1. У **поданні "Нове подання** " вкажіть повну URL-адресу, включно з протоколом (https://).
2. Якщо вибрати параметр **має бути заблоковано**, укажіть, чи є URL-адреса **фішинговим** або **зловмисним**.
3. Натисніть кнопку **Надіслати**.
