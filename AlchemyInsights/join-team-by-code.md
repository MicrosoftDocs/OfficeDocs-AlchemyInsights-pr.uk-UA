# <a name="join-a-team-by-code"></a>Приєднання до команди за кодом

Корпорація Майкрософт розуміє проблему, через яку користувачі не можуть створити приєднання до коду **групи** або приєднатися до команди за допомогою функції **JOIN by Code** . Як спосіб вирішення, власники групи можуть додавати користувачів за потреби вручну.
