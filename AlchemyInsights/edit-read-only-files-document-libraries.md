# <a name="edit-a-read-only-file"></a>Редагування файлу, доступний лише для читання

Ви можете виявити, що коли ви відкриваєте файли, вони відкриваються лише для читання. У деяких випадках це – додаткова безпека, наприклад, коли ви відкриваєте файли з Інтернету та інші часи, це може бути пов'язано з параметром, який можна змінити.

- [Файли SharePoint, відкриті лише для читання](https://docs.microsoft.com/sharepoint/troubleshoot/lists-and-libraries/files-open-as-read-only-and-cannot-check-in-or-out)
