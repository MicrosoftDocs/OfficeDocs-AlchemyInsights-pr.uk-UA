# <a name="add-or-remove-an-email-address-for-a-user"></a>Додавання або видалення адресу електронної пошти для користувача

[Відразу тут, щоб отримати роботу](https://portal.office.com/AdminPortal/Home#/AssistedGuide/addemailoptions).
    
 **Примітка**: ви можете мати більше одного адреси електронної пошти для користувача, але додаткові *псевдоніми* призначені для отримання електронної пошти. Якщо вам необхідно відправити електронне повідомлення з псевдоніма, створення [спільної поштової скриньки](https://support.office.com/article/871a246d-3acd-4bba-948e-5de8be0544c9). Таким чином ви отримаєте псевдонім для надсилання та отримання пошти.
