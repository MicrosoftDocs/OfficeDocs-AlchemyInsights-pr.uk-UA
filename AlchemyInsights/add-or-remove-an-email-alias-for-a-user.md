# <a name="add-or-remove-an-email-address-for-a-user"></a>Додавання або видалення адреси електронної пошти для користувача

Щоб почати роботу, виконайте наведені нижче [дії](https://portal.office.com/AdminPortal/Home#/AssistedGuide/addemailoptions) в центрі адміністрування Microsoft 365.

 **Зверніть увагу**: ви можете мати кілька адрес електронної пошти для користувача, але додаткові  *псевдоніми електронної пошти*  можна використовувати лише для отримання повідомлень електронної пошти. Якщо вам потрібно надіслати повідомлення електронної пошти з псевдонімом, створіть [спільну поштову скриньку](https://docs.microsoft.com/microsoft-365/admin/email/create-a-shared-mailbox). Таким чином, ви отримаєте псевдонім для обміну та отримання пошти.
