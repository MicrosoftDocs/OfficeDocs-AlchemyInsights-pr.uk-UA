# <a name="47500-server-busy-please-try-again-later"></a>4.7.500 Server зайнятий, повторіть спробу пізніше

Ця помилка виникає, коли обсяг трафіку електронної пошти з вихідної IP-адреси перевищує обмеження на основі репутації (або відсутності репутації) вихідної IP-адреси.

Блокування електронної пошти з вихідної IP-адреси завершується протягом години. Якщо вихідна IP-адреса – це локальний сервер електронної пошти, який вам належить, Перевірте конфігурацію з'єднувача передавання пошти. Якщо поведінка триває більше години, зверніться до служби підтримки, щоб запросити виняток для вихідної IP-адреси.
