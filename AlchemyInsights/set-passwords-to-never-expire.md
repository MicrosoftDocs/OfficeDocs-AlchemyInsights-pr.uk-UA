# <a name="set-passwords-to-never-expire"></a>Встановлювати паролі ніколи не закінчується

Встановити пароль для ніколи не втратять актуальності з [безпеки &amp; конфіденційності](https://portal.office.com/adminportal/home#/settings/security). Натисніть кнопку "Редагувати" поруч із пунктом пароль політики і звернутися до слайдера "на".
  
Для більш детальної інформації дивіться: [набір політики терміну дії пароля для вашої організації](https://support.office.com/article/set-the-password-expiration-policy-for-your-organization-0f54736f-eb22-414c-8273-498a0918678f).
