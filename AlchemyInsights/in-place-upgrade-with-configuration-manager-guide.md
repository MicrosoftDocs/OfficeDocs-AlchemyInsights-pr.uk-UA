# <a name="in-place-upgrade-with-configuration-manager-guide"></a>Оновлення на місці за допомогою посібника із керування конфігураціями

Процес оновлення на місці зберігає всі дані, настройки, програми та драйвери з наявної версії операційної системи. Він розроблений, щоб бути надзвичайно надійним і має можливість автоматично повернутися до попередньої операційної системи, якщо виникають проблеми.

Під час оновлення пристроїв під керуванням ОС Windows 7 і Windows 8,1 до найновішої версії Windows 10 використовуйте [посібник з настроювання на місці](https://admin.microsoft.com/adminportal/home#/win10upgrade) . Ви будете використовувати сценарій, наданий для перевірки попередніх вимог і автоматичного настроювання оновлення на місці.
