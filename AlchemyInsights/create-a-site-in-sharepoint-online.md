# <a name="create-a-site"></a>Створення сайту

У SharePoint Online можна створити різних типів сайтів. Щоб дізнатися, як для цього нового центру адміністрування SharePoint, див. [Створення сайту](https://go.microsoft.com/fwlink/?linkid=866295). Інформація про дозволяючи користувачам створювати свої власні сайти в розділі [керування створення сайту в SharePoint Online](https://go.microsoft.com/fwlink/?linkid=866296).
 
Для додаткової інформації про нових типів сайтів переглянути [Група пов'язаних сайтів груп](https://go.microsoft.com/fwlink/?linkid=866292) або [зв'язку сайтів](https://go.microsoft.com/fwlink/?linkid=866294).
