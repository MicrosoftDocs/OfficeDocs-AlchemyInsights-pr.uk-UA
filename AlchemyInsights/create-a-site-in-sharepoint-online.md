# <a name="create-a-site"></a>Створення сайту

Ви можете створити різні типи сайтів у службі SharePoint Online. Щоб дізнатися, як це зробити в новому центрі адміністрування SharePoint, перегляньте статтю [Створення сайту](https://go.microsoft.com/fwlink/?linkid=866295). Щоб отримати відомості про надання користувачам можливості створювати власні сайти, перегляньте статтю [керування створенням сайту в службі SharePoint Online](https://go.microsoft.com/fwlink/?linkid=866296).
 
Щоб отримати відомості про нові типи сайтів, перегляньте статтю [Група підключених сайтів](https://go.microsoft.com/fwlink/?linkid=866292) групи або [сайти для спілкування](https://go.microsoft.com/fwlink/?linkid=866294).
