# <a name="desktop-app-for-sharepoint"></a>Застосунок для настільних комп'ютерів SharePoint

Ваша організація вводиться в приватну програму попереднього перегляду для настільних додатків для SharePoint.

Наразі корпорація Майкрософт не підтримує програму робочого стола для SharePoint (приватний перегляд). Для отримання допомоги у програмі SharePoint для настільних комп'ютерів використовуйте власний попередній перегляд партнерів корпорації Майкрософт.
