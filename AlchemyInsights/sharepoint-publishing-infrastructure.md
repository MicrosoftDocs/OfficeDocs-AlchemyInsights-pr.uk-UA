# <a name="sharepoint-publishing-infrastructure"></a>Публікації інфраструктури SharePoint


У SharePoint передбачено класичний набір функцій, які дають змогу Створюйте та публікуйте багатих веб-сторінок для інтрамережі. Активувати SharePoint публікації інфраструктури на рівні колекції сайтів щоб увімкнути їх.

Для отримання додаткової інформації, будь ласка, відвідайте нижче посиланнями.

[Увімкнути засобами публікації](https://support.office.com/article/Enable-publishing-features-479677A6-8B33-4AC7-907D-071C1C7E4518)

[Засобами публікації](https://support.office.com/article/Features-enabled-in-a-SharePoint-Online-publishing-site-3AB3810C-3C2C-4361-9D0E-0CBE666EA0B0?wt.mc_id=O365_Portal_MMaven#__toc336865553)

Примітки:

- Класичний Видавничий шаблони сайтів включають засоби публікування за промовчанням.

- Класичний засобами публікації не надаються на сучасних сайтів. Сучасні сайти надають нові альтернативні за промовчанням.
