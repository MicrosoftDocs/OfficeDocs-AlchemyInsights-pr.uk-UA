# <a name="sharepoint-publishing-infrastructure"></a>Видавнича інфраструктура SharePoint


SharePoint містить класичний набір функцій, які дають змогу автору та публікувати багаті веб-сторінки для інтрамережі. Активуйте інфраструктуру публікування SharePoint на рівні колекції сайтів, щоб увімкнути їх.

Для отримання додаткової інформації, будь ласка, відвідайте посилання нижче.

[Увімкнути функції публікації](https://support.office.com/article/Enable-publishing-features-479677A6-8B33-4AC7-907D-071C1C7E4518)

[Функції публікації](https://support.office.com/article/Features-enabled-in-a-SharePoint-Online-publishing-site-3AB3810C-3C2C-4361-9D0E-0CBE666EA0B0?wt.mc_id=O365_Portal_MMaven#__toc336865553)

Нотатки:

- Класичні шаблони публікацій сайту містять функції публікації за промовчанням.

- Класичні функції публікації недоступні на сучасних сайтах. Сучасні сайти надають нову альтернативу за замовчуванням.
