# <a name="sharepoint-publishing-infrastructure"></a>Публікації інфраструктури SharePoint


<p>У SharePoint передбачено класичний набір функцій, які дають змогу Створюйте та публікуйте багатих веб-сторінок для інтрамережі. Активувати SharePoint публікації інфраструктури на рівні колекції сайтів щоб увімкнути їх.</p>  <p>Для отримання додаткової інформації, будь ласка, відвідайте нижче посиланнями.</p>  <ul>  <li><a href="https://support.office.com/en-us/article/Enable-publishing-features-479677A6-8B33-4AC7-907D-071C1C7E4518">Увімкнути засобами публікації</a></li>  <li><a href="https://support.office.com/en-us/article/Features-enabled-in-a-SharePoint-Online-publishing-site-3AB3810C-3C2C-4361-9D0E-0CBE666EA0B0?wt.mc_id=O365_Portal_MMaven#__toc336865553">Засобами публікації</a></li>  </ul>  <p>Примітки:</p>  <ul>  <li>Класичний Видавничий шаблони сайтів включають засоби публікування за промовчанням.</li>  <li>Класичний засобами публікації не надаються на сучасних сайтів. Сучасні сайти надають нові альтернативні за промовчанням.</li>  </ul>
