# <a name="use-microsoft-graph-api-to-manage-groups"></a>Керування групами за допомогою API Microsoft Graph

Групи – це колекції користувачів, а також інші керівники, які поділяють доступ до ресурсів у службах Microsoft Services або в програмі. Microsoft Graph надає інтерфейси API, які можна використовувати для створення різних типів груп і функцій групи відповідно до сценаріїв. Усі операції, пов'язані з групами в Microsoft Graph, потребують згоди адміністратора. Докладні відомості наведено в статті [Робота з групами в Microsoft Graph](https://docs.microsoft.com/graph/api/resources/groups-overview).
