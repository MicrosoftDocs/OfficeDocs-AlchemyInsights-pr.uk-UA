# <a name="change-payment-method"></a>Змінити спосіб оплати

Ви можете перевірити, спосіб оплати, який налаштовано в Microsoft Outlook 365 центру адміністрування. Виберіть підписку, який потрібно перевірити. Спосіб оплати, який настроєно для підписки відображається в розділі **виставлення рахунків**. 
  
1. У центр адміністрування, перейдіть на **платіжний** \> [продукції & послуг](https://go.microsoft.com/fwlink/p/?linkid=842054) сторінки.

2. Поруч із методу платежу для підписки, який потрібно змінити метод платежу виберіть **змінити**.

3. Дотримуйтесь інструкцій на екрані, щоб завершити зміну.

Крок за кроком інструкції, див: [Зміна методу платежу для Office 365 для бізнесу](https://docs.microsoft.com/office365/admin/subscriptions-and-billing/change-payment-method)
