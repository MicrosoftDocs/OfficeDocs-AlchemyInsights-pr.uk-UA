# <a name="change-payment-method"></a>Змінення методу оплати

Спосіб оплати можна змінити з **центру адміністрування Microsoft 365**.
  
1. У центрі адміністрування перейдіть до розділу продукти для **виставлення рахунків** \> [& послуги](https://go.microsoft.com/fwlink/p/?linkid=842054).

2. Знайдіть підписку, для якої потрібно змінити спосіб оплати. Біля способу оплати натисніть **змінити**.

3. Щоб завершити зміну, виконайте вказівки на екрані.

Щоб дізнатися більше, перегляньте, як [змінити спосіб оплати для Microsoft 365 для бізнесу](https://docs.microsoft.com/office365/admin/subscriptions-and-billing/change-payment-method).
