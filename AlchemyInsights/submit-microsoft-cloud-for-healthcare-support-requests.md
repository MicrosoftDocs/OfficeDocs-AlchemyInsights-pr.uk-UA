# <a name="submit-microsoft-cloud-for-healthcare-support-requests"></a>Надсилання запрошень до служби підтримки Microsoft Cloud для медичної допомоги

Служба Microsoft Cloud for Healthcare для клієнтів надає своїм квитком через новий [Центр адміністрування Power Platform.](https://admin.powerplatform.microsoft.com/support?newTicket&product=Flow)

Новий портал надає більш прямий шлях до підтримки, оптимізованої для хмарної служби Microsoft для охорони здоров'я.
