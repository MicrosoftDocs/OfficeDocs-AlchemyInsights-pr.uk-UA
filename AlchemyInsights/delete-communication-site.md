# <a name="delete-a-communication-site"></a>Видалити повідомлення сайту

1. Перейти до веб-сайту.
    
2. Клацніть піктограму Параметри у правому верхньому кутку.
    
3. Натисніть кнопку **Параметри сайту**. У розділі **Сайту інформація**натисніть кнопку **видалити цей сайт**і натисніть кнопку **Видалити**.
