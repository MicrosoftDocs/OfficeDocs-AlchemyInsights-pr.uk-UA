# <a name="get-started-with-microsoft-search"></a>Початок роботи з Microsoft Search

Використовуйте Microsoft Search, щоб знаходити користувачів, файли, організаційні діаграми, сайти, відповіді на поширені запитання та інший вміст, який може допомогти виконувати завдання в програмі, яку ви використовуєте.

Докладні відомості наведено в статті [Огляд Microsoft Search (пошук](https://docs.microsoft.com/microsoftsearch/overview-microsoft-search)).
