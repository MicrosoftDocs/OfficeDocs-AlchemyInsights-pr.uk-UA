# <a name="outlook-sync-status-bar-remains-at-399-gb"></a>У рядку стану синхронізації Outlook залишається на 3,99 ГБ

Спробуйте зменшити розмір вмісту папки або створити фільтр синхронізації для великих папок на або біля 4 ГБ.

Див стан рядка стану ніколи не показує більше 3,99 Гб, що залишилися на початкової [синхронізації великих папок для отримання додаткових відомостей](https://support.microsoft.com/help/2738323/status-bar-progress-never-shows-more-than-3-99-gb-remaining-on-initial).
