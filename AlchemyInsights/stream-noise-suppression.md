# <a name="stream-noise-suppression"></a>Придушення потокового шуму

Придушення шуму, коли ввімкнено власниками відео, дозволяє глядачам ізолювати мовлення з інших звуків і музики під час відтворення. Придушення шуму можна вмикати або вимикати для кожного щойно завантаженого відео. Для отримання додаткової інформації див [придушення шуму](https://docs.microsoft.com/stream/noise-suppression).
