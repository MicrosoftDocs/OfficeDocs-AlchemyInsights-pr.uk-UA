# <a name="calendar-freebusy-issues-in-hybrid-environments"></a>Календар зайнятість проблеми у гібридному середовищі

Якщо користувачі виникають проблеми з календарем відомості про зайнятість у змішаний середовищах, знову запустіть майстер гібридного настроювання. Щоб отримати додаткові відомості про майстер гібридного [Настроювання див.](https://go.microsoft.com/fwlink/p/?linkid=528149)

Для виправлення неполадок у гібридному середовищі перегляньте [проблеми з гібридним розгортанням](https://technet.microsoft.com/library/jj659053.aspx).
