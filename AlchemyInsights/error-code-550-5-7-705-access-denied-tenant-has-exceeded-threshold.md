# <a name="550-57705-access-denied-tenant-has-exceeded-threshold"></a>550 5.7.705 відмовлено в доступі, орендар перевищив поріг

Це повідомлення з'являється, коли занадто багато спаму або об'ємних пошти надіслано організації Office 365. Щоб видалити цей блок, потрібно зрозуміти і пояснити причину підтримки агент, а також пов'язану з нею неполадку виправлення.
