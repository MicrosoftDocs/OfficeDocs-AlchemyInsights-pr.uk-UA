# <a name="550-57705-access-denied-tenant-has-exceeded-threshold"></a>550 5.7.705 доступ заборонено, орендар перевищує граничне значення

Це повідомлення виникає, коли ваша організація надіслала забагато спаму або розсилок, і ми розміщуєте блок на вихідній пошті.
Типові причини скомпрометовані локальних серверів або скомпрометовані облікові записи адміністратора, які використовувалися для створення роз'ємів. Будь-яка умова може дозволити спаму проходити через вашу організацію.

Щоб видалити цей блок, потрібно зрозуміти і пояснити причину до агента підтримки, а також виправити основну проблему.
Натисніть **зв'язатися з нами** нижче, коли ви будете готові.
