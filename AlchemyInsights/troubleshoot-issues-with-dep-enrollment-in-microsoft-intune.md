# <a name="troubleshoot-issues-with-dep-enrollment-in-microsoft-intune"></a>Усунення несправностей із реєстрацією DEP в Microsoft InTune

Перегляньте наведені нижче ресурси, щоб вирішити проблему.
  
1. Якщо для пристрою DEP не вдається зареєструвати та (багатофакторну автентифікацію), вимкніть службу МЗС. Зараз МЗС не підтримується для реєстрації DEP

2. Використовуйте [портал виправлення неполадок](https://devicemanagement.microsoft.com/#blade/Microsoft_Intune_DeviceSettings/TroubleshootBlade) , щоб діагностувати та усунути поширені помилки в реєстрації. Перегляньте [цей документ](https://docs.microsoft.com/intune/help-desk-operators) , щоб отримати докладні відомості.

3. Перегляньте ці документи для списку поширених помилок, які перешкоджають виконанню та резолюціям кожного з них: [посібник із виправлення неполадок](https://support.microsoft.com/help/4039809/troubleshooting-ios-device-enrollment-in-intune) і [Виправлення неполадок, пов'язаних із документами](https://docs.microsoft.com/intune-classic/troubleshoot/troubleshoot-device-enrollment-in-intune)

4. [Дізнайтеся про програму реєстрації пристроїв](https://docs.microsoft.com/intune/device-enrollment-program-enroll-ios).
