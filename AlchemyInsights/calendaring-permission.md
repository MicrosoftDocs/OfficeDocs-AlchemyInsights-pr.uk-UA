# <a name="calendar-permissions"></a>Дозволи календаря

Якщо у вас виникли проблеми з доступом до календаря або інших календарів, скористайтеся [помічником зі служби підтримки Microsoft і відновленням microsoft 365](https://diagnostics.office.com/).
  
Щоб дізнатися більше про те, як ділитися календарем, перегляньте [спільний доступ до календаря Outlook з іншими людьми](https://support.office.com/article/353ed2c1-3ec5-449d-8c73-6931a0adab88.aspx).
