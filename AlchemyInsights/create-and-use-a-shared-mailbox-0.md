# <a name="create-and-use-a-shared-mailbox"></a>Створення та використання спільної поштової скриньки

Спільної поштової скриньки дає змогу кільком надсилати й отримувати електронну пошту для псевдонім, як **sales@contoso.com**, без призначення іншого ліцензії.
  
[Створення спільної поштової скриньки](https://support.office.com/article/Create-a-shared-mailbox-871a246d-3acd-4bba-948e-5de8be0544c9) пояснює, як: 
  
- Створення спільної поштової скриньки
    
- Надати користувачам дозволи на використання спільної поштової скриньки
    
- Доступ до спільної поштової скриньки в Outlook або в Outlook в Інтернеті
