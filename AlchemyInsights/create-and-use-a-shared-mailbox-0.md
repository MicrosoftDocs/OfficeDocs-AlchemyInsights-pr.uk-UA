# <a name="create-and-use-a-shared-mailbox"></a>Створення та використання спільної поштової скриньки

Спільна поштова скринька дозволяє більше однієї людини надсилати та отримувати повідомлення електронної пошти для псевдоніма, наприклад **Sales@contoso.com**, без призначення іншої ліцензії.
  
[Створення спільної поштової скриньки](https://docs.microsoft.com/office365/admin/email/create-a-shared-mailbox) пояснює, як: 
  
- Створення спільної поштової скриньки
    
- Надання користувачам дозволів на використання спільної поштової скриньки
    
- Доступ до спільної поштової скриньки в Outlook або Outlook, в Інтернеті
