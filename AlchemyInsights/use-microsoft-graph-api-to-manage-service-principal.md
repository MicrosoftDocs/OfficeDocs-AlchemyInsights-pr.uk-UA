# <a name="use-microsoft-graph-api-to-manage-service-principal"></a>Використання API Microsoft Graph для керування головним елементом служби

Використовуючи [основний API служби](https://docs.microsoft.com/graph/api/resources/serviceprincipal), можна програмно керувати екземплярами програм і керувати тим, що може виконувати програма в клієнті.

Щоб отримати докладні відомості про зв'язок між програмами та службами служб, ознайомтеся [з програмами та основними об'єктами служби в Лазурому Active Directory](https://docs.microsoft.com/azure/active-directory/develop/app-objects-and-service-principals).
