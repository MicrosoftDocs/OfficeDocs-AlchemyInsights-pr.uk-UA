# <a name="get-started-with-the-developer-tools-in-microsoft-edge-chromium"></a>Початок роботи з інструментами розробника в Microsoft EDGE (хром)

Microsoft EDGE прийняв проект з відкритим вихідним кодом хрому, який має на меті зробити веб-платформу більш сумісними та веб-платформами. Ця зміна надасть вам змогу використовувати Microsoft EDGE для створення та тестування веб-сайтів, які працюють на інших браузерах на основі хрому.

Microsoft EDGE (хром) також надає [кращий у своєму класі засоби розробника](https://go.microsoft.com/fwlink/?linkid=2134941) , які мають бути знайомі з вами та сумісні з інструментами, які ви вже використовуєте.

Щоб дізнатися більше, Дізнайтеся про [нові можливості DevTools Microsoft EDGE (хром)](https://go.microsoft.com/fwlink/?linkid=2135020).
