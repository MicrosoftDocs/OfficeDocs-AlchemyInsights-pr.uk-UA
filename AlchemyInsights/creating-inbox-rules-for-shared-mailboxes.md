# <a name="creating-inbox-rules-for-shared-mailboxes"></a>Створення правил для папки «Вхідні» для спільних поштових скриньок

Правила можна додати до спільної поштової скриньки, у якій ви маєте доступ подібним чином до способу Додавання правил до власного облікового запису.
  
Переконайтеся, що ви є членом спільної поштової скриньки, під питанням і виконайте дії, описані в цій статті: [додати правила до спільної поштової скриньки](https://support.office.com/article/b0963400-2a51-4c64-afc7-b816d737d164)
