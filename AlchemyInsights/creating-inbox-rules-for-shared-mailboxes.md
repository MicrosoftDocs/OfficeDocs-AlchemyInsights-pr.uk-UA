# <a name="creating-inbox-rules-for-shared-mailboxes"></a>Створити правила для папки «Вхідні» для спільної поштової скриньки

Можна додати правила на спільну поштову скриньку ви маєте доступ до подібним чином як додати правила до власного рахунку.
  
Переконайтеся, що ви належите спільної поштової скриньки в питанні а потім виконайте дії, описані в цій статті: [додати правила для спільної поштової скриньки](https://support.office.com/article/b0963400-2a51-4c64-afc7-b816d737d164)
