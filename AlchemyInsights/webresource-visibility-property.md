# <a name="webresource-visibility-property"></a>Властивість видимості веб-ресурсу

Слід вибрати параметр **видимий за промовчанням** для веб-ресурсу, який відображатиметься у формі, і бути доступним для користувачів. Докладніші відомості наведено в [властивостях веб-ресурсу для форм програм на основі моделі](https://docs.microsoft.com/powerapps/maker/model-driven-apps/web-resource-properties-legacy).
