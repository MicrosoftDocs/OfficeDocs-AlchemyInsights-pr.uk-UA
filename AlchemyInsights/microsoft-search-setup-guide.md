# <a name="microsoft-search-setup-guide"></a>Посібник із настроювання служби пошуку Microsoft

За допомогою [посібника з настроювання Microsoft Search](https://go.microsoft.com/fwlink/?linkid=2153798) можна настроїти та настроїти службу пошуку Microsoft Search. Якщо ви хочете, щоб він експериментів до групи користувачів або розкаже її всім у вашій організації. Ви призначите адміністраторам пошуку та пошукові редактори, а потім Настроюйте пошуковий запит для користувачів, які мають відповіді та додаткові параметри, наприклад додавання розширення Bing до Chrome або настроювання Bing як пошукової системи за замовчуванням.
