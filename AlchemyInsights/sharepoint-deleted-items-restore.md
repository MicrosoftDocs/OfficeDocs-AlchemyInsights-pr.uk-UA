# <a name="restore-items-deleted-from-sharepoint"></a>Відновити об'єкти, видалені із SharePoint

При видаленні елемента або сайт з Sharepoint, вона isn&rsquo;t негайно вилучені. Видалені елементи потрапляють до кошика для певного періоду часу. За цей час можна буде відновити елементи, які ви видалили на колишнє місце. Для отримання додаткової інформації будь ласка, відвідайте нижче посиланнями.</p> <p><strong>Відновити видалені файли, папки та сайти з кошика.</strong></p> <ul> <li style="font-weight: 400;"><a href="https://support.office.com/en-us/article/restore-deleted-items-from-the-site-collection-recycle-bin-5fa924ee-16d7-487b-9a0a-021b9062d14b?ui=en-US&amp;rs=en-US&amp;ad=US">Відновити елементи з кошика сайту SharePoint</a></li> </ul> <ul> <li style="font-weight: 400;"><a href="https://support.office.com/en-us/article/Restore-deleted-files-or-folders-in-OneDrive-949ada80-0026-4db3-a953-c99083e6a84f">Відновити видалені файли або папки в OneDrive</a></li> </ul> <ul> <li style="font-weight: 400;"><a href="https://docs.microsoft.com/sharepoint/restore-deleted-site-collection">Відновлення видаленої вузлів (у тому числі групи, зв'язку та інших сайтів)</a></li> </ul> <ul> <li style="font-weight: 400;"><a href="https://docs.microsoft.com/en-us/onedrive/restore-deleted-onedrive">Відновлення видаленої сайт OneDrive</a></li> </ul> <p>Для оптових recycle bin дії, адміністратори можуть розглянути можливість використання <a href="https://docs.microsoft.com/en-us/powershell/sharepoint/sharepoint-pnp/sharepoint-pnp-cmdlets?view=sharepoint-ps">Sharepoint онлайн PNP.</a></p> <p><strong>Функція відновлення файлів</strong></p> <p>Якщо багато OneDrive або Sharepoint файлів отримати видалені, перезаписані, пошкоджена або заражені шкідливим, всю медіатеку OneDrive або Sharepoint можна відновити попередній час&nbsp;за допомогою функції відновлення файлів.</p> <ul> <li><a href="https://support.office.com/en-us/article/restore-your-onedrive-fa231298-759d-41cf-bcd0-25ac53eb8a15">Відновити OneDrive бібліотека</a></li> <li><a href="https://support.office.com/en-us/article/restore-a-document-library-317791c3-8bd0-4dfd-8254-3ca90883d39a?ui=en-US&amp;rs=en-US&amp;ad=US">Відновити бібліотеки документів</a></li> </ul> <p><strong>Примітка:</strong> SharePoint Online зберігає резервні копії всіх контенту на 14 додаткових днів за межами фактичної видалення. Якщо відновити файли або вміст не можна відновити за допомогою кошика, адміністратор можете зв'язатися з підтримки Microsoft просити відновити будь-який час всередині вікна 14 днів. &nbsp;Реставрацій резервні копії може бути завершено лише для колекцій сайтів або дочірніх сайтів, не для певних файлів, списків і бібліотек.</p>
