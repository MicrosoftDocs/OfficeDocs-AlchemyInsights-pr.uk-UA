# <a name="sharepoint-online-throttling"></a>Регулювання SharePoint Online

SharePoint Online використовує дроселювання для підтримки оптимальної продуктивності та надійності служби SharePoint Online. Дроселювання обмежує кількість дій користувача або одночасних викликів (за сценарієм або кодом) для запобігання надмірного використання ресурсів. Якщо ви отримаєте обмежив, 99% від часу, що це з-за користувацького коду.
  
Щоб отримати додаткові відомості про дроселювання бачити, [не отримати обмежено або заблоковано в SharePoint Online](https://go.microsoft.com/fwlink/?linkid=2022019)
