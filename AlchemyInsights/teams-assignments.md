# <a name="teams-assignments"></a>Призначення команд

Завдання — завдання або одиниці роботи, призначені для студента або члена команди в класі в рамках їх вивчення. Можна створювати призначення в класі teams.

- Щоб дізнатися більше про призначення, ознайомтеся з [призначеннями Microsoft teams](https://support.microsoft.com/en-us/office/microsoft-teams-5aa4431a-8a3c-4aa5-87a6-b6401abea114#ID0EAABAAA=Assignments).
- Щоб дізнатися більше про керування призначеннями функцій у центрі адміністрування Microsoft teams, перегляньте [призначення в групах для освіти](https://docs.microsoft.com/microsoftteams/expand-teams-across-your-org/assignments-in-teams).
