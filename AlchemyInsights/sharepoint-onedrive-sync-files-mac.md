# <a name="install-setup-and-sync-onedrive-or-sharepoint-files-on-mac"></a>Інсталяція, настроювання та Синхронізація файлів OneDrive або SharePoint на комп'ютері Mac 

Встановити, налаштувати і синхронізувати OneDrive або SharePoint файли на Mac з інструкціями з [синхронізації файлів з клієнтом OneDrive синхронізації на Mac OS X](https://support.office.com/article/sync-files-with-the-onedrive-sync-client-on-mac-os-x-d11b9f29-00bb-4172-be39-997da46f913f).

Якщо програму OneDrive інстальовано та налаштовано на комп'ютері Mac, але користувачі відчувають проблеми з синхронізацією, перегляньте проблеми [з виправленням синхронізації з OneDrive (Mac)](https://support.office.com/article/fix-onedrive-sync-problems-on-a-mac-af3012d7-13ec-4ac9-bbb1-ebcd2a0cd756).

> [!NOTE]
> Відомі папки переміщення функції OneDrive синхронізації для ОС Windows недоступна для Mac.
