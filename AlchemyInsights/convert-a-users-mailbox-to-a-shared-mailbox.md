# <a name="convert-a-user-mailbox-to-a-shared-mailbox"></a>Конвертувати поштової скриньки користувача для спільної поштової скриньки

Під час перетворення окремого користувача поштову скриньку для спільної поштової скриньки, всі існуючі електронної пошти та елементи календаря зберігаються в архівній де кілька користувачів можуть отримати доступ до спільної поштової скриньки. Можна також перетворювати спільної поштової скриньки до поштової скриньки користувача (приватне).
