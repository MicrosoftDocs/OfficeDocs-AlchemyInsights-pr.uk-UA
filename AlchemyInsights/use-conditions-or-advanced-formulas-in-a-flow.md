# <a name="use-conditions-or-advanced-formulas-in-a-flow"></a>Використання умов або розширених формул у потоці

У Microsoft Flow можна настроїти потік, щоб виконувати завдання лише в тому випадку, якщо умова має значення TRUE. Докладні дані наведено в статті [Додавання стану до потоку](https://go.microsoft.com/fwlink/?linkid=872112).
  
Щоб створити розширену формулу, ознайомтеся [з мовною схемою "визначення робочого циклу" для програм "блакитні логічні програми](https://aka.ms/logicexpressions)".
