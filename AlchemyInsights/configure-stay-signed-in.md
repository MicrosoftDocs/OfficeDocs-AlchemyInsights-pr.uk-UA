# <a name="configure-stay-signed-in-for-accounts"></a>Настроювання входу для облікових записів

Не виходити (KMSI) відображає запит Не **виходити?** після успішного входу користувача. Якщо користувач **відповість** на цей запит так, служба KMSI надає йому постійний маркер оновлення. 

Докладні відомості про налаштування KMSI див. в запиті ["Не виходити?" для облікових записів Azure AD.](/azure/active-directory/fundamentals/keep-me-signed-in)
