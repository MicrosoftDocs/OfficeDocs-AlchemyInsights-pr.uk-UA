# <a name="offboarding-machines-from-the-microsoft-defender-atp-service"></a>Offboarding машини в службі "захисник" служби Microsoft

Якщо вам потрібна допомога з надання допомоги в автономному режимі, зверніться до інструкції для своєї операційної системи:  

- Для пристроїв із Windows 10 ознайомтеся [з пристроями offboard з служби "служба захисника" корпорації Майкрософт](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/offboard-machines#offboard-windows-10-devices).
- Для пристроїв, які не використовують Windows, перегляньте статтю [offboard, не з ОС Windows](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/configure-endpoints-non-windows#offboard-non-windows-devices).
