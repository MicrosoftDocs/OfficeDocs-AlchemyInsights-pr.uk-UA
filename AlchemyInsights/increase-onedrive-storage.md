# <a name="increase-onedrive-storage"></a>Збільшення обсягу сховища "OneDrive"

Як глобальний або адміністратор SharePoint у Microsoft 365, ви можете встановити місце для сховища для певного користувача безпосередньо в центрі адміністрування Microsoft.  

Докладні відомості наведено в статті [змінення простору сховища певного користувача](https://docs.microsoft.com/onedrive/change-user-storage).
