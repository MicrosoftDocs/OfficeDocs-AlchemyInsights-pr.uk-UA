# <a name="permissions-in-microsoft-stream"></a>Дозволи в Microsoft Stream

За замовчуванням під час передавання відео можна настроїти дозволи для відео, яке відображатиметься для всієї компанії. Якщо потрібно вказати дозволи для певних користувачів або груп, можна настроїти дозволи на відео.

- [Дозволи та конфіденційність у Microsoft Stream](https://docs.microsoft.com/stream/portal-permissions)

- [Дозволи групи](https://docs.microsoft.com/stream/portal-permissions#group-permissions)

- [Дозволи каналу](https://docs.microsoft.com/stream/portal-permissions#channel-permissions)
