# <a name="control-sync-throughput"></a>Керування пропускною здатністю синхронізації

Якщо вам потрібно керувати трафіком для синхронізації додатків, радимо використовувати політику обслуговування мережі (QoS) або політику QoS для Windows, якщо це можливо. Для отримання відомостей про настройки мережі, які можна зробити доступними для користувачів, див [.](https://support.office.com/article/71cc69da-2371-4981-8cc8-b4558bdda56e)
