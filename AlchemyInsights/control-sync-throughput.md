# <a name="control-sync-throughput"></a>Керування процінанням синхронізації

Якщо потрібно керувати трафіком програми синхронізації, радимо використовувати політики якості мережі (QoS) або політики Windows QoS, коли це можливо. Відомості про параметри мережі, які можна зробити доступними для користувачів, див. в розділі Змінення швидкості передавання або завантаження програми синхронізації [OneDrive.](https://support.office.com/article/71cc69da-2371-4981-8cc8-b4558bdda56e)
