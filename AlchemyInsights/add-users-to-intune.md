# <a name="add-users-to-intune"></a>Додавання користувачів до InTune

Ви можете вручну додати користувачів до свого передплатою, використовуючи [Центр адміністрування Microsoft 365](https://admin.microsoft.com/)   або [портал inune](https://portal.azure.com/#blade/Microsoft_Intune_DeviceSettings/ExtensionLandingBlade/overview).

Адміністратор може редагувати облікові записи користувачів, щоб призначати ліцензії на Inune. Ліцензії можна призначати в центрі адміністрування Microsoft 365 або на порталі Inune.

Докладні відомості про використання центру адміністрування Microsoft 365 наведено [в статті додавання користувачів окремо або навалом до центру адміністрування microsoft 365](https://support.office.com/article/Add-users-individually-or-in-bulk-to-Office-365-Admin-Help-1970f7d6-03b5-442f-b385-5880b9c256ec).

Щоб отримати докладні відомості про використання порталу Inune, перегляньте статтю [Add Users (додавання користувачів) і надайте дозволи для адміністрування для inune](https://docs.microsoft.com/intune/fundamentals/users-add).
