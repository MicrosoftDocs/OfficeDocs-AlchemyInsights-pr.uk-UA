# <a name="need-help-with-microsoft-365-business-premium-voice"></a>Потрібна допомога з Microsoft 365 бізнес Premium голосом?

Microsoft 365 бізнес Premium Voice полегшує для малих і середніх організацій, щоб перетворити Microsoft teams на потужну і гнучку телефонну систему. Щоб отримати додаткові відомості про бізнес-голос, подивіться, [що таке Microsoft 365 бізнес Premium голосом?](https://docs.microsoft.com/microsoftteams/business-voice/whats-business-voice)

Якщо ви вже придбали Microsoft 365 бізнес преміум голосом, [майстер початок роботи](https://docs.microsoft.com/microsoftteams/business-voice/use-getting-started-wizard) може допомогти вам отримати все відбувається.
