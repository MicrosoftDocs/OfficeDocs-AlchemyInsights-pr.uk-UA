# <a name="manage-synchronized-user-accounts"></a>Керування обліковими записами користувачів синхронізовані

Як правило, ви використовуєте Microsoft Outlook 365 центру адміністрування додавати та змінювати облікові записи. Тим не менш, неможливо використовувати його якщо увімкнуто реєстр sychronization або єдиного входу. Замість цього, керування користувачі з Windows PowerShell, як описано в [розділі Настроювання властивостей облікового запису користувача з Office 365 PowerShell](https://docs.microsoft.com/office365/enterprise/powershell/configure-user-account-properties-with-office-365-powershell ).
