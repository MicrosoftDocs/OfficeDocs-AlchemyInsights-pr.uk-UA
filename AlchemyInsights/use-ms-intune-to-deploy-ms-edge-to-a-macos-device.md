# <a name="use-microsoft-intune-to-deploy-microsoft-edge-to-a-macos-device"></a>Використання Microsoft Inune для розгортання Microsoft EDGE до пристрою macOS

Щоб розгорнути, настроїти, відстежувати та захистити програму на пристрої macOS, потрібно додати програму до Microsoft Inune. У програмі має бути тип, підтримуваний програмою Inune, а також варіанти керування, доступні в програмі Inune, залежать від типу програми. Використання функції Inune також означає, що вам не доведеться використовувати засіб упакування програм macOS.

Компанія підтримує Microsoft EDGE. Докладні відомості можна знайти в статті [Додавання Microsoft EDGE до macOS пристроїв за допомогою Microsoft Inune](https://go.microsoft.com/fwlink/?linkid=2134949).
