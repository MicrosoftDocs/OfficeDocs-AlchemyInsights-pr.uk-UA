# <a name="long-running-script"></a>Довгий запуск сценарію

Ми знаємо про проблеми з "тривалим сценарієм роботи" в Outlook.com. Щоб вирішити цю проблему, скористайтеся бета-версією Outlook.com.
  
[Дізнайтеся більше про бета-версію Outlook.com.](https://support.office.com/article/40676ad0-c831-45ac-a023-5be633be798d?wt.mc_id=Office_Outlook_com_Alchemy)
