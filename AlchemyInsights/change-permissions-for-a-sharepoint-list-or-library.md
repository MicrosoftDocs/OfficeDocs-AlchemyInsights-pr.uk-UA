# <a name="change-permissions-for-a-sharepoint-list-or-library"></a>Змінення дозволів для списку або бібліотеки SharePoint

1. Перейдіть на сайт SharePoint у браузері та відкрийте список або бібліотеку.
    
2. Клацніть піктограму настройки у верхньому правому куті, а потім виберіть пункт **настройки бібліотеки** або **Настройки списку**. (Якщо команда не відображається, виберіть вкладку **Бібліотека** або **список** , щоб відкрити стрічку, а потім виберіть **Параметри бібліотеки** або **Параметри списку** на стрічці.) 
    
3. У розділі **дозволи та керування**виберіть елемент **дозволи для цього списку** або **дозволи для цієї бібліотеки документів**.
    
Зверніть увагу, що за замовчуванням у списках і бібліотеках SharePoint успадковують їхні дозволи від сайту. Якщо ви порушите успадкування дозволів, поточні дозволи буде скопійовано до списку або бібліотеки та всього в ньому. Якщо дозволи для сайту згодом змінено, зміни не застосовуватимуться до списку або бібліотеки та елементи в ньому.
