3. <span data-ttu-id="7f9d9-106">У розділі **дозволи та керування**виберіть **дозволи для цього списку** або **дозволи для цієї бібліотеки документів**.</span><span class="sxs-lookup"><span data-stu-id="7f9d9-106">Under **Permissions and Management**, click **Permissions for this list** or **Permissions for this document library**.</span></span>

Якщо пізніше змінити дозволи для сайту, зміни не буде застосовано до список або бібліотека та елементи всередині неї.</span><span class="sxs-lookup"><span data-stu-id="7f9d9-p102">Note that by default, SharePoint lists and libraries inherit their permissions from the site. If the permissions for the site are later changed, the changes won't be applied to the list or library and the items within it.</span></span>
