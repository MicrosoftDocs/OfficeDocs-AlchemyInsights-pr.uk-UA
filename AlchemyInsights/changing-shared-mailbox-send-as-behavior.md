# <a name="changing-shared-mailbox-send-as-behavior"></a>Зміна спільної поштової скриньки, що надсилається як поведінка

За промовчанням повідомлення, надіслані зі спільної поштової скриньки, не зберігаються в папці "Надіслані" спільної поштової скриньки. Натомість вони зберігаються в папці «Надіслані» особи, яка надіслала повідомлення.
  
Якщо потрібно, щоб усі учасники бачили надіслане повідомлення електронної пошти, у центрі адміністрування Microsoft 365 відредагуйте відомості про спільну поштову скриньку та виберіть **надіслані** \> **Edit**.
  
Щоб дізнатися більше, перегляньте статтю [створення спільної поштової скриньки](https://docs.microsoft.com/microsoft-365/admin/email/create-a-shared-mailbox).
