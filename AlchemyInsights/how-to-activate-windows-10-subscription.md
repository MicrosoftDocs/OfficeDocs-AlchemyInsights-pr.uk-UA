# <a name="activating-windows-10-subscriptions"></a>Активація підписок на Windows 10

Microsoft 365 Business, Microsoft 365 освіти, Microsoft 365 і Enterprise надають цифрові ліцензії для Windows 10 у рамках передплати. Щоб активувати підписку, пристрій вже має активованого Windows 10 Pro (1703 або пізнішої версії). Щоб переглянути повний список вимог для активації передплати Windows 10, ознайомтеся з [вимогами щодо активації передплати Windows 10](https://docs.microsoft.com/windows/deployment/windows-10-subscription-activation#requirements).
