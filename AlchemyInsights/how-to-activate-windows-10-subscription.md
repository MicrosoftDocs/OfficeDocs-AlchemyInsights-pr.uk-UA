# <a name="activating-windows-10-subscriptions"></a>Активація передплати Windows 10

Корпорація Microsoft 365 Business Premium, Office 365 Education, Microsoft 365 і підприємство надають цифрові ліцензії для Windows 10 у рамках передплатою. Щоб активувати передплату, активуйте Windows 10 Pro (1703 або пізнішої версії) на пристрої. Щоб переглянути повний перелік вимог для активації передплатою на Windows 10, ознайомтеся з [вимогами до підключення до служби Windows 10](https://docs.microsoft.com/windows/deployment/windows-10-subscription-activation#requirements).
