# <a name="activating-windows-10-subscriptions"></a>Активація передплати Windows 10

Microsoft 365 Business преміум, Office 365 Education, Microsoft 365 і Enterprise надають цифрові ліцензії на Windows 10 у рамках передплати. Щоб активувати передплату, активуйте Windows 10 Pro (1703 або пізнішої версії) на пристрої. Повний список вимог для активації передплати на Windows 10 див. в статтях Вимоги до [активації передплати на Windows 10.](https://docs.microsoft.com/windows/deployment/windows-10-subscription-activation#requirements)
