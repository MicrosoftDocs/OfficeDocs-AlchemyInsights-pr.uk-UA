# <a name="asms-test-solution"></a>Тестоване рішення ASMS

Клієнти Azure, які укладають договір "Azure Enterprise Agreement" (EA), клієнтської угоди Майкрософт (MCA) або партнерської угоди Майкрософт (MPA), можуть переглядати й завантажувати ціни на портал Azure.
