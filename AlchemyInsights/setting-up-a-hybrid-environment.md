# <a name="setting-up-a-hybrid-environment"></a>Створення гібридних середовища

Щоб дозволити користувачам працювати плавно разом, незалежно якщо їх дані зберігаються на локальних серверах або у службі Office 365, може знадобитися налаштувати гібридних середовища. Щоб отримати уявлення про варіанти, перевірити [Office 365 інтеграції з локальних середовищах](https://support.office.com/article/263faf8d-aa21-428b-aed3-2021837a4b65.aspx).
