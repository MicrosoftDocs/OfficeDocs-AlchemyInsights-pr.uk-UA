# <a name="setting-up-a-hybrid-environment"></a>Настроювання гібридного середовища

Щоб ваші користувачі працювали без проблем, незалежно від того, чи їх дані зберігатимуться на локальних серверах або в Microsoft 365, можливо, знадобиться налаштувати гібридне середовище. Щоб отримати огляд параметрів, ознайомтеся з [інтеграцією Microsoft 365 з локальними середовищами](https://docs.microsoft.com/office365/enterprise/office-365-integration).
