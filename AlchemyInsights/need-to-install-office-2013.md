# <a name="office-2013-is-no-longer-supported-in-microsoft-365-subscriptions"></a>Office 2013 більше не підтримується в Microsoft 365 підписки

З 28 лютого 2017 Office 2013 більше не підтримується в Microsoft 365 підписки.
  
Щоб забезпечити безпеку вашого офісного комплекту та отримати всі найновіші функції, інсталюйте найновішу версію Office безкоштовно як частину передплати Microsoft 365. Переглянути [оновлення користувачів до останнього клієнта Office](https://docs.microsoft.com/office365/admin/setup/upgrade-users-to-latest-office-client).
