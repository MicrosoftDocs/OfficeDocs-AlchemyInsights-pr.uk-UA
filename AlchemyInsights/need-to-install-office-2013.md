# <a name="office-2013-is-no-longer-supported-in-microsoft-365-subscriptions"></a>Office 2013 більше не підтримується в підписок Microsoft 365

Починаючи з 28 лютого 2017, Office 2013 більше не підтримується в підписок Microsoft 365.
  
Щоб забезпечити безпеку програмного комплексу Office і отримати всі найновіші функції, інсталюйте найновішу версію Office безкоштовно як частину своєї передплатою Microsoft 365. Дізнайтеся [, як оновити користувачів до найновішого клієнта Office](https://docs.microsoft.com/microsoft-365/admin/setup/upgrade-users-to-latest-office-client).
