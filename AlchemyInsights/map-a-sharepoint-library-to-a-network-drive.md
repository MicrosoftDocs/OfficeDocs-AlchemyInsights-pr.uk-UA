# <a name="map-a-sharepoint-library-to-a-network-drive"></a>Карта бібліотеці SharePoint на мережному диску

Зіставлення бібліотеку як мережевий диск є тимчасовим і підтримуваних тільки за допомогою браузера Internet Explorer. Замість цього, [Синхронізація файлів SharePoint з новим клієнтом синхронізації OneDrive](https://support.office.com/article/6de9ede8-5b6e-4503-80b2-6190f3354a88.aspx) яка надає [Файли на вимогу](https://support.office.com/article/0e6860d3-d9f3-4971-b321-7092438fb38e.aspx). Отримувати доступ до ваших файлів у OneDrive за допомогою місцевих дискового простору.
  
[Отримати допомогу, настроювання та виправлення неполадок зіставлених мережних дисках](https://go.microsoft.com/fwlink/?linkid=872946)
