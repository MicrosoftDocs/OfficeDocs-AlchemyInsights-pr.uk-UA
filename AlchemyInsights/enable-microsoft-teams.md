# <a name="enable-and-use-microsoft-teams"></a>Увімкнення та використання команд Microsoft

Якщо у вас немає доступу до команд корпорації Майкрософт, Дізнайтеся, [як отримати доступ до команд Microsoft](https://support.office.com/article/How-do-I-get-access-to-Microsoft-Teams-fc7f1634-abd3-4f26-a597-9df16e4ca65b.aspx).

Якщо у вас уже є команди Microsoft і потрібно знати, як настроювати та керувати ними, Дізнайтеся, як [керувати настройками команд Microsoft для вашої організації](https://docs.microsoft.com/MicrosoftTeams/enable-features-office-365).
