# <a name="upload-process-overview-in-microsoft-stream"></a>Огляд процесу завантаження в Microsoft Stream

Щоразу, коли ви завантажуєте відео, Microsoft Stream автоматично кодує його на кілька бітрейт, так що відео може бути потоковим за допомогою адаптивної бітрейт потокового передавання. Microsoft Stream оптимізує бітрейт і якість відтворення відео на основі доступного пропускної здатності глядача.

- [Огляд процесу завантаження в Microsoft Stream](https://docs.microsoft.com/stream/upload-process-overview)
