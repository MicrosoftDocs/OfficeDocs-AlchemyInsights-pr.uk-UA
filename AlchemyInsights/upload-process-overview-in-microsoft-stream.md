# <a name="upload-process-overview-in-microsoft-stream"></a>Огляд процесу передавання даних у Microsoft Stream

Коли ви завантажуєте відео, Microsoft Stream автоматично кодує його на кілька біт, щоб відео можна було використовувати за допомогою адаптивного потокового передавання. Microsoft Stream оптимізує бітрейт і якість відтворення відео на основі доступної смуги пропускання глядача.
