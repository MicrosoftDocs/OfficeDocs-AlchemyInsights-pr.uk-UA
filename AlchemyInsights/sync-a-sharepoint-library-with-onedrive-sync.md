# <a name="sync-a-sharepoint-library-with-onedrive-sync"></a>Синхронізація бібліотеки SharePoint з синхронізацією OneDrive

Для того, щоб ваші файли SharePoint синхронізуватися з вашим локальним жорстким диском, виконайте наступні дії:

1. Перейдіть до бібліотеки документів, яку потрібно синхронізувати на локальному жорсткому диску.
2. Виберіть кнопку синхронізація з меню Параметри.
3. Виконайте крок за кроком, щоб завершити синхронізацію.
