# <a name="set-microsoft-edge-as-the-default-browser-on-a-windows-10-device"></a>Установлення Microsoft Edge браузером за замовчуванням на пристрої з Windows 10

Ось як це зробити.

1. Натисніть кнопку **Пуск** і введіть **стандартні програми**.

1. У результатах пошуку виберіть Програми **за замовчуванням**.

1. У **розділі Браузер** виберіть Microsoft **Edge**.
