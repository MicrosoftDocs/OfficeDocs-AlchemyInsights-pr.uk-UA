# <a name="report-on-file-and-folder-sharing-in-sharepoint-sites"></a>Звіт про спільний доступ до файлів і папок на сайтах SharePoint

Ви можете створити файл CSV, який детально описує всі унікальні файли, користувача, дозволи та посилання на вказаний сайт SharePoint або бібліотеку OneDrive. Це може допомогти вам дізнатися, як використовується спільний доступ, і чи надано спільний доступ до файлів і папок усім гостям.

Щоб отримати докладніші відомості, [повідомте про надання спільного доступу до файлів і папок на сайті SharePoint](https://docs.microsoft.com/sharepoint/sharing-reports).

> [!NOTE]
> Ця функція наразі розгорнута для клієнтів першого випуску та поступово згортується до всіх орендарів виробництва.
