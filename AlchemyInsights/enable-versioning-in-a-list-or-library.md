# <a name="enable-versioning-for-a-list-or-library"></a>Увімкнення відстеження версій для списку або бібліотеки

1. Перейдіть до списку або бібліотеки.
    
2. Натисніть кнопку **Параметри**і натисніть кнопку **настройки бібліотеки** або **списку**.
    
3. Натисніть кнопку **настройки керування версіями**.
    
4. Виберіть параметри, які ви хочете і натисніть кнопку **ОК**.
