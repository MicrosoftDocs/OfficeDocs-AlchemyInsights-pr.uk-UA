# <a name="enable-versioning-for-a-list-or-library"></a>Увімкнення керування версіями для списку або бібліотеки

1. Перейдіть до списку або бібліотеки.
    
2. Натисніть кнопку **Параметри**, а потім виберіть настройки **бібліотеки** або **Настройки списку**.
    
3. Виберіть пункт **настройки керування версіями**.
    
4. Виберіть потрібні параметри, а потім натисніть кнопку **OK**.
