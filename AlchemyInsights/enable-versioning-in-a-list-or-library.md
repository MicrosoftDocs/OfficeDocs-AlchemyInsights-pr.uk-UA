# <a name="enable-versioning-for-a-list-or-library"></a>Увімкнення керування версіями для списку або бібліотеки

1. Перейдіть до списку або бібліотеки.
    
2. Натисніть кнопку **настройки**, а потім виберіть пункт **настройки бібліотеки** або **Параметри списку**.
    
3. Виберіть **параметри керування версіями**.
    
4. Виберіть потрібні параметри та натисніть кнопку **OK**.
