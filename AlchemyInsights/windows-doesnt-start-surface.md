# <a name="windows-doesnt-start-on-surface"></a>Windows не запускається на поверхні

Якщо Windows не запускається на поверхні або пристрій не прокидається від сну, виконайте наведені нижче дії з виправлення неполадок.

- Якщо Windows не запускається належним чином, коли ви вмикаєте свою поверхню (наприклад, Windows не завантажується), щоб вирішити цю проблему, [Windows не запускається на поверхні](https://support.microsoft.com/surface/windows-doesn-t-start-on-surface-3dd47ea1-472a-4514-c8e1-ff81bd72be5c).

- Якщо поверхню не ввімкнуто або не прокинеться, для виправлення неполадок, які можуть відображатися на [поверхні, не вмикайте або не прокидаються з режиму сну](https://support.microsoft.com/surface/surface-won-t-turn-on-or-wake-from-sleep-1e181652-3db8-5ca1-9649-7390fafb102a).

- Якщо використовується пристрій сторонніх постачальників, перегляньте статтю [Виправлення неполадок для пристроїв сторонніх постачальників](https://support.microsoft.com/topic/b6f3408d-dac9-43e2-82f6-e620ca783636).
