# <a name="pst-data-files-in-outlookcom"></a>PST дані файли в Outlook.com

Це не можливо, щоб імпортувати в Outlook. PST файл даних до облікового запису Outlook.com за допомогою браузера.

Щоб імпортувати в. PST файл даних, скористайтеся застосунку Outlook. Щоб дізнатися більше, прочитайте [імпорт електронної пошти, контакти та календар із послугами Outlook. PST.](https://support.office.com/article/431a8e9a-f99f-4d5f-ae48-ded54b3440ac)
