# <a name="cannot-access-sharepoint"></a>Не вдалося отримати доступ до SharePoint

Існує кілька причин, чому SharePoint або OneDrive можуть стати недоступними. Якщо не вдається отримати доступ до SharePoint або OneDrive, скористайтеся нижченаведену посібнику, щоб вирішити цю проблему.

- [Не вдалося отримати доступ до SharePoint або OneDrive](https://docs.microsoft.com/sharepoint/troubleshoot/sharing-and-permissions/sharepoint-online-inaccessible)
