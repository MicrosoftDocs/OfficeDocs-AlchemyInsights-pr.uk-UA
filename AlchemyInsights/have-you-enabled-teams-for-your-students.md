# <a name="have-you-enabled-teams-for-your-students"></a>Ви ввімкнули команд для ваших студентів?

Факультету і персоналу мати доступ до команд, але ваші учні не можуть? Команди повинна бути явно включена для ваших студентів, а також для викладачів і співробітників. Для цього, будь ласка, дивіться [Увімкнути Microsoft команд для вашої школи](https://docs.microsoft.com/education/get-started/enable-microsoft-teams).
