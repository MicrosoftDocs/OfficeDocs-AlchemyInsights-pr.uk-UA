# <a name="calendar-not-updating-outlook-for-mac"></a>Календар не оновлюється в Outlook для Mac

Якщо елементи календаря відсутні або не оновлюються, ми зберемо у вас певні дані, щоб з'ясувати причину.

1. У відкритій програмі Outlook відкрийте меню Windows, а потім виберіть пункт **Помилки синхронізації.**

1. Клацніть піктограму чорної консолі ліворуч від носа. Папка діагностики відкриється автоматично.

1. Стисніть і завантажте найновіший файл діагностики для підтримки для аналізу.
