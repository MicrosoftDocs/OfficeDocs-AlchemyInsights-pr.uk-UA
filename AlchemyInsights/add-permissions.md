# <a name="add-permissions"></a>Додавання дозволів

1. Використовуйте кінцеву точку згоди адміністратора [, щоб настроїти список відповідних дозволів для програми](https://docs.microsoft.com/azure/active-directory/develop/v2-permissions-and-consent#to-configure-the-list-of-statically-requested-permissions-for-an-application).
1. Існує два дозволи для надання можливості створення реєстрацій програм, кожна з яких відрізняється від поведінки. Докладні відомості наведено в статті [дозволи на реєстрацію застосунку для користувацьких ролей у службі Azure Active Directory](https://docs.microsoft.com/azure/active-directory/roles/custom-available-permissions).
1. [Дозволи для ролей](https://docs.microsoft.com/azure/active-directory/roles/permissions-reference#role-permissions): таблиці, які описують певні дозволи в лазуровому Active Directory, надані кожній ролі. Деякі ролі можуть мати додаткові дозволи в службах Microsoft за межі служби Azure Active Directory.
