# <a name="connect-to-the-mscommerce-module"></a>Підключення до модуля MSCommerce

Перш ніж переглядати або встановлювати політику AllowSelfServicePurchase, потрібно підключитися до модуля MSCommerce.  

Щоб підключитися до модуля MSCommerce, у рядку PowerShell (PS C: \) , ввести таку команду:

`Connect-MSCommerce`

Відкриється діалогове вікно входу. Укажіть ім'я користувача та пароль для входу.

**Примітка.** &nbsp; &nbsp; Обліковий запис, який використовується для входу, має бути компанією або адміністратором виставлення рахунків.
