# <a name="calendar-publishing-information"></a>Відомості про публікацію календаря

- Адміністратори Microsoft 365 можуть вмикати або вимикати публікацію календаря та спільний доступ до календаря для організації. Для отримання додаткових відомостей див. [Увімкнення публікації календаря, щоб люди могли обмінюватися інформацією](https://support.office.com/article/EB432E21-AAF0-466B-BF85-CEFEC0C7C4FC) та [обмінюватися календарями з зовнішніми користувачами](https://docs.microsoft.com/office365/admin/manage/share-calendars-with-external-users).

- Користувачі Outlook можуть ділитися власними календарями. Для отримання додаткових відомостей див. [спільний доступ до календаря Outlook з іншими людьми](https://support.office.com/article/353ed2c1-3ec5-449d-8c73-6931a0adab88).
