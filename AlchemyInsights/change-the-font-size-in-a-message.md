# <a name="change-the-font-size-in-a-message"></a>Змінення розміру шрифту в повідомленні

1. Якщо ви вже ввели повідомлення, виділіть текст.
2. Виберіть піктограму "  **Формат** " під повідомленням (перша піктограма).
3. Виберіть піктограму  **розмір шрифту**  над повідомленням (це сьома піктограма).
4. Виберіть потрібний розмір.
