# <a name="profile-property-synchronization"></a>Синхронізація властивостей профілю

Не можна зіставити додаткові властивості для імпорту ОГОЛОШЕНЬ для синхронізації з Azure Active Directory до застосунку профілю користувача. Імпорт ОГОЛОШЕНЬ обмежується попередньо набором властивостей для забезпечення послідовного виконання завдання таймера.
  
[Докладніше про властивості профілю, які синхронізуються](https://go.microsoft.com/fwlink/?linkid=875671)
