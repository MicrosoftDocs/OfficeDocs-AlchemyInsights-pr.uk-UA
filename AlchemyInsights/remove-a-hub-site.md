# <a name="remove-a-hub-site"></a>Видалення сайту концентратора

Щоб скасувати реєстрацію сайту концентратора у новому центрі адміністрування SharePoint:
  
1. У новому центрі адміністрування SharePoint у розділі **сайти**виберіть пункт **активні сайти**.

2. Виберіть сайт, натисніть **сайт-концентратор**і натисніть кнопку **скасувати реєстрацію як концентратор сайту**.

3. Натисніть кнопку **OK**.

[Докладніше про сайти-концентратор](https://support.office.com/article/what-is-a-sharepoint-hub-site-fe26ae84-14b7-45b6-a6d1-948b3966427f)
