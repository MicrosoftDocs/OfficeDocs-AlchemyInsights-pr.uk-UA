# <a name="remove-a-hub-site"></a>Видалити сайт-концентратор

Скасування реєстрації сайт-концентратор у нового центру адміністрування SharePoint
  
1. Нові центру адміністрування SharePoint під **сайти**, клацніть **активних ділянок**.

2. Виберіть сайт, натисніть кнопку **сайт-концентратор**і натисніть кнопку **Unregister як сайт-концентратор**.

3. Натисніть кнопку **OK**.

[Дізнайтеся більше про концентратор сайти](https://support.office.com/article/what-is-a-sharepoint-hub-site-fe26ae84-14b7-45b6-a6d1-948b3966427f?ui=en-US&amp;rs=en-US&amp;ad=US)
