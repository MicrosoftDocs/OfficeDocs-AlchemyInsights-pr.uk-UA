# <a name="deploy-office-apps"></a>Розгортання програм Office

Розгортання програм Office для кількох кінцевих користувачів за допомогою засобу розгортання Office, Microsoft кінцевої точки Диспетчер конфігурацій, як частина образу диска операційної системи, або у спільному комп'ютері сценарій, використовуючи Windows Server з віддаленого робочого стола служби (RDS) або служби терміналів, спробуйте використовувати крок за кроком керівництво в **Office 365 ProPlus розгортання консультанта**.
  
Щоб почати роботу, переконайтеся, що ви ввійшли як адміністратор клієнта або Глобальний адміністратор, а потім клацніть тут: [https://aka.ms/proplusguidance](https://aka.ms/proplusguidance).
