# <a name="deploy-microsoft-365-apps"></a>Розгортання програм Microsoft 365

Щоб розгорнути програми Microsoft 365 до кількох кінцевих користувачів за допомогою засобу розгортання Office, диспетчер конфігурації кінцевої точки Microsoft, у складі операційної системи або на спільному комп'ютері, використовуючи Windows Server за допомогою служб віддаленого робочого стола або служб терміналів, радимо скористатися поетапним посібником в **програмах Microsoft 365 для консультанта із розгортання**.
  
Щоб почати, переконайтеся, що ви ввійшли як адміністратор клієнта або Глобальний адміністратор, а потім клацніть тут: [https://aka.ms/proplusguidance](https://aka.ms/proplusguidance) .
