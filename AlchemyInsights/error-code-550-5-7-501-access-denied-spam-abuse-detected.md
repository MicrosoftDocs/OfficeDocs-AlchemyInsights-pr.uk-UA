# <a name="550-57501-access-denied-spam-abuse-detected"></a>550 5.7.501 доступ заборонено, виявлено порушення спаму

Зазвичай це повідомлення виникає, коли користувачі надсилають повідомлення електронної пошти з IP-адрес за допомогою початкового домену *. onmicrosoft.com* , який призначається для нових орендарів у Microsoft 365. Найпростіший спосіб вирішити цю проблему:

1. [Додайте домен до свого клієнта](https://docs.microsoft.com//office365/admin/setup/add-domain).

2. [Змініть основну адресу електронної пошти користувачів](https://docs.microsoft.com//office365/admin/add-users/change-a-user-name-and-email-address) на новий користувацький домен, який ви щойно додали.
