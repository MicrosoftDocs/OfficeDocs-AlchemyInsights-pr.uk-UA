# <a name="how-permissions-inheritance-works-in-sharepoint"></a>Як працює успадкування дозволів в SharePoint

За промовчанням дозволи в SharePoint успадковано від вище вгору в ієрархії. Так файл успадковують його дозволи з папки, які успадковують його дозволи з бібліотеки, які успадковують його дозволи на сайті, який успадковує дозволи від колекції сайтів.
  
Щоб отримати відомості про видалення унікальні дозволи та відновлення успадкування побачити [редагувати і керувати дозволами для списку або бібліотеки](https://go.microsoft.com/fwlink/?linkid=869946).
