# <a name="how-permissions-inheritance-works-in-sharepoint"></a>Як працює успадкування дозволів у програмі SharePoint

За промовчанням дозволи в SharePoint успадковуються від вище в ієрархії. Таким чином, файл успадковує свої дозволи з папки, яка успадковує свої дозволи від бібліотеки, яка успадковує свої дозволи від сайту, який успадковує свої дозволи від колекції сайтів.
  
Для отримання відомостей про видалення унікальних дозволів і відновлення успадкування зверніться до [редагування та керування дозволами для списку або бібліотеки](https://go.microsoft.com/fwlink/?linkid=869946).
