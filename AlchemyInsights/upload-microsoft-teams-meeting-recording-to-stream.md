# <a name="upload-a-microsoft-teams-meeting-recording-to-stream-a1"></a>Передавання запису нарад групи Microsoft до потокового передавання (a1)

Для більшості користувачів служби Microsoft 365, щоб записи наради були автоматично передані до Microsoft Stream після наради. Однак, починаючи з  **20 серпня 2020**р., коли користувачі з a1 (включно з a1 Plus) запускаються в записі нарад групи, ця поведінка змінюється.  

Докладні відомості наведено в статті [передавання записів нарад команд Microsoft до потокового передавання](https://docs.microsoft.com/stream/portal-upload-teams-meeting-recording).
