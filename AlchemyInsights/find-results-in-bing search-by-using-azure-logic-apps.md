# <a name="find-results-in-bing-search-by-using-azure-logic-apps"></a>Пошук результатів пошуку в службі Bing за допомогою програм Azure Logic

За допомогою функції пошуку в службі Bing у програмі логіки можна автоматизувати обробку результатів пошуку та зробити ці результати доступними для інших дій. Наприклад, ви можете знаходити пункти новин, а потім надіслати їх як Tweets в каналі.

Докладні відомості наведено в статті [Пошук результатів у пошуковій службі Bing за допомогою програм Azure Logic](https://go.microsoft.com/fwlink/?linkid=2151928).
