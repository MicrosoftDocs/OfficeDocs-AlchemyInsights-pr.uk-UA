# <a name="list-view-threshold"></a>Граничне значення подання списку

Граничне значення подання списку має місце для підтримки продуктивності служби. Граничне значення не можна змінити в SharePoint Online. Щоб уникнути попадання цього порогу, перегляньте список [керування великими списками та бібліотеками в SharePoint](https://support.office.com/article/manage-large-lists-and-libraries-in-sharepoint-b8588dae-9387-48c2-9248-c24122f07c59).

Ця помилка також може з'явитися, якщо спробувати надати спільний доступ до успадкування папки або розірвати її успадкування. Див [.](https://docs.microsoft.com/SharePoint/troubleshoot/lists-and-libraries/error-share-break-inheritance)
