# <a name="sharepoint-migration-performance"></a>Ефективність перенесення

**Важливо**. Багато клієнтів SharePoint Online і OneDrive запускають у службі критично важливі для бізнесу програми, що працюють у фоновому режимі. До них належать перенесення вмісту, захист від втрати даних (DLP) і рішення для резервного копіювання. У цей безпрецедентний час ми працюємо над тим, щоб служби SharePoint Online і OneDrive залишалися високодоступними та надійними для користувачів, які залежать від них більше, ніж будь-коли, через віддалену роботу.

З цією метою ми впровадили жорсткіші обмеження кількості запитів для фонових програм (перенесення, DLP та рішення для резервного копіювання) у денний час у будні дні. Слід очікувати, що пропускну здатність цих програм буде дуже сильно обмежено протягом цього часу. Проте у вечірній час та по вихідних (для конкретного регіону) служба зможе обробляти значно більший обсяг запитів від фонових програм.

**Ефективність перенесення**

На ефективність перенесення можуть впливати мережева інфраструктура, розмір файлу, час перенесення та обмеження. Розуміння цього допоможе вам планувати та максимально підвищити ефективність перенесення.
