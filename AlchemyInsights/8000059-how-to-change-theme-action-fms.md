# <a name="how-to-change-your-theme"></a>Як змінити вашу тему

Щоб змінити тему в Outlook.com або Outlook в Інтернеті, зайдіть в **настройки** і вибрати тему під **теми**. Щоб переглянути всі доступні теми, виберіть **подання усі** під **теми**.
