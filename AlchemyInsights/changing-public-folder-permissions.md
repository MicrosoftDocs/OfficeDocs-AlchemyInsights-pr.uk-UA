# <a name="changing-public-folder-permissions"></a>Зміна дозволів для спільної папки

Ви та ваші користувачі можуть змінити дозволи спільної папки в Outlook. Див [встановити або змінити дозволи для спільної папки](https://support.office.com/en-us/article/set-or-change-permissions-for-a-public-folder-b2e0440c-7873-48ec-9ff2-b1a20b723005). Адміністратори можуть також змінювати дозволи Exchange центру адміністрування:
  
- В Office 365 центру адміністрування, перейти до **адмін центрів** \> **обміну**.
    
- Виберіть **спільні папки**.
    
Звідти призначаючи користувачів групи безпеки дозволи можна змінити дозволи для окремих спільних папок. Щоб змінити дозволи для спільної папки, потрібно бути власником його.
