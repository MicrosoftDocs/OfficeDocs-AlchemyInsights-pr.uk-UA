# <a name="how-to-change-your-theme"></a>Змінення теми

Щоб змінити тему в Outlook.com або Інтернет-версії Outlook, перейдіть до розділу **настройки** та виберіть тему в розділі **Тема**. Щоб переглянути всі доступні теми, виберіть пункт **Переглянути всі** в розділі **Тема**.
