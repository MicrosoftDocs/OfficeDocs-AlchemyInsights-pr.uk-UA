# <a name="submit-dynamics-365-support-requests"></a>Надсилання запитів на підтримку Dynamics 365

Якщо ви є клієнтом **програми з залучення клієнтів Microsoft Dynamics**, надішліть свою заявку через новий [центр адміністрування платформи Power](https://admin.powerplatform.microsoft.com/?ref=officemodern).
  
Цей новий портал пропонує пряміший шлях до професійної підтримки, оптимізований для програми залучення клієнтів Dynamics та інших продуктів платформи Power.
