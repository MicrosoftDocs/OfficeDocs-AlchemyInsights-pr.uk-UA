# <a name="submit-dynamics-365-support-requests"></a>Надсилання запитів на підтримку Dynamics 365

Якщо ви є клієнтом **Microsoft Dynamics Замовника участі** будь ласка, надсилайте ваш квиток через нові [платформи влада центру адміністрування](https://admin.powerplatform.microsoft.com/?ref=officemodern).
  
Новий портал надає більш прямий шлях до припиненою інтерактивною підтримкою, оптимізованими для динаміка залучення клієнтів та інших продуктів ГЕС платформи.
