# <a name="block-email-forwarding"></a>Блокувати пересилання електронної пошти

Щоб вимкнути пересилання для певної поштової скриньки електронної пошти, побачити [настроїти пересилання електронної пошти](https://support.office.com/client/15abf81d-5c5d-49da-ac81-1b4daa1809f6).

Щоб заблокувати пересилання електронної пошти для організації в цілому, [багато способів, щоб блокувати автоматичної електронної пошти, пересилання у службі Exchange Online](https://blogs.technet.microsoft.com/exchange/2017/12/22/the-many-ways-to-block-automatic-email-forwarding-in-exchange-online/).
