# <a name="block-email-forwarding"></a>Блокувати пересилання електронної пошти

Щоб вимкнути пересилання електронної пошти для певної поштової скриньки, перегляньте переглянути [налаштування пересилання електронної пошти](https://docs.microsoft.com/office365/admin/email/configure-email-forwarding).

Щоб заблокувати пересилання електронної пошти для всієї організації, [багато способів блокувати автоматичне пересилання електронної пошти в Exchange Online](https://blogs.technet.microsoft.com/exchange/2017/12/22/the-many-ways-to-block-automatic-email-forwarding-in-exchange-online/).
