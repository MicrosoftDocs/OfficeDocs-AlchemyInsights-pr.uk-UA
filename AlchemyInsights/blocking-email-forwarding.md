# <a name="block-email-forwarding"></a>Блокування пересилання електронної пошти

Щоб вимкнути пересилання електронної пошти для певної поштової скриньки, перегляньте статтю [настроювання пересилання електронної пошти](https://docs.microsoft.com/microsoft-365/admin/email/configure-email-forwarding).

Щоб заблокувати пересилання електронної пошти для всієї організації, [багато способів заблокувати автоматичне пересилання електронної пошти в службі Exchange Online](https://blogs.technet.microsoft.com/exchange/2017/12/22/the-many-ways-to-block-automatic-email-forwarding-in-exchange-online/).
