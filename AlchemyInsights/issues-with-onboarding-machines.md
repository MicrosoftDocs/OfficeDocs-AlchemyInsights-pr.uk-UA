# <a name="issues-with-onboarding-machines"></a>Проблеми з машинами для ознайомлення

Можливо, виникли проблеми з машинами для ознайомлення з службою MDATP. Якщо ви можете отримати доступ до машини кінцевого користувача, виконайте такі дії:

1. Завантажити засіб діагностики [підключення до клієнта](https://aka.ms/mdatpanalyzer) .
2. Розпакуйте і запустіть MDATPAnalyzer. cmd.
3. Знайдіть журнал діагностики в папці Mdatpклієнтаналізрезультат, ту саму папку, де завантажується засіб аналізатора.
4. Перегляньте файл журналу MDATPClientAnalyzer.txt, щоб знайти настройки підключення до Інтернету або проксі-сервера.
