# <a name="quick-edit"></a>Швидке редагування

Є два способи редагування елемента у списку SharePoint. Можна змінити окремий елемент у поданні списку, який є методом за промовчанням. Або навіть більш продуктивною, можна редагувати кілька елементів у поданні швидкого редагування. Для редагування елементів потрібен дозвіл на рівні внеску або вищий. Переглянути [редагування елементів списку в SharePoint Online](https://support.microsoft.com/office/dac1a1c3-a80b-4082-ba57-715cf613d0f7).
