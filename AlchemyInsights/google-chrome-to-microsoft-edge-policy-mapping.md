# <a name="learn-about-mapping-between-google-chrome-policies-and-microsoft-edge-policies"></a>Відомості про зіставлення політики Google Chrome і політики Microsoft EDGE

Політики Google Chrome зіставляються з відповідними політиками Microsoft EDGE, підтримуваними у версії 80 браузера. Докладні відомості наведено в статті [Google Chrome, щоб відобразити політику Microsoft EDGE](https://go.microsoft.com/fwlink/?linkid=2141933).
