# <a name="microsoft-stream-storage"></a>Microsoft потік сховище

Microsoft потік орендарів, отримують фіксований розподіл 500 ГБ пам'яті та додаткові 0,5 ГБ пам'яті на ліцензованого користувача.
Для отримання додаткової інформації про сховище див.:

- [Огляд потокового передавання даних Microsoft](https://docs.microsoft.com/stream/license-overview#storage)
- [Додавання сховища до Microsoft Stream](https://docs.microsoft.com/stream/storage-add-on)
- [Квоти та обмеження у Microsoft Stream](https://docs.microsoft.com/stream/quotas-and-limitations)
