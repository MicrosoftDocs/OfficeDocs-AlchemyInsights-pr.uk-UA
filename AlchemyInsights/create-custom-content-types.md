# <a name="create-custom-content-types"></a>Створення користувацьких типів вмісту

1. Виберіть список, до якого потрібно працювати, на панелі швидкого запуску або натисніть кнопку **настройки**  (піктограма шестерні), а потім виберіть пункт  **Параметри сайту**.
2. Виберіть пункт типи **вмісту сайту**  в  **колекції веб-дизайнера**.
3. На сторінці "типи вмісту сайту" натисніть кнопку Створити.
4. На сторінці новий тип вмісту сайту Введіть відомості для настроюваного типу вмісту.

Щоб отримати докладні відомості, ознайомтеся з  [настроюваними типами вмісту](https://support.microsoft.com/office/e1277a2e-a1e8-4473-9126-91a0647766e5#__toc323548991).
