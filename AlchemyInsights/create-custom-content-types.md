# <a name="create-custom-content-types"></a>Створення користувацьких типів вмісту

1. Виберіть список, з якого потрібно працювати на панелі швидкого запуску, або натисніть кнопку **настройки** (піктограма шестірні), а потім виберіть пункт **настройки сайту**.
2. Натисніть кнопку типи **вмісту сайту** в розділі **галереї веб-дизайнера**.
3. На сторінці типи вмісту сайту натисніть кнопку Створити.
4. На сторінці новий тип вмісту сайту Введіть відомості для настроюваного типу вмісту.

Додаткові відомості наведено в додаткових [типах вмісту](https://support.microsoft.com/office/e1277a2e-a1e8-4473-9126-91a0647766e5#__toc323548991).
