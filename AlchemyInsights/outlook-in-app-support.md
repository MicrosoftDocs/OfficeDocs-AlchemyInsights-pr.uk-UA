# <a name="in-app-support-in-outlook-for-windows"></a>Підтримка в додатку в Outlook для Windows

Як адміністратор клієнта, ви можете отримати в програмі підтримки для проблеми в Outlook для Windows. У програмі Outlook виберіть **Довідка**  >  **зверніться до служби підтримки**. Введіть ваше запитання, і член команди підтримки Outlook для Windows відповість на ваше повідомлення.
