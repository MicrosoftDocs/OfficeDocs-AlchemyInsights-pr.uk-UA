# <a name="in-app-support-in-outlook-for-windows"></a>Підтримка в програмі Outlook для Windows

Як адміністратор клієнта, ви можете отримати підтримку в програмі Outlook для Windows. У програмі Outlook виберіть елемент **Довідка**  >  **зі служби підтримки**. У вас є запитання, а учасник команди підтримки Outlook для Windows відповість на ваше повідомлення.
