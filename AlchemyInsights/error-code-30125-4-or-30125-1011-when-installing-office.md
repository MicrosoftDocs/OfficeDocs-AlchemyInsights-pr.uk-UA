# <a name="solutions-for-error-30125-4-or-30125-1011"></a>Рішення для помилка 30125-4 або 30125 1011

Цю помилку може відображатися, якщо антивірусне програмне забезпечення, брандмауер, настройки проксі-сервера або підключення перешкоджати інсталяції Office. Ось декілька речей, які ви можете спробувати вирішити проблему:
  
- Перезавантажте комп'ютер і повторно інсталювати Office
    
- Інсталяції Office, використовуючи [автономні установки Office](https://support.office.com/article/f0a85fe7-118f-41cb-a791-d59cef96ad1c.aspx)
    
- Тимчасово вимкніть налаштування проксі
    
- Тимчасове вимкнення антивірусного програмного забезпечення
    
- Тимчасове вимкнення брандмауера
    
Для отримання додаткових відомостей див. [код помилки 30125-4 або 30125 1011 під час інсталяції Office](https://support.office.com/article/7bfabec6-76be-4cde-880e-819a9c569612.aspx)
