# <a name="unable-to-add-an-attachment-to-an-externally-facing-form"></a>Не вдалося додати вкладення до форми з зовнішнім обличчям

Якщо ви намагаєтеся додати вкладення до зовнішнього опитування Microsoft Forms, ця функція ще не доступна. Для відстеження стану цього запиту функції перейдіть до [цього запису UserVoice](https://go.microsoft.com/fwlink/?linkid=2133069)і додайте свій голос.
