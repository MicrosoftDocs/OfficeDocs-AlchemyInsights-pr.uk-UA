# <a name="restore-a-previous-file-version"></a>Відновлення попередньої версії файлу

За допомогою онлайнової історії версій можна переглядати та відновлювати старіші версії файлів у OneDrive або SharePoint. Журнал версій працює з усіма типами файлів, включаючи PDF, файли CAD, фотографії та відео. Щоб відновити попередню версію файлу:
  
1. Відкрийте бібліотеку документів SharePoint або OneDrive розташування.
    
2. Клацніть ліворуч від файлу, щоб вибрати його, і натисніть кнопку **журнал версій**. Якщо на панелі команд не відображається Історія версій, натисніть кнопку Додатково ( **...** ). 
    
3. Виберіть версію, яку потрібно переглянути, або натисніть кнопку Додатково ( **...** ) праворуч від версії та натисніть кнопку **відновити**.
