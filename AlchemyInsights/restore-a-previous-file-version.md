# <a name="restore-a-previous-file-version"></a>Відновити попередню версію файлу

Онлайн-версії історії ви можете переглянути і відновити старі версії файлів в OneDrive або SharePoint. Історія версій працює з всіх типів файлів, включаючи PDF-файли, файли CAD, фото та відео. Щоб відновити попередню версію файлу:
  
1. Відкрийте бібліотеку документів SharePoint або OneDrive розташування.
    
2. Клацніть зліва від файлу, щоб виділити його та клацніть **журнал версій**. Якщо журнал версій не відображається на панелі команд, натисніть Далі ( **...** ). 
    
3. Виберіть версію, яку ви хочете переглянути, або натисніть кнопку Далі ( **...** ) праворуч від версії і натисніть кнопку **відновити**.
