# <a name="working-with-sharepoint-permission-levels"></a>Робота з рівнями дозволів SharePoint

SharePoint постачається з рівнями дозволів за промовчанням, які можна використовувати **, наприклад,** лише для створення та **перегляду**. Але якщо один із них не відповідає вашим потребам, можна створити новий рівень дозволів. Пам'ятайте, що якщо не змінити типові значення, легше відстежувати рівні дозволів.

[Створення та редагування рівнів дозволів](https://docs.microsoft.com/sharepoint/how-to-create-and-edit-permission-levels)
