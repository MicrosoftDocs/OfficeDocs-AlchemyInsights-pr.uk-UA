# <a name="error-your-new-install-has-newer-features-than-your-current-version-do-you-want-to-activate"></a>Помилка: Нова інсталяція має нові функції, ніж поточна версія. Ви хочете активувати...

Щоб виправити цю помилку Office, виберіть автоматизоване рішення для своєї версії Office.

- Для Microsoft 365 завантажте та запустіть Помічник із [підтримки й відновлення](https://aka.ms/SaRA-OfficeActivation-Chat). Дотримуйтеся вказівок і підказок, щоб виправити помилку активації Office.

- Для Office 2019 або Office 2016 виконайте вказівки, наведені в статті дії з [неліцензованим продуктом і помилки активації в Office](https://support.microsoft.com/office/0d23d3c0-c19c-4b2f-9845-5344fedc4380#bkmk_fixyourself).

- У службі Office 2013 перегляньте статтю [Microsoft Easy Fix Solutions](https://support.microsoft.com/topic/microsoft-easy-fix-solutions-have-been-discontinued-b0f4b5f9-3b5a-bd9e-d75d-d45e2f12e16c).
