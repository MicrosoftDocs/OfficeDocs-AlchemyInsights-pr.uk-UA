# <a name="distribution-group-aimkiobject-not-showing-in-groups-list-after-aimkitimeelapsed-minutes"></a>Групи розсилки "{AimKIObject}", не відображається у списку груп через {AimKITimeElapsed} хвилин

Ваша група «{AimKIObject}» був створений {AimKITimeElapsed} хвилин тому. Зазвичай не потрібно більше, ніж 60 хвилин для вашого нові групи розсилки з'являтися центру адміністрування.
  
Може мати можливість використовувати функції груп > одержувачів [Exchange центру адміністрування](https://outlook.office365.com/ecp/?rfr=Admin_o365&amp;exsvurl=1&amp;mkt=en-US.aspx) керувати вашої групи зараз або скористайтеся параметром "назвемо ви" говорити до допоміжного інженер.
