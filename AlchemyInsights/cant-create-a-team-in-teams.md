# <a name="cant-create-a-team-in-teams"></a>Не вдається створити команду в командах

Можливо, адміністратор групи надав вам дозволи, необхідні для створення команд.  

Перегляньте адміністратор про створення або змінення настройок групи безпеки організації в Microsoft 365. Якщо ви додали до групи безпеки, ви можете створити команди.
