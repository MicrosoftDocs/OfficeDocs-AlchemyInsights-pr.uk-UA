# <a name="copy-and-paste-in-outlookcom"></a>Скопіюйте та вставте в Outlook.com

Є немає копіювання або вставлення кнопки в Outlook.com, але ви можете використовувати сполучення клавіш:

**На ПК:** CTRL + C, щоб скопіювати, CTRL + X, щоб вирізати і CTRL + V, щоб вставити.

**На Mac:** CMD + C, щоб скопіювати, CMD + X, щоб вирізати і CMD + V, щоб вставити.
