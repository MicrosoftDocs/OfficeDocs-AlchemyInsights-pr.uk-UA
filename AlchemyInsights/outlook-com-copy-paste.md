# <a name="copy-and-paste-in-outlookcom"></a>Скопіюйте та вставте в Outlook.com

У Outlook.com немає кнопок копіювання або вставки, але ви можете використовувати комбінації клавіш:

**На ПК:** CTRL + C щоб скопіювати, CTRL + X, щоб вирізати, і CTRL + V для вставки.

**На комп'ютері Mac:** CMD + C скопіювати, CMD + X нарізати, і CMD + V для вставки.
