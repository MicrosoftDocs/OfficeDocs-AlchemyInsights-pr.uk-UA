# <a name="create-a-folder"></a>Створити папку

1. На лівій панелі виберіть **Створити нову папку** поруч із **папки**. 
    
2. У полі нову папку введіть ім'я для папки та натисніть клавішу Enter.
    
Щоб отримати додаткові відомості, див. [Створення папки в Outlook.com](https://go.microsoft.com/fwlink/p/?linkid=873114).
