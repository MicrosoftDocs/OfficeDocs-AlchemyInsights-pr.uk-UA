# <a name="entity-type-not-showing-in-search-results"></a>Тип сутності, який не відображається в результатах пошуку

Для типу сутності, який відображатиметься в єдиному інтерфейсі Dynamics 365, потрібно спочатку додати до модуля програми.

Примітка: під час пошуку вкладень у відповідність пошуку потрібно додати об'єкт вкладення до модуля програми, навіть якщо батьківський запис відображається в результатах.
