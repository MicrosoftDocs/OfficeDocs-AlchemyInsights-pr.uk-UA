# <a name="entity-type-not-showing-in-search-results"></a>Тип сутності, який не відображається в результатах пошуку

Для сутності типу відображення в категоріям Пошук або релевантність пошукових в Dynamics 365 єдиному інтерфейсі, він повинен бути перший додані app модуля.

Примітка: У пошуках вкладення в релевантність пошукових вкладення сутності слід додати до модуля app хоча батьківський запис пункти результати.
