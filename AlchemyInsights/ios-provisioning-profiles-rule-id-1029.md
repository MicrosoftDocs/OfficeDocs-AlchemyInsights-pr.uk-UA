# <a name="ios-provisioning-profiles"></a>Профілі підготовки iOS

Apple iOS лінійний бізнес використовує концепцію профілю підготовки, який використовується для перевірки цілісності додатків і дотримання правил, визначених у профілі. Ці профілі підготовки закінчуються через 12 місяців, і тому новий профіль повинен бути пов'язаний з розгорнутими програмами для того, щоб програми могли продовжувати працювати.
  
У цій статті описується, як створити пов'язати новий профіль для підготовки з iOS бізнес-застосунок, розгорнуті через InTune: [використання IOS застосунок підготовка профілі, щоб запобігти, що ваші програми не закінчується](https://docs.microsoft.com/intune/app-provisioning-profile-ios)
