# <a name="enable-seamless-sso"></a>Увімкнути безшовні єдиного входу

Блакитні Active Directory (AD блакитні) безшовні єдиного входу (SSO безшовні) здійснює автоматичний вхід користувачам розташовані на своїх корпоративних комп'ютерів, підключених до корпоративної мережі.
  
[Дотримуйтесь інструкцій у цій статті, щоб увімкнути безшовні єдиного входу](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-sso-quick-start)
