# <a name="enable-seamless-sso"></a>Увімкнути безшовні SSO

Azure Active Directory (Azure AD) безшовні єдиного входу (безшовний SSO) автоматично входить у користувачів, коли вони на корпоративному комп'ютері, підключені до корпоративної мережі.
  
[Виконайте дії, описані в цій статті, щоб увімкнути безшовні єдиного входу](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-sso-quick-start)
