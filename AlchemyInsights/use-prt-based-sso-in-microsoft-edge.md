# <a name="use-prt-based-sso-in-microsoft-edge"></a>Використання єдиного входу на основі PRT у Microsoft Edge

Основний [маркер оновлення (PRT)](https://go.microsoft.com/fwlink/?linkid=2133632) – це ключ Azure Active Directory, який використовується для автентифікації на пристроях із Windows 10, iOS і Android. PRT вмикає єдиний доступ для всіх програм, які використовуються на цих пристроях.

У Microsoft Edge підтримка єдиного входу на основі PRT є власною і не вимагає додаткових розширень. У Windows 10 RS3 та вище користувачі, які ввійшли в систему, отримає єдиний вхід із PRT на веб-сайти, які підтримують єдиний вхід на основі PRT.
