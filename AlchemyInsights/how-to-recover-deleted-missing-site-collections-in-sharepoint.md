# <a name="recover-missing-or-deleted-site-collections"></a>Відновлення відсутніх або видалених колекцій сайтів

Перегляньте наведені нижче статті, щоб отримати відомості про те, як відновити відсутні або видалені колекції сайтів у службі SharePoint.

[Відновлення видалених файлів](https://docs.microsoft.com/sharepoint/restore-deleted-site-collection)

[Відновлення за допомогою модуля SPO PowerShell](https://support.office.com/article/Introduction-to-the-SharePoint-Online-Management-Shell-C16941C3-19B4-4710-8056-34C034493429)

Якщо ви не можете знайти сайт, який може знадобитися для [пошуку в журналі аудиту в &amp; центрі відповідності системи безпеки Microsoft 365](https://docs.microsoft.com/microsoft-365/compliance/search-the-audit-log-in-security-and-compliance).

**Примітка.** "Сучасні" сайти групи не відображаються в **КЛАСИЧНОМУ** інтерфейсі адміністрування SharePoint. Ви можете отримати доступ до списку "сучасних" сайтів із [нового центру адміністрування SharePoint](https://docs.microsoft.com/sharepoint/get-started-new-admin-center).
