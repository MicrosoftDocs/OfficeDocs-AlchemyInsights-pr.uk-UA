# <a name="delete-a-folder"></a>Видалення папки

1. У лівій області клацніть правою кнопкою миші папку або вкладену папку, які потрібно видалити і виберіть **видалити папку**.
2. Виберіть **"OK"** для підтвердження.

Щоб отримати додаткові відомості, див. [видалення папки в Outlook.com](https://go.microsoft.com/fwlink/p/?linkid=873134).
