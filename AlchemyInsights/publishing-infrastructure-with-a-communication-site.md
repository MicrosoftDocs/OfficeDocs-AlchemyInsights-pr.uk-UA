# <a name="publishing-infrastructure-with-a-communication-site"></a>Видавничий інфраструктури з повідомлення сайту

Увімкнення функцію SharePoint Server Видавничий інфраструктури є **не підтримується** поточним зв'язку сайтів. Будь ласка, надайте відгук у [UserVoice](https://go.microsoft.com/fwlink/?linkid=2047322&amp;clcid=0x409). 
  
Для отримання додаткової інформації див [Увімкнути засобами публікації](https://support.office.com/en-us/article/Enable-publishing-features-479677A6-8B33-4AC7-907D-071C1C7E4518) .
