# <a name="publishing-infrastructure-with-a-communication-site"></a>Інфраструктура публікування з сайтом спілкування


Увімкнення функції інфраструктури публікування SharePoint Server **не підтримується** на сайтах зв'язку. 
  
Докладні відомості наведено в статті [перехід від публікації сайтів до сайтів зв'язку](https://docs.microsoft.com/sharepoint/publishing-sites-classic-to-modern-experience).
