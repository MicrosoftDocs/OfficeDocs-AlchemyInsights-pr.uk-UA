# <a name="publishing-infrastructure-with-a-communication-site"></a>Видавнича інфраструктура з комунікаційними сайтом


Увімкнення функції публікації інфраструктури SharePoint Server **не підтримується** комунікаційними сайтами. 
  
Для отримання додаткових відомостей див. [перехід від сайтів публікації до сайтів зв'язку](https://docs.microsoft.com/sharepoint/publishing-sites-classic-to-modern-experience).
