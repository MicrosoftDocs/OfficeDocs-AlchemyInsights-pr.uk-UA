# <a name="how-to-customize-permissions-for-a-sharepoint-list-or-library"></a>Настроювання дозволів для списку або бібліотеки SharePoint

Адміністратор або власник бібліотеки, списку або опитування дає змогу змінити дозволи, щоб дозволити одній групі користувачів отримувати доступ до даних, які їм потрібні під час обмеження інших.
