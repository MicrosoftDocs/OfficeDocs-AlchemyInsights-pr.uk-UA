# <a name="learn-about--the-mapping-between-microsoft-edge-legacy-policies-and-microsoft-edge-policies"></a>Відомості про зіставлення між політикою Microsoft EDGE Legacy і політиками Microsoft EDGE

Групові політики Microsoft EDGE Legacy (версії 45 і раніших), а також карти керування мобільним пристроєм (MDM) у відповідній політиці Microsoft EDGE, версії 80. Докладні відомості наведено в статті [Microsoft EDGE Legacy для зіставлення політики Microsoft EDGE](https://docs.microsoft.com/deployedge/microsoft-edge-policy-map-legacy-to-newedge).
