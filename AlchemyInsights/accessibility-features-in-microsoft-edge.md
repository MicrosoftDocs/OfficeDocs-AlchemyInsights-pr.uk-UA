# <a name="accessibility-features-in-microsoft-edge"></a>Функції спеціальних можливостей у Microsoft EDGE

Microsoft EDGE постачається з вбудованими параметрами спеціальних можливостей, які дають змогу переглядати веб-сторінки таким чином, що найкраще відповідає вашим потребам.

Щоб отримати докладні відомості, ознайомтеся [з функціями спеціальних можливостей у Microsoft EDGE](https://go.microsoft.com/fwlink/?linkid=2153648).
