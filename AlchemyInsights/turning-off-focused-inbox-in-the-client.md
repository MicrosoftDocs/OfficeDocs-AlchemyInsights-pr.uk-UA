# <a name="turn-off-focused-inbox-in-email-clients"></a>Вимкнення функції "важливі Вхідні" в клієнтах електронної пошти

Щоб вимкнути функцію "важливі Вхідні" в Outlook 2016, Інтернет-версії Outlook, Outlook.com/Hotmail.com, Outlook 2016 для Mac або пошта для Windows 10, перегляньте статтю [вимкнути функцію "важливі Вхідні"](https://support.office.com/article/f714d94d-9e63-4217-9ccb-6cb2986aa1b2.aspx).

Щоб вимкнути функцію "важливі Вхідні" для всіх користувачів в організації, ознайомтеся [з настроюванням папки "важливі Вхідні" для всіх користувачів вашої організації](https://docs.microsoft.com/microsoft-365/admin/setup/configure-focused-inbox).
