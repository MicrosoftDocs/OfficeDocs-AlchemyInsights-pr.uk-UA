# <a name="cant-add-default-2010-approval-workflow"></a>Не вдалося додати робочий цикл затвердження за замовчуванням 2010

У колекції сайтів Microsoft SharePoint не можна додати глобальний робочий цикл багаторазового використання (наприклад, "затвердження – SharePoint 2010") до списку або бібліотеки. Можливо, робочий цикл затвердження відсутній. Щоб вирішити цю проблему, зверніться до того, що [не вдалося додати глобальний робочий цикл багаторазового використання до списку або бібліотеки SharePoint](https://support.microsoft.com/help/4467263/sharepoint-designer-2013-shows-empty-wfpub-library).

Докладні відомості про затвердження робочого циклу наведено в статті [Усі відомості про робочі цикли затвердження](https://support.office.com/article/All-about-Approval-workflows-078C5A89-821F-44A9-9530-40BB34F9F742). 
 
Дізнайтеся більше про [сучасні затвердження в потоці](https://flow.microsoft.com/blog/introducing-modern-approvals).
