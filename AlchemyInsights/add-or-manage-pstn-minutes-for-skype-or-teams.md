# <a name="add-or-manage-pstn-minutes-for-skype-or-teams"></a>Додавання або керування хвилинами ТМЗК для Skype або команд

Якщо ви отримали попередження про те, що ваші хвилини Skype або робочі групи майже використовуються, ознайомтеся з інформацією про те, як [додавати кошти та керувати кредитами для вашого](https://docs.microsoft.com/microsoftteams/add-funds-and-manage-communications-credits)клієнта.
