# <a name="how-to-access-outlookcom-while-traveling"></a>Як отримати доступ до Outlook.com під час подорожей

Для доступу до Outlook.com під час подорожі, переконайтеся, що Microsoft облікового запису відомості про безпеку - послугами альтернативну електронну адресу або номер телефону можна добратися коди безпеки - в актуальному стані. Щоб дізнатися більше, побачити, [як отримати доступ до Outlook.com під час подорожі](https://support.office.com/article/c44f16da-7156-4890-853c-286aafeda87e?wt.mc_id=Office_Outlook_com_Alchemy).
