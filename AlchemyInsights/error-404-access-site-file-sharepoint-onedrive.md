# <a name="error-404-when-accessing-files-in-sharepoint-or-onedrive"></a>Помилка 404, коли доступ до файлів у SharePoint або OneDrive

**Помилка 404** отримав, коли користувачі намагаються отримати доступ до сайту або файлів SharePoint або OneDrive. Це може бути спричинене сайту або txt або група отримання перейменовано, переміщено або видалено.
Користувачі будуть випробовувати помилки 404, намагається отримати доступ до кореневої колекції сайтів та її видалено.

Щоб усунути помилку 404 для сайту, який має перейменовано, переміщено або видалено: класичний сайтів, які існують в центру адміністрування класичний дізнатися [відновити видалену вузлів](https://docs.microsoft.com/sharepoint/restore-deleted-site-collection).

Для сучасних сайтів (зв'язку, підключених до групи, або інші сайти) які існують в новий центр admin попереднього перегляду переглянути [подання та відновлення видалених сайтів нового центру адміністрування SharePoint](https://docs.microsoft.com/sharepoint/view-and-restore-deleted-sites-in-new-admin-center).

Щоб вирішити файл (або іншого елемента) що перейменовано, переміщено або видалено, перейдіть на сайт SharePoint або OneDrive і переглянути кошика від змісту сайту, побачити [відновити елементи з кошика сайту SharePoint](https://support.office.com/article/Restore-items-in-the-Recycle-Bin-of-a-SharePoint-site-6df466b6-55f2-4898-8d6e-c0dff851a0be).

 Якщо ви не можете знайти елемент, можна здійснювати пошук у журналі аудиту Якщо журналювання ввімкнуто, побачити [пошуку журнал аудиту в Office 365 безпеки &amp; відповідно центр](https://support.office.com/client/search-the-audit-log-in-the-office-365-security-compliance-center-0d4d0f35-390b-4518-800e-0c7ec95e946c).
