# <a name="make-microsoft-bing-your-default-search-engine"></a>Перетворення Microsoft Bing на стандартну пошукову систему

Щоб зробити Bing стандартною пошуковою системою Microsoft Edge, Google Chrome або Internet Explorer, дотримуйтеся вказівок у статті Створення [Bing пошукової](https://go.microsoft.com/fwlink/?linkid=2148834)системи за замовчуванням.
