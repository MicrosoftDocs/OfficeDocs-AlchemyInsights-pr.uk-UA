# <a name="calendar-sharing"></a>Спільний доступ до календаря

Якщо у вас виникли проблеми з доступом до календаря або інших календарів, використовуйте [помічник із підтримки й відновлення Microsoft для microsoft 365](https://diagnostics.office.com/).
  
Докладні відомості про надання спільного доступу до календаря наведено в статті [надання спільного доступу до календаря Outlook іншим користувачам](https://support.office.com/article/353ed2c1-3ec5-449d-8c73-6931a0adab88.aspx).
