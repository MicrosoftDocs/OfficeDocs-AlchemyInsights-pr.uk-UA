# <a name="software-updates-are-not-being-deployed"></a>Оновлення програмного забезпечення не розгортатиметься

Якщо ви налаштували оновлення програмного забезпечення за допомогою кільця оновлення Windows 10, але оновлення не розгорнуто, виконайте наведені нижче дії.  

- Спробуйте змінити обслуговування Windows із  **Напівщорічного**  типу випуску каналів на більш суворий і частий тип випуску.
- Установіть прапорець деферальний період для  **оновлення якості**  та  **оновлення функцій**. Період дефермального може призвести до затримки в оновленні оновлень до 180 днів.
