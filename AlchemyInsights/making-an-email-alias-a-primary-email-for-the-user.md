# <a name="make-an-email-alias-the-primary-address-for-a-user"></a>Створення псевдоніма електронної пошти основної адреси для користувача

Щоб отримати вказівки, перегляньте статтю [Додати іншого псевдоніма електронної пошти для користувача](https://docs.microsoft.com/microsoft-365/admin/email/add-another-email-alias-for-a-user).

**Примітка**. Якщо ваша організація перебуває в гібридному режимі, потрібно змінити основну адресу електронної пошти в локальній організації Exchange. Докладні відомості наведено в статті [Додавання та видалення адрес електронної пошти для поштової скриньки](https://technet.microsoft.com/library/bb123794.aspx).
