# <a name="onedrive-troubleshoot-processing-changes"></a>OneDrive: Виправлення 'Обробки зміни'

Якщо ви бачите "Обробки зміни" протягом тривалого періоду часу, це може бути тому, що у вас є онлайн файл відкритий, купою архівів застряг в черзі, або дуже великий файл синхронізацію, побачити [OneDrive застряг на обробку](https://support.office.com/article/onedrive-is-stuck-on-processing-changes-b386b813-9b66-4e47-8c4c-2b45533edccd).

Прочитайте цю статтю, виправлення неполадок "Обробки змін". Ви також побачите це повідомлення, якщо у вас є файл, який змінює весь час (наприклад, Outlook. PST файлу або бази даних). Дізнайтеся, як [перемістити файли даних Outlook](https://support.office.com/article/how-to-remove-an-outlook-pst-data-file-from-onedrive-b6b9e522-59bd-40f7-949f-168d0aa9b38e) , ви використовуєте регулярно з OneDrive.
