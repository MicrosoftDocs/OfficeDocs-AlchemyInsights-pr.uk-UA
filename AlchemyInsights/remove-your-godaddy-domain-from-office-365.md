# <a name="remove-your-godaddy-domain-from-microsoft-365"></a>Видаліть GoDaddy домену з Microsoft 365

Ми не можемо видалити ваш домен GoDaddy від Microsoft 365 для вас. Зателефонуйте [GoDaddy обслуговування клієнтів](https://aka.ms/contact-godaddy) за допомогою.
  
Якщо назва організації не містить "NETORG" або не придбано в GoDaddy, видаліть домен, виконавши наведені нижче дії, [видаліть домен Microsoft 365.](https://docs.microsoft.com/microsoft-365/admin/get-help-with-domains/remove-a-domain)
