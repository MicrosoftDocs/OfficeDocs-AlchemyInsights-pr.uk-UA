# <a name="why-cant-i-edit-this-file"></a>Чому я не можу редагувати цей файл?

Якщо документ, аркуш або презентацію заблоковано для редагування, і його не можна редагувати в програмах для настільних комп'ютерів Windows, можливо, файл редагується іншим користувачем, або файл узято витягнутим.

Ознайомтеся з деякими з причин, через [які ви не зможете редагувати файл](https://support.office.com/article/why-can-t-i-edit-this-file-97315f48-aa5e-49d3-a4ae-a14b73daf87b).
