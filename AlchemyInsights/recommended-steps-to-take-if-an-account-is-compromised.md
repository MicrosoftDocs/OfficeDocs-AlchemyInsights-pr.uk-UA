1. <span data-ttu-id="90d6b-p101">[Скидання пароля користувача](https://support.office.com/article/7a5d073b-7fae-4aa5-8f96-9ecd041aba9c) відразу. 2. <span data-ttu-id="90d6b-106">Видалити будь-які підозрілі [пересилання адреси](https://support.office.com/article/ab5eb117-0f22-4fa7-a662-3a6bdb0add74) встановлена на рівні поштової скриньки.</span><span class="sxs-lookup"><span data-stu-id="90d6b-106">Remove any suspicious [forwarding addresses](https://support.office.com/article/ab5eb117-0f22-4fa7-a662-3a6bdb0add74) set at the mailbox level.</span></span> 
    
3. <span data-ttu-id="90d6b-107">Видалити будь-які підозрілі [Вхідні правила](https://support.office.com/article/1433E3A0-7FB0-4999-B536-50E05CB67FED) Верзилії поштової скриньки.</span><span class="sxs-lookup"><span data-stu-id="90d6b-107">Remove any suspicious [inbox rules](https://support.office.com/article/1433E3A0-7FB0-4999-B536-50E05CB67FED) set within the mailbox.</span></span> 
    
4. <span data-ttu-id="90d6b-p102">Якщо користувач намагається перейти відправка електронної пошти, [перейдіть на обмежені користувачі розблокувати обліковий запис](https://protection.office.com/?hash=/restrictedusers). 5. <span data-ttu-id="90d6b-110">Видалити обліковий запис користувача з будь-якої [адміністративної рольових груп](https://support.office.com/article/eac4d046-1afd-4f1a-85fc-8219c79e1504) , поки ви впевнені, що рахунок більше не скомпрометована.</span><span class="sxs-lookup"><span data-stu-id="90d6b-110">Remove the user account from any [administrative role groups](https://support.office.com/article/eac4d046-1afd-4f1a-85fc-8219c79e1504) until you are confident that the account is no longer compromised.</span></span>
