# <a name="create-a-folder-in-outlookcom"></a>Створіть папку в Outlook.com

1. У нижній частині панелі папок виберіть **нову папку**. Може знадобитися Прокрутіть вниз, на панелі папок бачити кнопку **Нова папка** .
2. У текстовому полі введіть ім'я для папки та натисніть клавішу Enter.

Щоб отримати додаткові відомості, див. [Створення папки в Outlook.com](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a?wt.mc_id=Office_Outlook_com_Alchemy).
