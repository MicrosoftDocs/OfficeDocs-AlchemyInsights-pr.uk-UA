# <a name="block-pop-up-windows-in-microsoft-edge"></a>Блокування спливаючих вікон у Microsoft EDGE

1. Перейти до **... (Настройки та інше)**  >  **Параметри**  >  **Дозволи для сайту**.
2. Виберіть пункт **спливаючі вікна та перенаправлення**.
3. Увімкніть **блок**.
