# <a name="how-to-type-special-characters-on-a-mac"></a>Як ввести спеціальні символи на комп'ютері Mac

Щоб ввести спеціальні символи на вашому Mac, відкрийте меню **спеціальні символи** з клавіатури ярлик ⌘ **команди** + **контроль** + **простору**.

Ярлик для @ є також **ALT** + **64**.
