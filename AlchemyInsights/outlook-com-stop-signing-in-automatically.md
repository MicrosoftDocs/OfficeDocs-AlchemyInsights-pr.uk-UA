# <a name="how-to-stop-signing-in-automatically"></a>Як зупинити входу автоматично

Щоб зупинити, виконуючи вхід у службу Outlook.com автоматично, зніміть прапорець для **тримати мене в системі** на сторінці, де ви вводите пароль.
  
Щоб дізнатися більше, див [статтю вхід до або з Outlook.com](https://support.office.com/article/e08eb8ac-ac27-49f4-a400-a47311e1ee7e?wt.mc_id=Office_Outlook_com_Alchemy).
