# <a name="live-events-in-microsoft-stream"></a>Концертні події в Microsoft Stream

Ви можете створити інтерактивні події за допомогою Microsoft Stream в організації. Ви можете запланувати, підготувати та доставити інтерактивні події для різних сценаріїв, таких як події для всієї компанії, оновлення керівництва тощо.

- [Створення події наживо в Microsoft Stream](https://docs.microsoft.com/stream/live-create-event)
- [Початок роботи з командами Microsoft "події" в реальному режимі](https://support.office.com/article/get-started-with-microsoft-teams-live-events-d077fec2-a058-483e-9ab5-1494afda578a)
- [Настроювання кодери для потокового передавання наживо в Microsoft Stream](https://docs.microsoft.com/stream/live-encoder-setup)
