# <a name="live-events-in-microsoft-stream"></a>Живі події в Microsoft Stream

Можна створювати живі події за допомогою Microsoft Stream через організацію. Ви можете запланувати, виготовити та доставити живі події для різних сценаріїв, таких як події в усьому компанії, оновлення лідерства тощо.

- [Живі події в Microsoft Stream](https://docs.microsoft.com/stream/live-event-overview)
- [Створення події в прямому ефірі в Microsoft Stream](https://docs.microsoft.com/stream/live-create-event)
- [Початок роботи з подіями Microsoft teams Live](https://support.office.com/article/get-started-with-microsoft-teams-live-events-d077fec2-a058-483e-9ab5-1494afda578a)
- [Настроювання кодери для прямої трансляції в Microsoft Stream](https://docs.microsoft.com/stream/live-encoder-setup)
