# <a name="create-sharepoint-sites-using-templates"></a>Створення сайтів SharePoint за допомогою шаблонів

Шаблони сайтів SharePoint є готові визначень призначені навколо потреби конкретного бізнесу. Докладніше, перегляньте <a href="https://support.office.com/en-us/article/using-templates-to-create-different-kinds-of-sharepoint-sites-449eccec-ff99-4cf3-b62e-dcfee37e8da4">використання шаблонів для створення різних видів сайтів SharePoint.</a></span></p> <p><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Нижче наведено деякі поширені питання/рішення щодо збереження через сайт або список як шаблон у Sharepoint Online.</span></p> <p><strong style="mso-bidi-font-weight: normal;"><u><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Зберегти сайт/список шаблон кнопка не буде доступна або відсутні.</span></u></strong></p> <ul> <li><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Адміністратори повинні дозволити користувацький скрипт для ввімкнення функції шаблон. Докладно процедуру, приклади і міркування в розділі </span> </span> <a style="orphans: 2; -webkit-text-stroke-width: 0px; word-spacing: 0px;" href="https://docs.microsoft.com/en-us/sharepoint/allow-or-prevent-custom-script">дозволити або заборонити власний сценарій</a>.</li> <li><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Сайт зберегти як шаблон команда не підтримується і може викликати проблеми на сайтах, які використовують Видавничий інфраструктури SharePoint Server.</span></li> </ul> <p><strong style="mso-bidi-font-weight: normal;"><u><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Шаблон сайту неможливо створити або працює неправильно</span></u></strong></p> <ul> <li><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Шаблон можуть бути відсутні <a href="https://social.technet.microsoft.com/wiki/contents/articles/14423.sharepoint-2013-existing-features-guid.aspx">функції</a> і виграв&rsquo;t активувати. Якщо функція недоступна для активації в поточній колекції сайтів, це неможливо використовувати шаблон сайту для створення сайту.</span></li> <li><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Перевірте, якщо будь-які списки або бібліотеки перевищує <a href="https://support.office.com/en-us/article/Manage-large-lists-and-libraries-in-SharePoint-B8588DAE-9387-48C2-9248-C24122F07C59">Ліміт граничне значення подання списку з</a> 5000 пунктів, як це може блокувати створення шаблон сайту.</span></li> <li><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Можливо, сайт використовує забагато ресурсів і тому шаблон сайту перевищує обмеження в 50 МБ.</span></li> <li>
Є проблеми відображення даних зі списку, який використовує стовпець підстановки. Докладніше, перегляньте </span> <span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;"> <a style="box-sizing: border-box; -webkit-text-stroke-width: 0px; word-spacing: 0px;" href="https://support.office.com/en-us/article/template-generated-list-doesn-t-display-correct-data-for-a-column-in-sharepoint-online-20430b62-e40c-4f6f-8889-aa24e80d605a"> <span style="color: #0067b8; text-decoration: none; text-underline: none;">створеного шаблону списку doesn&rsquo;t відображення даних зі списку правильний підстановки в SharePoint Online.
