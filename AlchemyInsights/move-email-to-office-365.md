# <a name="move-email-to-microsoft-365"></a>Переміщення електронної пошти до Microsoft 365

У вас є кілька варіантів перенесення електронної пошти до Microsoft 365. [Порадник з міграції електронної пошти Microsoft 365](https://aka.ms/alchemyinsight-mailmigrationadvisor) дає змогу вибрати найкращу для вашого середовища, а також проведе вас через перенесення пошти.
  
Щоб дізнатися про свої варіанти, не запускаючи перенесення, перейдіть [на шлях перенесення](https://docs.microsoft.com/Exchange/mailbox-migration/decide-on-a-migration-path).

Крім того, ви можете перевірити, чи можна отримати довідку з перенесення [Microsoft 365 із сторінкою FastTrack](https://www.microsoft.com/fasttrack/microsoft-365/office-365) для допомоги з міграцією.
