# <a name="move-email-to-microsoft-365"></a>Перемістити повідомлення до Microsoft 365

У вас є кілька варіантів для міграції електронної пошти до корпорації Майкрософт 365. [Порадник з міграції до Microsoft 365](https://aka.ms/alchemyinsight-mailmigrationadvisor) допоможе вам вибрати найкращий для вашого середовища, а також проведе вас через міграцію пошти.
  
Щоб прочитати про варіанти, не почавши міграцію, перегляньте [рішення щодо шляху міграції](https://docs.microsoft.com/Exchange/mailbox-migration/decide-on-a-migration-path).

Будь ласка, також перевірте [прийняти Microsoft 365 з FastTrack](https://www.microsoft.com/fasttrack/microsoft-365/office-365) сторінку для допомоги з міграцією, якщо це право.
