# <a name="microsoft-bookings-payment-options"></a>Способи оплати за бронювання Microsoft

Параметри оплати в Microsoft-бронювання були вийшли на пенсію 2020 березня. Організації тепер використовують рішення сторонніх виробників, щоб додати спосіб оплати резервування.
