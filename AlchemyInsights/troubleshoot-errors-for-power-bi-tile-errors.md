# <a name="troubleshooting-refresh-scenarios---tile-errors"></a>Усунення неполадок refresh - плитка помилки

Якщо у вас виникли плитка refresh помилки в рамках влади BI послуги, будь ласка, почекайте близько 10-15 хвилин для оновлення може бути відображено в dashboard плитки. Якщо він все ще не показують, повторно закріпити візуалізації до приладної дошки.

Для отримання додаткової інформації:[https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios](https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios)
