# <a name="how-to-manage-add-ins-while-reading-a-message"></a>Керування надбудовами під час читання повідомлення

1. У веб-програмі Outlook в Інтернеті виберіть повідомлення.
    
2. У верхній частині повідомлення виберіть елемент **додаткові дії** .

3. Перейдіть до нижньої частини списку та виберіть пункт **отримати надбудови**.
    
4. На сторінці надбудови для Outlook виберіть потрібну надбудову.
    
    Для безкоштовних надбудов натисніть кнопку Add ( **Додати** ), щоб додати його до поштової скриньки.
    
    Для платних надбудов і безкоштовних ознайомлювальних версій натисніть кнопку **отримати** , щоб почати придбання.
    
Докладні відомості наведено в статті [використання надбудов в Інтернет-версії Outlook](https://support.microsoft.com/office/using-add-ins-in-outlook-on-the-web-8f2ce816-5df4-44a5-958c-f7f9d6dabdce).
