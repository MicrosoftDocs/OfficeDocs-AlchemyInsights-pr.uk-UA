# <a name="use-your-browser-to-check-spelling"></a>Перевірка правопису за допомогою браузера

Немає варіанту перевірки орфографії в Outlook.com. Для перевірки правопису, потрібно буде використовувати свій веб-браузер. Перевірка правопису доступний Microsoft краю, Internet Explorer 10 та пізнішими версіями та поточні версії Firefox, Chrome і Safari. Перевірте параметри веб-браузера, щоб дізнатися більше про те, як перевіряти орфографію.
