# <a name="sharepoint-online-calendar-overlay"></a>Накладання календаря SharePoint Online

Ви можете накладати ще один календар SharePoint, оскільки обидва календарі SharePoint розташовані в тій самій колекції сайтів. Можливість використовувати накладання календаря в календарі SharePoint Online або SharePoint Server, який отримано з Exchange Online, не підтримується.

Служба SharePoint Online пропонує сучасний досвід роботи з календарями групи Microsoft 365. Докладні відомості наведено в статті [використання веб-частини "Календар групи"](https://support.microsoft.com/en-us/office/use-the-group-calendar-web-part-eaf3c04d-5699-48cb-8b5e-3caa887d51ce).
