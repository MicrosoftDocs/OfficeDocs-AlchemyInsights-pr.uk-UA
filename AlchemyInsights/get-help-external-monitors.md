# <a name="get-help-with-external-monitors"></a>Отримання довідки з зовнішніх моніторів

Ви можете легко відкрити свої параметри проекту, одночасно натиснувши такі клавіші:

- Клавіша Windows (кнопка Пуск із емблемою Windows) + P
- Виберіть один із **повторюваних**, **подовжувати** або **другий екран лише** в залежності від ваших уподобань.

Якщо у вас виникли проблеми з зовнішніми моніторами, виконайте наведені нижче дії з виправлення неполадок.

- Від'єднайте кабель для підключення ПК до монітора та підключіть його знову.
- Відключіть всі несуттєві пристрої на ПК (наприклад, адаптери або доки).

- [Використання кількох моніторів у Windows 10](https://support.microsoft.com/windows/how-to-use-multiple-monitors-in-windows-10-329c6962-5a4d-b481-7baa-bec9671f728a)
