# <a name="create-users-in-hybrid-deployments"></a>Створення користувачів у гібридному розгортанні

Якщо організація є гібридним розгортанням між локальним середовищем і Microsoft 365, потрібно створити всі об'єкти користувача в локальній службі Active Directory. Створення об'єктів користувача у відповідному організаційному підрозділі (ОП), тому вони синхронізуються в організації.
