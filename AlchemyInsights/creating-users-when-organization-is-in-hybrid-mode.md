# <a name="create-users-in-hybrid-deployments"></a>Створення користувачів у гібридному розгортаннях

Якщо ваша організація – це гібридне розгортання між локальним середовищем і Microsoft 365, потрібно створити всі об'єкти користувача в локальній службі Active Directory. Ви створюєте об'єкти користувача у відповідному організаційному підрозділі (OU), щоб вони синхронізуватися в свою організацію.
