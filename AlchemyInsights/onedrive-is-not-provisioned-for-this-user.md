# <a name="onedrive-is-not-provisioned-for-this-user"></a>OneDrive не підготовлено для цього користувача

Для новостворених облікових записів користувачам потрібно ввійти до центру адміністрування Microsoft 365 за допомогою власного облікового запису та клацнути OneDrive, щоб вручну запустити надання OneDrive сайту.
  
Однак, якщо користувач вже натиснув OneDrive і їх персональний сайт не підготовлено протягом 24 годин, будь ласка, увійдіть підтримку квитка для подальшої допомоги.
