# <a name="cant-download-files-in-chrome"></a>Не вдається завантажити файли в Chrome

**Питання** Нам відомо про проблему, яка перешкоджає деякі клієнти в змозі завантажувати файли, коли за допомогою веб-переглядача Google Chrome. 

**Обхідний шлях** За допомогою [Microsoft краю](https://www.microsoft.com/en-gb/windows/microsoft-edge) або інший веб-браузер.
Дізнайтеся більше про те, як [завантаження файлів, що передаються Outlook.com](https://support.office.com/article/8d7c1ea7-4e5f-44ce-bb6e-c5fcc92ba9ab).
