# <a name="got-a-parameter-cannot-be-found-error"></a>Помилка "не вдалося знайти параметр"?

Якщо з'являється повідомлення про помилку **"не вдалося знайти параметр, який відповідає імені параметра" EmailAddresses "**, це означає, що це трохи більше часу для Microsoft 365, щоб завершити настроювання клієнта або настроюваного домену, якщо ви нещодавно додали одну.
  
Процес інсталяції може тривати до 4 годин, щоб завершити. Зачекайте деякий час, щоб процес настроювання встиг завершити, а потім повторіть спробу.
