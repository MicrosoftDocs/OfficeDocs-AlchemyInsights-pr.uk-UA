# <a name="how-do-i-turn-off-skype-in-outlookcom"></a>Як можна вимкнути Skype в Outlook.com?

Не вдалося видалити Skype з Outlook.com, але можна вимкнути сповіщення:

1. Відкрийте **Параметри** > **загальні** > **[сповіщення](https://go.microsoft.com/fwlink/?linkid=2031594)**. 
2. Під **чат і дзвінок**вимкнути сповіщення про типи непотрібних.

Для детальнішої інформації про використання Skype читати, [розмовляти з родиною та друзями по Skype в Outlook.com](https://support.office.com/article/83c6a5b1-3921-479c-b9e9-e753ce59c1fa).
