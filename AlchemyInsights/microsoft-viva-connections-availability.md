# <a name="microsoft-viva-connections-availability"></a>Доступність підключень Microsoft Viva

Підключення Microsoft Viva буде доступно для настільних комп'ютерів у першій половині 2021 і для мобільних пристроїв влітку 2021. Додаткові функції, які будуть відкотитися протягом 2022.

Щоб отримати докладні відомості, Дізнайтеся, як [зв'язки Microsoft Viva допомагають всім працювати та отримувати інформацію](https://techcommunity.microsoft.com/t5/microsoft-viva-blog/microsoft-viva-connections-helps-everyone-to-stay-engaged-and/ba-p/2107009).
