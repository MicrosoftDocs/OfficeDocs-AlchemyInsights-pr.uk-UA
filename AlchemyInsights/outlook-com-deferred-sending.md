# <a name="delay-or-schedule-sending-email-messages"></a>Затримка або розклад надсилання повідомлень електронної пошти

Вибачте, але ви не можете відкласти або Запланувати надсилання повідомлень електронної пошти в мобільному додатку або веб-переглядачі. Для цього скористайтеся [Outlook для Windows або Outlook для Mac](https://products.office.com/outlook/email-and-calendar-software-microsoft-outlook).

**Скасувати надсилання**

Ви можете змінити налаштування Outlook.com, щоб відкласти надсилання повідомлення на 10 секунд. Ви можете скасувати надсилання повідомлення протягом цього періоду. Щоб увімкнути цю функцію, відкрийте параметри [скасування надсилання](https://outlook.live.com/mail/options/mail/messageContent/undoSend) .

Дізнайтеся більше про [відкликання електронної пошти в Outlook.com](https://support.office.com/article/c069ddde-5282-4085-8f4c-d7b133324f8a?wt.mc_id=Office_Outlook_com_Alchemy).
