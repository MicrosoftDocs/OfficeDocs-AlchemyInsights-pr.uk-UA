# <a name="delay-or-schedule-sending-email-messages"></a>Затримка або планування надсилання повідомлень електронної пошти

На жаль, ви не можете відкласти або Запланувати надсилання повідомлень електронної пошти у веб-програмі для мобільних пристроїв або в браузері. Для цього скористайтеся [програмою Outlook для Windows або Outlook для Mac](https://products.office.com/outlook/email-and-calendar-software-microsoft-outlook).

**Скасування дії "Відправити"**

Ви можете змінити настройки Outlook.com, щоб відкласти надсилання повідомлення на щонайбільше 10 секунд. Ви можете скасувати надсилання повідомлення протягом цього періоду. Щоб увімкнути цю функцію, відкрийте параметри [скасувати надіслані](https://outlook.live.com/mail/options/mail/messageContent/undoSend) .

Дізнайтеся більше про [відкликання повідомлень електронної пошти в Outlook.com](https://support.office.com/article/c069ddde-5282-4085-8f4c-d7b133324f8a?wt.mc_id=Office_Outlook_com_Alchemy).
