# <a name="removing-the-last-on-premises-exchange-server"></a>Видалення останнього на локальному сервері Exchange

Перед видаленням останнього сервера Exchange від локальної організації, слід переконатися, ви розумієте наслідки робимо це, щоб допомогти вам вирішити, якщо вам потрібно що останній сервер для інших цілей. Для отримання додаткової інформації, ознайомтеся з [як і коли, щоб вивести з експлуатації вашого локальні сервери Exchange співіснування](https://technet.microsoft.com/library/dn931280%28v=exchg.150%29.aspx).
