# <a name="removing-the-last-on-premises-exchange-server"></a>Видалення останнього локального сервера Exchange

Перш ніж видалити останній сервер Exchange із локальної організації, ви повинні переконатися, що ви розумієте наслідки цього, щоб допомогти вам вирішити, якщо вам потрібен останній сервер для будь-яких інших цілей. Щоб отримати додаткові відомості, перевірте, [як і коли для розпакування ваш локальний Exchange серверів гібридне розгортання](https://technet.microsoft.com/library/dn931280%28v=exchg.150%29.aspx).
