# <a name="removing-the-last-on-premises-exchange-server"></a>Видалення останнього локального сервера Exchange

Перш ніж видалити останній сервер Exchange із локальної організації, слід переконатися, що ви усвідомлюєте наслідки цього, щоб вирішити, чи потрібно використовувати останній сервер для будь-яких інших цілей. Щоб отримати докладні відомості, Дізнайтеся, [як і коли потрібно вивести з редагування локальні сервери Exchange у гібридному розгортанні](https://technet.microsoft.com/library/dn931280%28v=exchg.150%29.aspx).
