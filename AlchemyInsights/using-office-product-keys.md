# <a name="using-office-product-keys"></a>Використання ключів продукту Office

**Ключі продуктів Office**

Використовуйте обліковий запис Microsoft, а не ключ продукту, щоб інсталювати та активувати Office, а також окремо придбані програми Microsoft 365, такі як Project, Visio, Word, Excel і Outlook. Щоб отримати додаткові відомості, див [Office, запитувати ключ продукту](https://support.office.com/article/12a5763a-d45c-4685-8c95-a44500213759?ui=en-US&rs=en-US&ad=US#bkmk_promptforpkey)

- Якщо ви придбали Office разом із ключем продукту, Дізнайтеся, [де ввести ключ продукту Office](https://support.office.com/article/Where-to-enter-your-Office-product-key-0a82e5ae-739e-4b92-a6f4-2ec780c185db).

- За допомогою Office 365 домашній або персональний ключ продукту дивіться [використання ключів продукту з офісом](https://support.office.com/article/using-product-keys-with-office-12a5763a-d45c-4685-8c95-a44500213759).

**Активація передплати на Office** 

- Якщо ви придбали передплату Microsoft 365 з роздрібного магазину, перегляньте [розділ введіть ключ продукту для застосунків Microsoft 365 для бізнесу](https://docs.microsoft.com/microsoft-365/commerce/enter-your-product-key).

- Якщо під час спроби активувати ключ продукту з'являється повідомлення про помилку, перегляньте [проблеми з Microsoft 365 для ключа продукту для бізнесу?](https://docs.microsoft.com/microsoft-365/commerce/product-key-errors-and-solutions)
