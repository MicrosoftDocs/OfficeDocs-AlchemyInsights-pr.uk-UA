# <a name="adding-a-mailbox-to-a-user"></a>Додавання поштової скриньки користувачу

Щоб додати поштову скриньку до вже створеного користувача або нового користувача, користувачу потрібно призначити ліцензію для Exchange Online. Якщо користувач уже має ліцензії, виберіть Exchange Online в розділі програми на сторінці ліцензії, щоб розпочати процес створення поштової скриньки для користувача. Для отримання додаткових відомостей див. [призначення ліцензій користувачам](https://docs.microsoft.com/microsoft-365/admin/manage/assign-licenses-to-users).
