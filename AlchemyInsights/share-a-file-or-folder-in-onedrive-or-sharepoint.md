# <a name="share-a-file-or-folder-in-onedrive-or-sharepoint"></a>Спільний доступ до файлу або папки в OneDrive або SharePoint

1. Перейдіть до файлу або папки, які ви хочете поділитися, виберіть його та натисніть кнопку **спільний доступ**.
    
2. (Необов'язково) У діалоговому вікні "Надіслати посилання" натисніть стрілку вниз, щоб змінити тип посилання. Під **які ви хочете це посилання функціонувало для**, зміни, які можуть перейти по посиланню. Якщо ви хочете мати можливість переглядати, але не редагувати файли, зніміть прапорець **Дозволити редагування** . 
    
    Натисніть кнопку **Застосувати**.
    
    > [!NOTE]
    > Якщо спільне використання потрібний параметр недоступний, ви повинні переконатися, допускається для вашої організації, так і для колекції сайтів. Для цього треба переглянути [Увімкнення зовнішніх обміну або вимкнути для SharePoint Online](https://go.microsoft.com/fwlink/?linkid=866426). 
  
3. Введіть імена людей, які ви бажаєте дозволити спільний доступ і натисніть кнопку **Надіслати**.
