# <a name="outlook-error-cannot-start-microsoft-outlook"></a>Помилка Outlook: не вдається запустити Microsoft Outlook

Виникли проблеми з запуском Outlook 2016, Outlook 2013 або Outlook 2010 або отримання "не вдається запустити Microsoft Outlook. Не вдається відкрити вікно Outlook "повідомлення про помилку?
  
По-перше, скористайтеся порадами з виправлення неполадок у [цій статті](https://support.office.com/article/I-can-t-start-Microsoft-Outlook-2016-2013-or-2010-or-receive-the-error-Cannot-start-Microsoft-Office-Outlook-Cannot-open-the-Outlook-Window-d1f69da6-b333-4650-97bf-4d77bd7abb85) , щоб допомогти вирішити ці проблеми з запуском. 
  
Також можна запустити [засіб розширеного діагностики](https://aka.ms/SaRA-OutlookAdvDiagnostics) на проблемному комп'ютері.
