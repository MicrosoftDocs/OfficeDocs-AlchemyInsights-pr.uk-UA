# <a name="install-and-configure-onedrive"></a>Інсталяція та настроювання служби "OneDrive"

Клієнт синхронізації OneDrive попередньо інстальовано в ОС Windows 10. Щоб інсталювати службу OneDrive для Windows 7 або Mac або отримати програму для мобільних пристроїв на телефоні або планшеті, перейдіть на сторінку [https://onedrive.com/download](https://onedrive.com/download) .
  
Щоб настроїти клієнт синхронізації OneDrive у Windows, див.:
  
- [Настроювання комп'ютера для синхронізації файлів у службі "OneDrive для бізнесу"](https://go.microsoft.com/fwlink/?linkid=533375)

- [Синхронізація файлів SharePoint за допомогою нового клієнта синхронізації OneDrive](https://go.microsoft.com/fwlink/?linkid=871666)
