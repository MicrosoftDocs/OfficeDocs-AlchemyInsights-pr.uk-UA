# <a name="install-and-configure-onedrive"></a>Інсталяція та настроювання OneDrive

Клієнт синхронізації OneDrive попередньо інстальовано з Windows 10. Щоб інсталювати OneDrive для Windows 7 або Mac або отримати мобільну програму для телефона або планшета, перейдіть до [https://onedrive.com/download](https://onedrive.com/download).
  
Щоб настроїти клієнт синхронізації OneDrive у Windows, див.:
  
- [Настроювання комп'ютера для синхронізації файлів OneDrive для бізнесу](https://go.microsoft.com/fwlink/?linkid=533375)

- [Синхронізація файлів SharePoint за допомогою нового клієнта синхронізації OneDrive](https://go.microsoft.com/fwlink/?linkid=871666)
