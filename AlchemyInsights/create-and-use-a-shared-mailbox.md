# <a name="create-and-use-a-shared-mailbox"></a>Створення та використання спільної поштової скриньки

Спільної поштової скриньки дає змогу кільком надсилати й отримувати електронну пошту для псевдонім, як sales@contoso.com, без призначення іншого ліцензії.
  
- [Відразу тут, щоб отримати роботу](https://portal.office.com/AdminPortal/Home#/AssistedGuide/addemailoptions)
    
Ви також можете прочитати статтю [створити спільну поштову скриньку](https://support.office.com/article/https://support.office.com/article/Create-a-shared-mailbox-871a246d-3acd-4bba-948e-5de8be0544c9.aspx) , яка розповість вам усе, що вам потрібно знати про спільні поштові скриньки, включаючи те, як використовувати їх.
