# <a name="create-and-use-a-shared-mailbox"></a>Створення та використання спільної поштової скриньки

Спільна поштова скринька дозволяє більше однієї людини надсилати та отримувати повідомлення електронної пошти для псевдоніма, наприклад sales@contoso.com, без призначення іншої ліцензії.
  
- Щоб розпочати роботу, [виконайте наведені нижче дії](https://portal.office.com/AdminPortal/Home#/AssistedGuide/addemailoptions) в центрі адміністрування Microsoft 365. 

Також можна прочитати статтю [створення спільної поштової скриньки](https://docs.microsoft.com/office365/admin/email/create-a-shared-mailbox) , яка повідомить Вам усе, що потрібно знати про спільні поштові скриньки, зокрема способи їх використання.
