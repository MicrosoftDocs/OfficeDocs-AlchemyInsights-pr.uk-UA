# <a name="import-contacts"></a>Імпорт контактів

Перед тим, як імпортування контактів з Outlook в Інтернеті, вам потрібно експортувати їх від іншого постачальника послуг електронної пошти як CSV-файл. Дотримуйтесь інструкцій нижче.

1. У нижній частині сторінки виберіть піктограму людей, щоб перейти на сторінку "люди".
2. На панелі інструментів виберіть **керування** > **імпорту контактів**.
3. Виберіть **Огляд**, вибрати файл CSV а потім виберіть **Відкрити**.
4. Виберіть **імпортувати**.

Щоб дізнатися більше, див [імпорту контактів для Outlook](https://support.office.com/article/bb796340-b58a-46c1-90c7-b549b8f3c5f8#ID0EAACAAA=Outlook_on_the_web).
