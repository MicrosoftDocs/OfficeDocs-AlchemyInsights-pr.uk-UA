# <a name="cant-add-or-change-teams-profile-picture"></a>Не вдається додати або змінити зображення профілю групи

Щоб додати або змінити зображення профілю, користувачі мають бути повністю доступні, виконавши наведені нижче дії.

- Exchange Online
- Служба SharePoint Online
- Можливість створювати групи Microsoft 365

Ознайомтеся з адміністратором команди про те, як отримати необхідні дозволи. Докладні вказівки наведено [в статті змінення зображення в командах](https://support.microsoft.com/office/change-your-picture-in-teams-7a711943-9248-420e-b814-c071aa8d9b9c).
