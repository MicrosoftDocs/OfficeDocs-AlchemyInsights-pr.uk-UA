# <a name="hybrid-and-federated-searches"></a>Гібридні та Федеративні пошукові запити 

Відомості про змішаний та інтегрований пошук у SharePoint Online наведено в режимі [змішаний пошук у SharePoint](https://docs.microsoft.com/sharepoint/hybrid/hybrid-search-in-sharepoint).

Відомості про планування змішаний федеративного пошуку переглянути [план змішаний інтегрований пошук для SharePoint Server](https://docs.microsoft.com/sharepoint/hybrid/plan-hybrid-federated-search).
