# <a name="changes-to-external-user-access"></a>Зміни для зовнішніх користувачів доступ

Починаючи 23 березня 2018 року, зовнішнім користувачам більше не буде мати доступ до вмісту, який було спільним зі всім, всім автентифікованим користувачам або всіх форм користувачів, навіть якщо вони запросили або додані до статутного груп. Зовнішні користувачі тільки буде мати доступ до вмісту, який був спільно з користувача або групи, до яких належить користувач. Якщо ви хотіли б зберегти існуючі налаштування, будь ласка, дивіться:https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
