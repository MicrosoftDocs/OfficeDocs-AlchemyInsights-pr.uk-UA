# <a name="public-folder-removed-from-outlook-on-the-web-owa-favorites"></a>Спільна папка, видалена Outlook в Інтернеті (OWA) уподобання

Відома проблема, коли спільні папки видаляються з уподобань у веб-програмі OWA під час кожного виходу з програми або оновлення екрана. Цю проблему буде вирішено найближчим часом. До цього знову додайте папку для доступу з OWA.
