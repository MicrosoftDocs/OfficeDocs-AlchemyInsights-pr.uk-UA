# <a name="how-to-right-click-on-a-tablet"></a>Як правою кнопкою мишки на планшет

Клікнути правою кнопкою на поверхні або Windows планшет, натисніть і утримуйте виділення за лічені секунди. Якщо відпустити, з'явиться меню правої кнопки миші.
