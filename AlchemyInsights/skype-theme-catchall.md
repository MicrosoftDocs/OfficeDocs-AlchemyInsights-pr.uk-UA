# <a name="skype-for-business-common-issues-and-resolutions"></a>Поширені проблеми та вирішення в службі "Skype для бізнесу" 

Якщо у вас є конкретні запитання про програму "Skype для бізнесу", стисло перефразуйте своє запитання, щоб описати свою проблему, зокрема помилки, функції, які ви використовуєте тощо. 

Нижче наведено кілька поширених тем.

**Усунення несправностей із входом у "Skype для бізнесу"** Спочатку переконайтеся, що [служба "Skype для бізнесу"](https://admin.microsoft.com/Adminportal/Home?source=applauncher#/servicehealth) справна. Якщо це так, ознайомтеся зі реєстром Виправлення помилок входу в ["Skype](https://docs.microsoft.com/SkypeForBusiness/set-up-skype-for-business-online/troubleshooting-sign-in-errors-for-admins#check-for-common-causes-of-skype-for-business-online-sign-in-errors) для бізнесу – онлайн", щоб усунути поширені причини труднощів із входом.
 
**Усунення несправностей, пов'язаних із аудіо та відео в "Skype для бізнесу":** Перегляньте [способи вирішення поширених проблем зі звуком і відео в "Skype для бізнесу".](https://support.office.com/article/Troubleshoot-audio-and-video-in-Skype-for-Business-62777bc6-c52b-47ae-84ba-a8905c3b71dc) 

**Виправлення неполадок із підключенням у "Skype для бізнесу"** Перегляньте [способи вирішення поширених проблем з підключенням у "Skype для бізнесу".](https://support.office.com/article/troubleshoot-connection-issues-in-skype-for-business-ca302828-783f-425c-bbe2-356348583771)

**Спілкування із зовнішніми контактами Skype для бізнесу (федерація):** налаштуйте Skype для бізнесу на зв'язок із зовнішніми користувачами [Skype для бізнесу.](https://docs.microsoft.com/SkypeForBusiness/set-up-skype-for-business-online/allow-users-to-contact-external-skype-for-business-users)

**Спілкування з контактами Skype.** [Дозвольте користувачам Skype для бізнесу додавати контакти Skype.](https://docs.microsoft.com/SkypeForBusiness/set-up-skype-for-business-online/let-skype-for-business-users-add-skype-contacts)

**Оновіть "Skype для бізнесу" до Microsoft Teams:** Ознайомтеся [зі стилями](https://docs.microsoft.com/microsoftteams/upgrade-start-here) Початок роботи [](https://docs.microsoft.com/microsoftteams/coexistence-chat-calls-presence) з оновленням Microsoft Teams і дізнайтеся більше про співпрацю з Microsoft Teams під час роботи в організації.
