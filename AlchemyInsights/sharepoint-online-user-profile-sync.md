# <a name="sharepoint-online-user-profile-synchronization"></a>Синхронізація профілів користувачів служби SharePoint Online

Служба Microsoft SharePoint використовує завдання синхронізації Active Directory, щоб імпортувати відомості про атрибут користувача та групи до застосунку профілю користувача (УПА).Щоб отримати докладніші відомості, ознайомтеся зі статтею нижче.

- [Синхронізація профілів користувачів служби SharePoint Online](https://docs.microsoft.com/sharepoint/user-profile-sync)
