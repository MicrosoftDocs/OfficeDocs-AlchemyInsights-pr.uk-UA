# <a name="plan-your-passwordless-deployment"></a>Планування розгортання без пароля

Оновіть систему до альтернативного входу за допомогою методу автентифікації без пароля. Двофакторна перевірка, генератори телефону або генератори коду можуть підвищити рівень безпеки та змусити інших користувачів перенести ваш обліковий запис. 

Виберіть один із розділів безпеки Windows Hello для бізнесу, програми автентифікації Microsoft або FIDO2. Щоб визначити найкращий спосіб і як розгорнути їх на пристрої, скористайтеся [посібником із розгортання без пароля.](https://admin.microsoft.com/adminportal/home?#/modernonboarding/passwordlesssetup)
