- Для користувачів див. [змінення мови інтерфейсу та часового поясу у Microsoft 365 для бізнесу для всіх програм](https://support.microsoft.com/office/6f238bff-5252-441e-b32b-655d5d85d15b).
- Для клієнта перегляньте [Налаштування мови та регіональні параметри для Microsoft 365](https://docs.microsoft.com/office365/troubleshoot/access-management/set-language-and-region).
