# <a name="create-email-signatures"></a>Створення підписів електронної пошти

Перегляньте наведені нижче розділи, щоб дізнатися, як створювати підписи електронної пошти в програмі Outlook.
  
- Створіть підпис електронної пошти для власних повідомлень: [створення та Додавання підпису до повідомлень](https://support.office.com/article/8ee5d4f4-68fd-464a-a1c1-0e1c80bb27f2.aspx) або [створення та Додавання підпису електронної пошти у веб-програмі Outlook Web App](https://support.office.com/article/0f230564-11b9-4239-83de-f10cbe4dfdfc.aspx).
    
- Створіть глобальний підпис електронної пошти для всіх повідомлень, надісланих від компанії (також відомих як застереження): [застереження, підписи, нижні колонтитули або заголовки](https://go.microsoft.com/fwlink/p/?linkid=391096).
