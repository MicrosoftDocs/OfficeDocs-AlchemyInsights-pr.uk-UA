# <a name="port-google-chrome-extensions-to-microsoft-edge-chromium"></a>Порт Google Chrome із розширеннями Microsoft EDGE (хром)

Просто [порт Google Chrome можна легко перенести на Microsoft EDGE (хром)](https://docs.microsoft.com/microsoft-edge/extensions-chromium/developer-guide/port-chrome-extension). У більшості випадків потрібні лише мінімальні зміни, щоб запускати ці розширення на сайті Microsoft EDGE.

Інтерфейси API для розширень і Маніфесту, які підтримуються в Google Chrome, сумісні з кодами Microsoft EDGE. Тим не менш, Microsoft EDGE не підтримує розширення API Chrome. GPCM, Chrome. Identity. getAccounts, хром. Identity. Getauthмаркер і хром. instanceID.
