# <a name="port-google-chrome-extensions-to-microsoft-edge-chromium"></a>Порт розширень Google Chrome до Microsoft Edge (Chromium)

Ви легко переносите [розширення Google Chrome до Microsoft Edge (Chromium).](https://docs.microsoft.com/microsoft-edge/extensions-chromium/developer-guide/port-chrome-extension) Здебільшого для запуску цих розширень у Microsoft Edge потрібні лише мінімальні зміни.

API розширень і маніфестні ключі, які підтримуються Google Chrome, сумісні з Microsoft Edge. Проте в Microsoft Edge не підтримується розширення API chrome.gcm, chrome.identity.getAccounts, chrome.identity.getAuthToken і chrome.instanceID.
