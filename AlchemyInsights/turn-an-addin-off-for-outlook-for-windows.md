# <a name="turn-an-add-in-off-for-outlook-for-windows"></a>Вимкнення надбудови для програми Outlook для Windows

Якщо ви не хочете використовувати надбудову в програмі Outlook 2013 або Outlook 2016 для Windows, ви можете вимкнути його, щоб він не з'являвся в повідомленнях.  

1. У програмі Outlook виберіть **файл > керування надбудовами**.
2. Клацніть елемент  **Керування надбудовами**
3. Порада: відкриється інтернет-версія Outlook.
4. У розділі Керування надбудовами в стовпці увімкнуто **зніміть прапорець для надбудови**  , яку потрібно вимкнути.
5. Увімкнення або вимкнення надбудов

Порада: щоб знову ввімкнути надбудову, установіть один і той самий прапорець.
