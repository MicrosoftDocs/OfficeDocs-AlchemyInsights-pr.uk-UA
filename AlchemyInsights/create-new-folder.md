# <a name="create-a-new-folder"></a>Створити нову папку

1. У розділі **папки** в області переходів виберіть **нову папку**.

2. Введіть ім'я для папки та натисніть клавішу Enter.

Щоб отримати додаткові відомості, див. [Створення папки в Outlook.com](https://support.office.com/article/5fa8de74-3562-4729-ac1d-5599f470b25a?wt.mc_id=Office_Outlook_com_Alchemy).
