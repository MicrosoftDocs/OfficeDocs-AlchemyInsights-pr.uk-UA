# <a name="microsoft-365-urls-and-ip-address-ranges"></a>Діапазони URL- і IP-адрес Microsoft 365

Microsoft 365 потребує підключення до Інтернету за допомогою певних IP-адрес, які називаються *кінцевими точками.*
Список кінцевих точок, до яких потрібно отримати доступ, див. в статті [Діапазони URL- і IP-адрес Microsoft 365.](https://docs.microsoft.com/office365/enterprise/urls-and-ip-address-ranges) 

Веб-служба IP-адреси та URL-адреси Microsoft 365 допомагає краще визначати й розрізняти мережевий трафік Microsoft 365, що спрощує оцінювання, настроювання та оновлення змін. Докладні [відомості див. в статті IP-адреса та веб-служба URL-адрес Microsoft 365.](https://docs.microsoft.com/office365/enterprise/office-365-ip-web-service)
