# <a name="create-a-folder"></a>Створити папку

1. У нижній частині панелі папок в Outlook, в Інтернеті виберіть **нову папку**. Може знадобитися Прокрутіть вниз, на панелі папок бачити кнопку **Нова папка** .
1. У текстовому полі введіть ім'я для папки та натисніть клавішу Enter.

Щоб дізнатися більше, див. [Робота з теками повідомлень в Outlook, в Інтернеті](https://support.office.com/article/ae0f10d6-54e7-4f29-acd3-78cdc3fdcb9f).
