# <a name="managing-microsoft-stream-licenses"></a>Керування ліцензіями на Microsoft Stream

**Важливо:** Ви повинні бути в ролі адміністратора клієнта для виконання операцій, описаних у цьому розділі.

Ви повинні бути глобальним адміністратором або адміністратором управління користувачами. Для отримання додаткових відомостей див. [про ролі адміністратора](https://docs.microsoft.com/microsoft-365/admin/add-users/about-admin-roles).

Ви можете призначити [ліцензії для облікових записів користувачів за допомогою Office 365 PowerShell](https://go.microsoft.com/fwlink/p/?linkid=850410).
