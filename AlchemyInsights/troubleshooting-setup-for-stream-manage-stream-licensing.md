# <a name="managing-microsoft-stream-licenses"></a>Керування Microsoft потік ліцензій

**Важливі:** Ви повинні бути в ролі адміністрування компонентів для виконання операцій, що описані в цьому розділі.

Ви повинні бути глобального адміністратора або користувача керування адміністратора. Докладніше перегляньте статтю [ролей адміністратора про Office 365](https://support.office.com/article/About-Office-365-admin-roles-da585eea-f576-4f55-a1e0-87090b6aaa9d).

Ви можете призначити [ліцензій для облікових записів Office 365 PowerShell](https://go.microsoft.com/fwlink/p/?linkid=850410).
