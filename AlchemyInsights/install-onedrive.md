# <a name="install-onedrive"></a>Установити OneDrive

OneDrive синхронізація клієнт є попередньо встановленої з Windows 10. Щоб встановити OneDrive для Windows 7 або Mac або отримати мобільний додаток для телефону або планшетного ПК, перейдіть на [http://onedrive.com/download](http://onedrive.com/download).
