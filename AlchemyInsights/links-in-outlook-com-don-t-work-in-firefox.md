# <a name="links-in-outlookcom-dont-work-in-firefox"></a>Не працюють у Firefox посилання в Outlook.com

Посилання в повідомленнях електронної пошти Outlook.com не працюють, коли вони ви переглядали в Firefox браузерів. Mozilla є [працює над її вирішенням](https://go.microsoft.com/fwlink/p/?linkid=2001502&amp;clcid=0x409) для Firefox, але до цих пір:
  
- Будь ласка, використовуйте [Microsoft краю](https://go.microsoft.com/fwlink/p/?linkid=2001503&amp;clcid=0x409) або інший веб-браузер.
- Переглянути повідомлення в області читання. Для отримання додаткової інформації, читати як [змінити зовнішній вигляд поштовій скриньці Outlook.com](https://support.office.com/article/b41c2ecb-f23c-42b3-b7f8-659646d5e58c?wt.mc_id=Office_Outlook_com_Alchemy).
