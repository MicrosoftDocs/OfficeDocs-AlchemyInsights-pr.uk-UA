# <a name="gatewaynotreachable-when-setting-credentials"></a>Якщо настроїти облікові дані, функція Gatenotнедоступний

Якщо під час спроби настроїти облікові дані для джерела даних у службі Power BI, можливо, ви можете зіткнутися з Gatenotenotдосяжним. Це може бути результатом застарілого шлюзу. Інсталюйте найновіший шлюз і повторіть спробу.
