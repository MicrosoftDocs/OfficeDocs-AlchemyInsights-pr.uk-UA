# <a name="enable-external-communications-and-add-contacts"></a>Увімкнення зовнішніх зв'язків і додавання контактів

У програмі "Skype для бізнесу" можна додати людей до списку контактів, щоб вони могли бути лише одним клацанням миші. Кожному доданому контакту призначається членство в одній або кількох групах ваших контактів. Перегляньте [Додавання контакту в "Skype для бізнесу](https://support.office.com/article/add-a-contact-in-skype-for-business-89338023-2adf-4f5c-90b6-f8b6f72fadd1) ", щоб отримати докладні відомості. 

Переконайтеся, що адміністратор клієнта стежив за кроками, щоб [дозволити зовнішнім зв'язкам](https://docs.microsoft.com/skypeforbusiness/set-up-skype-for-business-online/allow-users-to-contact-external-skype-for-business-users).

Ви хочете, щоб ваші користувачі могли зв'язатися з користувачами Skype для споживачів? Дізнайтеся, як [надати користувачам Skype для бізнесу можливість додавати контакти Skype](https://docs.microsoft.com/skypeforbusiness/set-up-skype-for-business-online/let-skype-for-business-users-add-skype-contacts).
