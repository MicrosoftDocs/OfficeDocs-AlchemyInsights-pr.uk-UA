# <a name="credit-request-due-to-a-service-outage"></a>Запит на отримання кредиту через відключення служби

Щоб подати заявку на отримання кредитного запиту через відключення служби, відкрийте запит на обслуговування.

Переконайтеся, що ІДЕНТИФІКАТОР інциденту, знайдений на [сторінці "справність служби](https://docs.microsoft.com/office365/enterprise/view-service-health) " в центрі адміністрування Microsoft 365.

Просимо вас повідомити, що, коли ваш запит обробляється, він буде проходити валідації, щоб спостерігати за наслідками відключення.
