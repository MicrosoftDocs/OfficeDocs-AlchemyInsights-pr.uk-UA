# <a name="credit-request-due-to-a-service-outage"></a>Запит на кредит через збій у роботі служби

Щоб подати заявку на отримання кредиту через збій у роботі служби, будь ласка, відкрийте запит на обслуговування.

Будь ласка, переконайтеся, що ІДЕНТИФІКАТОР інциденту, знайдений на [сторінці "справність служби](https://docs.microsoft.com/office365/enterprise/view-service-health) " в центрі адміністрування Microsoft 365.

Будь ласка, зверніть увагу, що, як ваш запит обробляється, він буде проходити перевірки, щоб спостерігати за наслідками відключення.
