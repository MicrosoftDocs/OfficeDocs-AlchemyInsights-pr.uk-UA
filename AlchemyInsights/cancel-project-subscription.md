# <a name="cancel-project-subscription"></a>Скасування передплати на проект

Адміністратори можуть скасувати передплату проекту для своєї організації в центрі адміністрування Microsoft 365.

1. Перейдіть на сторінку " **виставлення рахунків** на \> [продукти](https://go.microsoft.com/fwlink/p/?linkid=842054) ".
2. На вкладці **продукти** виберіть передплату проекту, яку потрібно скасувати. Виберіть пункт **додаткові дії** (три крапки), а потім натисніть **кнопку Скасувати передплату**.
3. В області **скасування передплати** виберіть причину скасування. За потреби надайте будь-які відгуки.
4. Натисніть кнопку **зберегти**.

**Повернення коштів:** Будь-який розрахунковий рахунок буде повернуто вам протягом наступного циклу рахунків.

Докладні відомості наведено в статті [скасування передплати](https://docs.microsoft.com/microsoft-365/commerce/subscriptions/cancel-your-subscription).
