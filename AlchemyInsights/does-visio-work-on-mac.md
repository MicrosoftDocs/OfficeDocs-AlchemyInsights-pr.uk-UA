# <a name="does-visio-work-on-a-mac"></a>Чи працює програма Visio на комп'ютері Mac?

Класичної програми Visio для Mac не існує. Однак, якщо для користувача, який має потребу в програмі, [призначено ліцензію Visio Online](https://docs.microsoft.com/microsoft-365/admin/add-users/add-users), вони можуть [переглядати, створювати та редагувати схеми у Visio Online](https://support.office.com/article/06f04845-91b8-4e8f-881f-a43c970735fc?wt.mc_id=OfficeAdm_ClientDIA_Alchemy1799) у браузері.
