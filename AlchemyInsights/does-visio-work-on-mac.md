# <a name="does-visio-work-on-a-mac"></a>Visio працює на Mac?

Немає застосунку Visio робочого стола для Mac. Однак, тих пір, поки користувач, який потребує додаток має [ліцензію Visio на сайті призначені](https://docs.microsoft.com/office365/admin/subscriptions-and-billing/assign-licenses-to-users?wt.mc_id=OfficeAdm_ClientDIA_Alchemy1799), вони можуть [Перегляд, створення та редагування діаграми в Visio онлайн](https://support.office.com/article/06f04845-91b8-4e8f-881f-a43c970735fc?wt.mc_id=OfficeAdm_ClientDIA_Alchemy1799) у веб-браузері.
