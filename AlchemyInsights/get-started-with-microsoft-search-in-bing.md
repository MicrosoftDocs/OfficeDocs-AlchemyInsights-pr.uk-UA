# <a name="get-started-with-microsoft-search-in-bing"></a>Початок роботи з пошуком Microsoft у Bing

Якщо ваша організація новачок у службі Microsoft Search у Bing, радимо переглянути посібник з [Microsoft Search у Bing Admin Essentials,](https://go.microsoft.com/fwlink/p/?linkid=2127979)який містить відомості про налаштування, настроювання та розгортання пошуку Microsoft у службі Bing.

Щоб підвищити обізнаність користувачів про пошук Microsoft у Bing, запропонуйте їм пакет [adoption](https://go.microsoft.com/fwlink/p/?LinkID=2114710) (ZIP-файл), який надає навчальні курси та ресурси для спілкування.

Докладні відомості та засоби див. в статті Як покращити пошук [Microsoft у Bing для організації.](https://go.microsoft.com/fwlink/?linkid=2152022)
