# <a name="upgrade-to-the-full-version-of-microsoft-teams"></a>Оновлення до повної версії Microsoft teams

Потрібно оновити безкоштовно до повної версії Microsoft teams? Переглянути [оновлення Microsoft teams безкоштовно для microsoft 365 підписка версія](https://docs.microsoft.com/microsoftteams/upgrade-freemium)

Вам потрібно буде виконати ці вимоги:

- Ви особа, яка підписався на поточну безкоштовну версію команд.
- Якщо у вас є власний домен, він вже не може бути пов'язаний з Azure Active Directory.
- Для передавання даних під час оновлення необхідно підписатися під час процесу оновлення. Якщо ви придбали підписку за межами процесу оновлення, дані не можна передавати.
