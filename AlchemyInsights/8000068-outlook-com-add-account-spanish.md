# <a name="agregar-sus-otras-cuentas-de-correo-electrnico"></a>Agregar sus Отрас cuentas de correo electrónico

Cuando agregue Отрас cuentas de correo electrónico Outlook.com, puede VFD зйомки Outlook.com para enviar y leer mensajes de correo electrónico de estas cuentas гріх salir de Outlook.com. Cada cuenta que se agrega Outlook.com se denomina una cuenta conectada. También puede [crear ООН псевдонім de correo electrónico](https://support.office.com/es-es/article/agregar-o-quitar-un-alias-de-correo-electrónico-en-outlook-com-459b1989-356d-40fa-a689-8f285b13f1f2) que утилізац la misma Bandeja de entrada, enviar, elementos y lista de contactos como су cuenta de Outlook.com existente.

[Леа cómo agregar sus Отрас cuentas de correo electrónico в Outlook.com.](https://support.office.com/es-es/article/agregar-sus-otras-cuentas-de-correo-electrónico-a-outlook-com-c5224df4-5885-4e79-91ba-523aa743f0ba?ui=es-ES&rs=es-ES&ad=ES)
