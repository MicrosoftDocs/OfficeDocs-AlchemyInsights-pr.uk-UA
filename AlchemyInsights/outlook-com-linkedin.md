# <a name="issues-with-junk-email-claiming-to-be-from-linkedin"></a>Проблеми з небажаної пошти, стверджуючи, бути з LinkedIn

Нам відомо, що деякі користувачі отримують небажані листи, які нібито від LinkedIn.

Рекомендовано використовувати вхідні правила для фільтрування ці листи.

Для отримання додаткової інформації, читати як [використовувати вхідні правила в Outlook.com](https://support.office.com/article/4b094371-a5d7-49bd-8b1b-4e4896a7cc5d?wt.mc_id=Office_Outlook_com_Alchemy).
