# <a name="need-to-configure-your-live-event-with-an-external-encoder"></a>Потрібно налаштувати пряму подію за допомогою зовнішнього кодувальника?

Якщо у вас виникли проблеми з отриманням кодувальника для роботи з командами Live Event, зверніться до документації з настроювання кодувальника: [Настроювання кодери для потокового передавання в Microsoft Stream](https://docs.microsoft.com/stream/live-encoder-setup)
