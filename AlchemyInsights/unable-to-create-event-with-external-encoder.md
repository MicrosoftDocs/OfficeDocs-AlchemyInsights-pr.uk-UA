# <a name="need-to-configure-your-live-event-with-an-external-encoder"></a>Потрібно настроїти подію наживо за допомогою зовнішнього кодувальника?

Якщо у вас виникли проблеми з вашим кодувальником, щоб працювати зі своїми командами в режимі реального часу, ознайомтеся з документацією з настроювання кодувальника: [налаштування кодери для потокового передавання в програмі Microsoft Stream](https://docs.microsoft.com/stream/live-encoder-setup)
