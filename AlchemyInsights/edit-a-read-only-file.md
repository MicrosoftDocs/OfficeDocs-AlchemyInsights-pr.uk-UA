# <a name="edit-a-read-only-file"></a>Редагувати файл лише для читання

У бібліотеках документів при натисканні на ім'я документа, програми Office може несподівано відкрити його як файл лише для читання. Щоб відкрити його для правки, натисніть кнопку **Відкрити** (три крапки) і натисніть кнопку **ред.**
