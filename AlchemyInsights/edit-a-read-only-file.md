# <a name="edit-a-read-only-file"></a>Редагування файлу лише для читання

У бібліотеках документів, якщо клацнути ім'я документа, програма Office може несподівано відкрити його як файл лише для читання. Щоб відкрити його для редагування, натисніть кнопку **Відкрити** (три крапки) і виберіть пункт **редагувати.**
