# <a name="change-billing-country-or-region"></a>Змінити країну або регіон для виставлення рахунків

Змінити країну або регіон із Облікових записів для виставлення рахунків у Центрі адміністрування   >   Microsoft 365 не можна. Потрібно зареєструвати новий обліковий запис і вибрати потрібну країну або регіон.
