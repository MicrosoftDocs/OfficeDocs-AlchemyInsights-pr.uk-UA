# <a name="troubleshoot-search-issues"></a>Усунення проблем із пошуком

Якщо у вас виникли проблеми з пошуком у програмі Outlook, ознайомтеся з наведеними нижче посиланнями для довідки та виправлення неполадок.

Нижче наведено найпоширеніші проблеми, які можна почати.

- [Пошук у програмі Outlook не повертає результатів](https://support.office.com/article/2556b11f-f4d8-46be-b0a7-de33a3f4f066#bkmk_noresults)
- [Пошук у програмі Outlook неповних результатів](https://support.office.com/article/2556b11f-f4d8-46be-b0a7-de33a3f4f066#bkmk_incompleteresults)
- [Пошук у програмі Outlook не вдається знайти старіших електронних листів](https://support.office.com/article/2556b11f-f4d8-46be-b0a7-de33a3f4f066#bkmk_olderemails)

Можливі рішення для менших поширених проблем можна знайти тут:

- [Лише Outlook 2016: вкладка "Пошук" недоступна](https://support.office.com/article/2556b11f-f4d8-46be-b0a7-de33a3f4f066#bkmk_greytab)
- [Спільна поштова скринька: проблеми з результатами пошуку](https://support.office.com/article/2556b11f-f4d8-46be-b0a7-de33a3f4f066#bkmk_sharedmailbox)

- [У папках пошуку відображаються несподівані об'єкти, які не містять жодної теми](https://support.microsoft.com/help/4035436/outlook-search-folders-show-items-with-blank-subject)
