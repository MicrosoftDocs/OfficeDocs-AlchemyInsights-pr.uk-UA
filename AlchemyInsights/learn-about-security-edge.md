# <a name="learn-about-the-security-features-of-microsoft-edge"></a>Відомості про функції безпеки Microsoft Edge

ІТ-адміністратори підприємства впораються з забагатою наявних і нових проблем, пов'язаних із безпекою. Вирішення цих проблем – це дії, які потрібно виконати адміністраторам, щоб захистити корпоративну мережу та пристрої від атак зловмисників і запобігти несанкціонованому доступу та витоку корпоративних даних. Щоб допомогти адміністраторам вирішити ці проблеми, Microsoft Edge (версія 77 або пізніша) надає кілька унікальних вбудованих можливостей. 

Докладні відомості див. в [статті Безпека в Microsoft Edge для вашого бізнесу.](/DeployEdge/ms-edge-security-for-business)
