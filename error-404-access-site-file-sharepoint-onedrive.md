# <a name="error-404-when-accessing-files-in-sharepoint-or-onedrive"></a>Помилка 404, коли доступ до файлів у SharePoint або OneDrive

<strong>Помилка 404 </strong>отримав, коли користувачі намагаються отримати доступ до сайту або файлів SharePoint або OneDrive. Це може бути спричинене сайту або txt або група отримання перейменовано, переміщено або видалено.
Користувачі будуть випробовувати помилки 404, намагається отримати доступ до кореневої колекції сайтів та її видалено.

Щоб усунути помилку 404 для сайту, який має перейменовано, переміщено або видалено: класичний сайтів, які існують в центру адміністрування класичний дізнатися <a href="https://docs.microsoft.com/en-us/sharepoint/restore-deleted-site-collection">відновити видалену вузлів</a>.&nbsp;</li> <li>Для сучасних сайтів (зв'язку, підключених до групи, або інші сайти) які існують нові admin центр попереднього, побачити <a href="https://docs.microsoft.com/en-us/sharepoint/view-and-restore-deleted-sites-in-new-admin-center">вигляд і відновити видалені сайтів нового центру адміністрування SharePoint. 

Щоб вирішити файл (або іншого елемента) що перейменовано, переміщено або видалено, перейдіть на сайт SharePoint або OneDrive і переглянути кошика від змісту сайту, побачити [відновити елементи з кошика сайту SharePoint](https://support.office.com/en-us/article/Restore-items-in-the-Recycle-Bin-of-a-SharePoint-site-6df466b6-55f2-4898-8d6e-c0dff851a0be).
